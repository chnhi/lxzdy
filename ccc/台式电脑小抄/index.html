# 配置单

# 近期要点

  近期显卡价格由于供应短缺与挖矿需求，价格高企，cpu价格也虚高，推荐先用核显凑合着等显卡价格回落。

  需要买显卡可以考虑二手rx460 2g，显存太小挖不了矿，能开核提升百分之10的性能。PUBG，GTA5等游戏低画质60帧。或者一手的马甲RX560，659左右。

# 主流配置

  • CPU AMD 3500X (散片750)

  • 散热器雅浚B3+IFE2防拔出支架(119)

  • 主板 华硕 TUF B450M Pro S (京东史低539)

  • 内存 光威弈pro(3000mhz) 8Gx2 （京东史低179x2）

  • 显卡 映泰RX560 （天猫659）

  • 硬盘光威 弈Pro nvme缓存版 512g（京东489）

  • 电源航嘉jumper450B （京东史低219）

  • 机箱先马 小黑洞（京东239）

  总价3299左右。预留一定升级潜力。cpu全核4.3g是基本盘。内存3200cl16是基本盘，再朝上超就要看体质。机箱可以换成先马 平头哥m1或者游戏帝国 寂静岭自己配风扇，小黑洞多了消音棉且标配3把风扇。

  内存弈pro是国产颗粒。也可以考虑换科赋普条或马甲条，cjr颗粒，3466cl16是基本盘。如果金士顿fury更便宜也可以考虑。

# 入门配置

  • CPU 200ge (255)

  • 散热器超频三七星瓢虫V4（39）

  • 主板 映泰 (BIOSTAR)A32M2（299）

    • 昂达（ONDA）A320S全固版（289）

  • 内存 威士奇DDR4 4Gx2 （99x2）

  • 显卡 集成

  • 硬盘致钛 256g M.2接口 PC005 Active系列（京东史低249）

  • 电源首席玩家 火玫瑰400 （京东史低119）

  • 机箱游戏帝国 寂静岭（京东99）

      总价1299左右，如果无所谓升级潜力，品牌机，整机差不多配置也这个价格。

      英特尔方案g6400+梅捷h410m焱龙，如果要加显卡可以考虑。

      最好加一把机箱风扇在主板接口旁边那个风扇位，对散热改善很大。

      电源同价位有牌子更大但是是被动式的鑫谷s7，也可以考虑加钱换航嘉450b。
