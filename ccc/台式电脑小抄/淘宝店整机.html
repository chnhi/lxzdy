    # 淘宝店整机

      要求不高，凑合能用。在奸商里找出相对合理，又实惠的，但他们摸棱两可的配件喜欢利润率高的产品。从某种意义上来说，性价比再高，基本还是一个档次的产品，性能不会超出一个数量级，也许省心省力的傻瓜方案才是更明智的选择。

    # 主流

    淘宝摩西电脑 (opens new window)默认配置就有较高合理性，没有为了价格战弄个奇怪的默认配置，有压倒性优势的产品会选用，摸棱两可的配件会用利润率高的。在淘宝店装机里会属于比较贵，自己买配件照着我推荐的选购渠道很容易比他便宜。

    装机猿合作·淘宝西安水元素 (opens new window)大部分配件都喜欢利润率高但又不算特别坑的，默认主板最好换成我推荐里出现的主板，内存要联系客服插两根。自己买配件要比他便宜有点麻烦，没那么简单。

    淘宝蒙面IT侠 (opens new window) 相对合理也便宜，疑似七彩虹代理，什么都用七彩虹，主板可以考虑替换成我的推荐。

    # 入门

    淘宝河东数码 (opens new window)搭载3200g 200ge机型

    入门价位，只要求电脑满足办公，轻度游戏又尽可能便宜。其中a8-9600那款不用考虑，办公无所谓，轻度游戏要联系客服插两根内存，具有升级75w级别显卡的潜力。

    淘宝捷盛电脑科技 (opens new window)搭载3200g 200ge机型

    贵一点，但是机箱是乔斯伯的铝机箱，好看些，默认配置还算合理。但是店铺看百度地图是开在小区里，不是电脑城卖电脑的，备货情况要打个问号。
