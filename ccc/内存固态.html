# 内存和固态

更新于2021.2.26

# 内存

  海盗船，芝奇等品牌4000及以上高频条基本是10层板，低频条不清楚。通常海盗船会是相对便宜的高频条。

  • 十铨 开创者最便宜10层板马甲条。
  • tt TOUGHRAM Z-ONE 最便宜10层板灯条。
  • 影驰名人堂疑似最便宜b-die
注意层数多不代表颗粒体质好。笔记本不用想高频条了。
  • 光威弈Pro国产长鑫颗粒

  不买高频条首选，3200cl16是基本盘。超频潜力不如英睿达或科赋。

  再便宜一点，雷克沙（长鑫颗粒）。

  还要再便宜，玖合，酷兽，威士奇，宏想，金百达，拆机，二手找便宜的，能点亮基本都稳定，拆机片，降级片居多。大牌也很容易买到降级片，还不如买杂牌，便宜。

# 固态

  固态的跑分通常只能展现爆发力，但我觉得持久力更重要。离散测试或者数据量够大的跑分测试有一定价值。评测常做的pcmark8综合吞吐量一般要测50g以上数据，有一定价值，测试也相对比较好找。但还是不够表现脏盘，接近满盘情况下的表现。

# m.2-PCI-E 3.0

  • 致钛 nvme PC005 Active SM2262EN 长江原厂颗粒 外置缓存

  8通道主控。512g版本pcmark8综合吞吐量559.64mb/s。慧荣方案爆发力比弈Pro差，但重度负载波动小，反而吞吐量更大。

  • 光威 弈Pro忆芯STAR1000P（28nm双核） 长江原厂颗粒 外置缓存

  8通道主控。512g版本pcmark8综合吞吐量420mb/s。1t版本pcmark8综合吞吐量546.09mb/s。光威全国产方案，空盘跑分更高，也更便宜。

  国产双雄与同价位4通道产品如wd sn550或其他国产牌子的8通道白片方案没有拉开性能差距。同时工包pm981a性能更高，价格更实惠。

  有售后最便宜8通道主控原厂正片。对比友商的8通道主控方案如三星970evo plus，三星970 evo(工包pm981a)，wd SN750，浦科特M9PE，爆发与缓外性能差一些，价格便宜些。

  • 海康c2000prosm2262en 紫光自封片 外置缓存

  紫光自封片，除非开车。

  • 影驰 铁甲战将PS5013-E13T 群联自封片 内置缓存

  群联白片是固态硬盘的下限。更便宜的颗粒来源复杂，不能控制颗粒品质。群联方案其他牌子更便宜的也可以。

  影驰有线下渠道，淘宝有便宜的。

  nvme方案的群联自封片品质比sata方案好。

# 旗舰

  只有长时间重度负载或者作为阵列缓存才需要考虑旗舰固态。玩游戏或者日常正常使用的没必要考虑。作为缓存，拆机傲腾16g，32g分别是50块，120块左右，买两块组raid也是不错的选择。

  • 三星970 pro

  作为缓存或者长时间重度读写负载不掉速，MLC颗粒。工包sm961，pm961大概率不是新的。如果没货的话可以考虑970evo plus，缓外写入也有1.65g/s，便宜多了。pcie4.0方案的三星980pro，缓外写入前300g 1.85g/s，之后掉到1.65g/s。

  pcie4.0方案还有群联e18的影驰 名人堂hof EXTREME，wd SN850靠全盘模拟SLC撑场面，包括三星随着空间变小，表现相对pcie3.0的产品没有显著提升，酌情考虑。wd SN850 缓外前500g 2.7g/s，之后1.25g/s。群联e18也是全盘动态slc，搭载e16的hof pro应该要清仓了，可以抄底，缺点是28nm制程主控。几款产品表现都很不错，我推荐哪个便宜买哪个。

# 2.5寸-SATA

  • 致钛SC001 Active 联芸MAS0902A 长江原厂颗粒 内置缓存

  英特尔原因为原厂颗粒被乱吹的话，长江原厂盘为什么不可以？

  同价位的凯侠tc10性能差不多。wd线下渠道价格低，绿盘性能实在烂，蓝盘性能差不多。860evo更贵也更强，但是可以买m.2。

  • 影驰 铁甲战将PS3111-S11 群联自封片 内置缓存

  480g版pcmark8吞吐量150~200mb/s，除非预算实在紧张，否则不推荐。

  群联白片是固态硬盘的下限。更便宜的固态颗粒来源复杂，不能控制颗粒品质。群联方案其他牌子更便宜的也可以。

  海康sata也是群联方案，看他开车的时候有没有比影驰便宜。
