# 青柠超滤

超滤膜丝拆洗 (opens new window)

青柠超滤是效果最好的前级，一次性成本较高，但实际折算成本还可以，可以降低纯水机耗材成本，如省掉PP棉。

但是隔几个月要手动排污，甚至拆洗。如果把排污管接到马桶或专门的“拖把龙头”，生活用水就能实现自动冲洗膜丝。甚至一两年才需要拆洗。

青柠后面不要接压力桶搞反冲洗。

# 施工中

强烈推荐直接买 北方水处理 的DIY成品，所有配件选择都非常合理，都是有产品通过NSF认证的国际大牌。

北方水处理 (opens new window) 机架式用陶氏原装500g膜，大概1650左右。膜要560元左右。网友拆箱 (opens new window)

我因为要用国产RO膜只能自己组装，用美国陶氏膜，我就直接买北方水处理的成品了。我自己装，省掉第5级，主料辅料都用有NSF认证的品牌，不算RO膜成本在900左右。北方水处理不算RO膜，1100左右。走线没他的美观。

开泉净水 (opens new window)推荐用崧泉PP滤瓶的产品，最便宜的大概900左右。辅料用国产没去认证的。

希库科技 (opens new window) 一些主料用的是比北方水处理更贵的，辅料没写。

六七百的用料堪忧，特别是几个打价格战的贴牌大王，基本都是ABS滤瓶，几块一个，关键没听说过食品级ABS塑料。崧泉PP滤瓶四五十一个，国产PP滤瓶，没去花钱认证的大概三十块一个。也因此，其他DIY成品基本不推荐了，不是是跟厂货一样，配件选择都藏着掖着，就是性价比还不如开泉净水。

# 滤芯

RO膜前级滤芯主要作用是保护RO膜，过滤影响RO膜寿命的物质。

  • PP棉，除颗粒物。

  • UDF颗粒活性炭，除氯，会掉渣。

  • CTO压铸活性炭，进一步除氯。挡前级UDF掉渣。

  • 复合滤芯就是把这三级压成一根滤芯。

    • 买厂牌复合滤芯产品，前级必须有青柠超滤或自己加一根PP棉滤芯，延长复合滤芯寿命，不然滤芯里的PP棉一两个月就黄了。多级滤芯，单换PP棉几块钱。

  • 小米复合滤芯活动价还算合理。有NSF认证的溢泰，力晶一套三级滤芯也要100多块。

    • 多级滤芯厂货，PP棉，后置T33活性炭推荐买兼容滤芯。

  • 后置T33活性炭，接在RO膜纯水后面，主要作用是改善口感，但是会滋生细菌，推荐是三个月更换。我个人觉得DIY产品可以不加这一级。

  • 超滤膜，可以除细菌，病毒，水生生物。

  • RO膜，过滤后几乎是纯净水。

一旦无氯，又不是纯水，就会大量滋生细菌。

明白各种常见滤芯的作用，DIY是可以自由发挥的。比如前级有青柠超滤，可以把第一级滤瓶省掉或换成活性炭。也可以先过活性炭，超滤，用来洗菜洗水果，再过一些高端滤芯有NSF53认证初步除重金属之类的，再过RO。

用成本较低的RO膜，可以考虑PP棉+颗粒炭+PP棉这样较为便宜的前级。

# 75g有桶 vs 400g无桶 陈水难题

75g流量很小，要有一个压力桶储水，因为是纯水，即使细菌滋生也是比较有限的。破拆压力桶视频 (opens new window)显示压力桶里头还是比较干净的。小流量滤芯的除盐率一般都相对会更高一些，成本因为寿命不一样，不好比较，理论上实际成本应该差不多。75g滤芯售价要便宜很多，用的泵之类的也可以便宜一些，动静也会更小。

大流量无桶机最大的短板是，纯水端的陈水会被污染，通常隔夜后，放水30s~50s之后才是纯水。市面上的零陈水方案基本都是加一个0.5g小压力桶，并联在RO膜出水口和进水口之间。每次出水后，额外做一些纯水，流到ro膜进水口。对于高频次少量取水而言，这样会有一点浪费水。

也可以考虑艾文水盒子，出水后检测TDS，高TDS的陈水作为废水排掉，纯净水才从水龙头出水。

# 台式净饮机

RO膜需要保持浸液防堵，这种净饮机受限于体积，基本只能用厂货滤芯，结果是成本高，寿命短。制的纯水存在纯水仓里，竟然还会滋生大量细菌，可见可靠性存疑，出水的时候用紫外线灯装模做样的照一下除菌，能除多少，值得怀疑。更有UV照射塑料等槽点。

废水流量调节

废水阀是关闭后，还会留个小洞的进水阀，流量不同既洞口大小不同。根据流量换算废水比准确性有限。比如1000cc，额定流量1L/min，400g的膜通常额定制水400加仑/天，折算1L/min，废水比1:1。但是这是理想状态，至少要膜前压力0.7MPa，增压泵流量2L/min，往往就达不到。

流量调节阀和进水阀组成的组合阀可以自由调节废水流量。调节流量阀实际改变的是膜前压力，而制水量和膜前压力正相关。以汇通膜为例，膜前压力在0.7MPa既俗称的7公斤水压。大概会达到1：1废水比。脱盐率95%。膜前压力越低，制水量就越少，废水比就越低。通常推荐高于0.35MPa。如果原水TDS很高，可以降低膜前压力到0.4MPa。
