2024.4.9

我自己用中资镜片厂-明月， 详见文末。推荐超韧，1.71，PMC，1.6MR-8。

# 度数概述

  • 600度以内散光50度以内，不复杂的，推荐后文-实惠。性价比实在太高。
  • 贵一点，依视路1.6膜岩，选1.6（400度以内）或1.591（300度以内），详见文末中资镜片厂-明月对镜片材料的介绍。
  • 400~600度，1.6实惠，综合视觉品质最优明月1.71，详见后文-折射率与阿贝数。不差钱蔡司1.67智锐铂金。
  • 700度以上，蔡司，依视路，万新1.74等原厂加工，海淘jins1.74。
  • 实惠的1.74镜片，确定店家是全自动磨边机，或协商请蔡司或依视路地区代理代加工。详见后文-实惠。

高度近视叠加高度散光，最好做自由曲面镜片，如蔡司泽锐，能下加光的如依视路爱赞，个化定制如尼康八轴，蔡司智锐。或内非球面，双非球面等镜片。

国内有一些厂家提供相对实惠的自由曲面定制片。骏丰光学（曾代工精工全球），承启光学（洪旭），积家，五彩，天润（代工罗敦司得），苏大明世。

做自由曲面定制片，需要验光师见到镜架。

骏丰工厂在苏州，可在工厂验配。价格比其他的国产要贵一些，但是实力不俗，基础膜层也会镀成蓝绿膜。绿膜最厚，蓝绿膜，蓝紫膜依次更薄，理论上减反射性能更好，可能更耐久。

虽然1.6单非实惠，但500度以上已经开始有可感知的边缘畸变了。

不同折射率更多的是材料不同，不同材料有不同特性，建议了解材料特性后，自行判断，材料特性详见文末中资镜片厂-明月。

高度近视各种效应的影响都会放大，对精度要求更高，单单只是镜架倾斜角的改变都会极大的影响效果和舒适性，甚至成像形变。

# 配眼镜：镜架、镜片、验光、加工、调整

五点缺一不可。镜片物料成本很低，蔡司出厂价低到1折，而实体店的房租是成本大头。

网购蔡司

阿里巴巴（如深圳博学，山东正视）或淘宝（现片2~4折，定制3~5折，智锐1.6单光现片￥700+50原厂加工，定制￥860+50原厂加工），蔡司公众号有电子保修单。

公众号（卡尔蔡司光学）小程序（蔡司睛选）寻镜列出了大部分蔡司产品的标价与技术特点。只要能获取优秀合理的验光数据，网购下单加工同实体店验配是一样的。

蔡司定制片有无忧售后，不舒服可以免费重做。现片没有无忧售后，网购现片做好没有售后的心理准备。

验配是一个系统工程，验光师验光时需要见到镜架才能确定瞳高，前倾角等参数，做渐进，个化等更需要验光师推荐合适的镜架。

没有精准验配的蔡司对比国产镜片，无非就是透光率高1个百分点，更耐磨，根本无法感知。甚至如果优质光学区偏移到镜架外，光学体验还不如精准验配的国产镜片，是典型的买椟还珠。

# 找优秀验光服务

这些店铺做蔡司往往7折起，而验光一般99~399￥，开价最高的璞瑞699￥。 验配要负责售后，单验光则无所谓，更需要自己了解优秀合理的验光流程。

优秀合理的验光应做项目见后文-验光。

没做过 视功能检查 优先去医院或视光中心做一次视功能检查，避免对验光产生干扰。验光中很大一部分是主观感受是否清晰，如果实质是视功能异常，就会误判。

公众号（卡尔蔡司光学）小程序（蔡司睛选）寻店，列出了蔡司专业服务体验店和购置的蔡司验光设备。（现片5~7折，定制6~7折，智锐1.6单光现片￥1100定制￥1750）

采购IPP屈光分析仪的店，普遍7折起，提供优秀验光服务的概率会大一些。店长之类的大C可能要预约。

1、归根结底是希望找到认真负责的验光师。只是定位高的店，概率越高。眼镜行业是红海，细分程度非常高，验配质量要求很高的门店，基本都会往专业化方向转型。

2、通常这些店铺提供的套餐镜架都不怎么样，还贵，或只有高溢价品牌镜架。

3、推荐参照镜架要点，网购镜架。

只采购扫脸机器的店，大都提供及格的验光服务，只能保证下限，宝岛眼镜，一代眼镜，jins等一般也能保证下限。

两个搜索关键词，**检影验光，视功能检查。**通常验光师科班出身的概率较大，自我定位也是专业大C。

科班出身的验光师一般更客观，更敏感。

# 务必保持怀疑

@网友_思有崖而知无涯：高级验光店也会有学徒，经验丰富的视光师也是人，也会秀逗，所以务必保持怀疑！

# 蔡司验光仪器介绍：

  • IPP：屈光分析仪（i.ProfilerPlus）提供i.Scription光优视优化所需验光数据。普通验光主要是检验光线明亮情况下的屈光度，而IPP能分析出各种不同情况下的屈光度。
  • IPP光优视只让线下验配用。

愿意费劲用检影验光精确测散光的验光师凤毛麟角，而IPP眼底扫描就能测。如果波前像差很大，经过光优视优化，晚上看不清或者星芒现象都要好很多，最关键的是扫过一次眼底，知道自己散光，夜间波前像差的情况。

  • 扫脸（参考视频 (opens new window)）：三维定位配镜（VISUFIT 1000）（新），数字化定位系统（I.Terminal2）针对镜片，镜架，脸型进行通道调整，验证镜架是否适合脸型。

究其本质，就是拍照定位，会有误差，必须人工复检。

# 实惠

高溢价镜片目标人群是**“贵就是好”，究其本质，就是从单非镜片升级成“自由曲面”镜片。**

明月，伟星，万新，凯米，康耐特，优立，格林视通等1.6mr-8，同蔡司智锐1.6都是一样的mr-8材料。

镜片的优质光学区（请参考后文-折射率与阿贝数）成像质量没有区别，无非用机器测试，同度数折射率，蔡司铂金膜普遍比友商高1%~2%，而明月pmc一般比友商高1%不到，我不认为人眼能够感知这1%的透光率差异。

商家宣传会吹嘘98%~99%的透光率，但那是平光镜，有度数会下降，但一般就连批发5块一片的镜片，都能达到90%的透光率。

**“自由曲面”**镜片无非边缘成像桶形畸变更小，像差更小，但实际上除非是高度近视，不然很难感知。而且“自由曲面”不是没有代价，详见后文-“自由曲面”边缘模糊。

凯米是最大的代工厂，依视路很多贴牌oem都是凯米代工，很多国产贴牌凯米，或买1.6，1.74凯米基片，明月1.71基片，再自己镀膜，所以产品力都相近。一般阿贝数40的1.6基本都是mr-8，对材料的介绍详见文末中资镜片厂-明月。

# 店铺加工

商家一般贵的镜片直接“原厂加工”，店铺加工的磨边机不需要买多好的，而且单店订单量不够，不足以摊销进口全自动磨边机的成本，大都用半自动磨边机，用国产全自动磨边机的都很少。

半自动磨边机的下限实在太低了，全自动磨边机的下限要高很多。

  • 所谓蔡司依视路“原厂加工”，一般是各地区代理开的加工中心。所以有些店做实惠的镜片，也可以请蔡司依视路代理加工。
  • 做万新的连锁店，如一代眼镜，万新被依视路收购，可以由依视路代理 加工。万新默认的绿膜（翡翠膜）硬度偏低，e洁膜，莱宝膜会硬一点。最好mr-8（估计1.6套餐要400以上）
  • JINS (opens new window)国内连锁，399套餐1.6，自建加工中心。号称豪雅镜片，但要看镜片袋上的制造商，上海豪雅是经销商，会贴牌；豪雅广州光学镜片是豪雅工厂，验光时主动提出MPMVA足矫。
  • 配镜节（京东以往在4月中旬），618，双十一等活动，在镜宴或明月旗舰店等店网购。
  • 通常依视路旗下镜宴做1.6膜岩约250元左右，叠加券可能更低。
  • -
  • 有加工中心的连锁店，单量多，可以摊销成本，且用得上全自动磨边机的速度。但也不是绝对的。
  • -
  • 店铺加工，自己看看磨边机，长得像配钥匙的机器，一根轴上两个夹具，就是半自动磨边机，全自动磨边机一般会有个像显微镜一样的“扫描仪”。
  • 通常开在写字楼的连锁店性价比相对高一点，采购扫脸机器的蔡司专业服务店一般下限还行。
  • -
  • 有全自动磨边机的淘宝店，我看到上海信义眼镜实体店有依视路全自动磨边机，有淘宝店，偏贵。

如JINS等开在大商场的快消品牌，以JINS为例，海淘1.74好像才四五百。这类品牌的平价镜片个个都是大牌，但一分价钱一分货是亘古不变的。这些品牌用的尼康就是贴牌凯米，矬子里拔将军，可以拔豪雅，验光时要主动提出MPMVA足矫。

高度近视做1.74，本来就贵，尽量不要店铺加工，除非确认是依视路或尼德克的全自动磨边机。

1、店铺加工看师傅的态度和价值观，是否重视细节，有偏差是否愿意重做。

2、大部分成镜都合格。

3、不推荐网购特别便宜的非原厂加工，重做一幅甚至亏本。

4、网购店铺加工往往除了加工数据符合国标，其他一概不管。

# 镜架

亚洲人推荐有鼻托的。

单眼移心量=(镜片宽度+中梁宽度-瞳距)/2 ，虽然希望移心量尽可能小，但也要考虑不要夹头。3mm及格，1.5mm优秀。

一般镜架标注的53-17既镜片宽度和中梁宽度。

镜片高度一般35~38mm近似达到视野极限，大框没有意义，做大框必须点瞳高。高度近视酌情缩小到30mm。

越是移心量大，特别是大框镜架，越容易导致瞳孔无法对准光心，或优质光学区偏移导致被切削掉，非要做大框镜架，尽量要调整过前倾角使加工瞳高下移的镜架。 进一步了解前倾角。 (opens new window)

高度近视推荐蔡司镜架或裤阿短镜架 (opens new window)，调整过前倾角使得加工瞳高下移。推荐尽量避开大框，细镜腿等头重脚轻的镜架。

我自己瞳距65mm，400度，施洛华sc321。51-17，镜架高度38mm。

眼睛平面到耳朵的距离既颞侧距+12~15mm，是比较推荐的镜腿长度。

要达成移心量为零的镜框，可以考虑试试3D打印，感觉尼龙或尼龙碳纤维材料可行，镜腿和鼻托留螺丝孔。

国产品牌镜架：施洛华，派丽蒙，保圣，陌森，暴龙，海伦凯勒等，品牌镜架往往溢价很高，有很多经销商在淘宝闲鱼打骨折，往往100来块就能入手。阿里巴巴 (opens new window)淘宝上好点的白牌镜架也要近100。

倭国躬匠精神几千块的手工镜架，复刻版批发其实只要几十块。国产品牌钛架和高溢价品牌材料区别不大，镜架也是经典瞄准“贵就是好”。

# 验光

业余做法：电脑读数，插片验光看看没问题就拉倒。业余的行业主流。

合格做法：测试散光度数和轴位，红绿测试验证，双眼平衡测试。做自由曲面镜片，务必要求足矫。原因见后文-“自由曲面”边缘模糊。

如果情况不是很复杂，比如就四五百度左右的近视，合格做法也够用了。

优秀做法：

建议没做过 视功能检查 优先去医院或视光中心做一次视功能检查，避免对验光产生干扰。验光中很重要的部分是主观感受是否清晰，如果实质是视功能异常，就会产生干扰，造成误判。

  • 检查眼位
  • 角膜曲率仪
  • 裂隙灯检查角膜，虹膜，晶体
  • 检查主导眼确保双眼平衡
  • 综合检眼仪多步骤主觉验光
  • JCC交叉柱镜检查散光度数和轴位
  • 检影验光，客观判断，避免主觉验光无法确定是散光还是度数
  • 根据MPMVA确定最佳度数
  • 交叉复查

# 折射率与阿贝数

1.6折射率的镜片阿贝数要高一些，但是阿贝数只是一个指标。阿贝数较低的镜片会产生色散，色散参考三棱镜。

但也是要超过4°以上的偏心角才会产生色散，既视觉中心是不会产生色散的。镜片上以光心为中心，偏心距为半径的圆内是不会产生色散的。偏心距(cm)=(12×阿贝数)➗近视光度

度数与偏心距（cm）参考表：

通常不转动眼睛，正视前方，偏心距只要在0.6cm以内是不会产生色散的。

**低度数的情况下高阿贝数是无法发挥作用的。**除了色散，还要考虑到几何畸变，既成像比实际物体小。折射率越大镜片越薄，几何畸变就越小。不能片面的只看阿贝数。600度左右，1.71折射率是综合视觉品质最好的。

# 防蓝光=放屁

不要自欺欺人麻痹自己，关爱眼睛应该保持良好的用眼习惯，而不是跟着营销风口到处乱飞。

  • 目前为止没有任何研究证明防蓝光镜片有改善视力下降、防止眼底病变的作用。
  • 确定有害的是400nm以下的紫外线，常规镜片就有防UV膜，防紫外线还没色差。
  • 可见光阻隔率高必然产生色差，这是客观物理规律。

很多防蓝光镜片实际是三井UV420，主防紫外线，不阻隔数码蓝光，网上一搜，却各种舒服。

对数码蓝光的阻隔主要靠膜层反射，这就带来一个很可笑的现实，根据光路可逆原理，正面射入的蓝光被大量反射，那么从侧面和背面射入的蓝光，也会被反射进人眼！

  • 明月1.71防蓝光，1.6防蓝光PRO基片+镀膜防蓝光。450nm以下阻隔率都很高。
  • 依视路X4，膜致，凯米U6，万新，康耐特等品牌的防蓝光基本都是三井UV420，对数码蓝光430~445nm阻隔率并不高，但是防紫外线性能很不错。
  • 依视路A4，凯米BRC，蔡司钻立方防蓝光膜，膜层防430~440nm，理论上蓝光危害最大的波段。蔡司防紫外线性能影响较小，而凯米BRC，依视路A4，防紫外线性能拉跨，经典捡了芝麻丢了西瓜。
  • 蔡司防蓝光PLUS，三井UV420加膜层阻隔数码蓝光。

Rucker F, Britton S, Spatcher M, Hanowsky S. Blue light protects against temporal frequency sensitive refractive changes. Invest Ophthalmol Vis Sci. 2015;56(10):61216131.

户外运动可以预防近视，竟然可能和蓝光有关。阳光更富含短波长光，这转而通过视网膜多巴胺释放机制来缩短眼睛长度，减少散光，甚至逆转近视。

# 镜片黄

镜片黄，和uv阻隔有关。明月pmc这种uv380~390的就会白透，蔡司一般是UV400~405，就会显得黄一点。可见光波长在390nm-780nm，现有研究显示uv光对眼睛是不好的。

所谓uv380~390既约380~390nm更短波长的紫外线会被阻隔。

自由曲面镜片尽可能买智锐。自由曲面镜片更薄，边缘桶形畸变更小，但是光学性能未必更好。智锐加工精度高，不同位置的切向曲率与轴向曲率选择更合理，可以改善光学性能。

智锐-亚洲版，定制倾斜角，如果验光师愿意调整镜架的倾斜角，使镜片光轴与眼球回旋点重合，那么可以改善镜片度数与有效度数之间由倾斜角引起的误差。屈光度越高，倾斜角引起的误差就越大。

亚洲版数码型，支持50度以内下加光。40岁以上开始有老花的可以考虑，没老花不要做数码型。

最硬最耐磨的镜片是只镀一层加硬膜的加硬白片。增透膜，疏水膜等膜层都镀在加硬膜上，反而降低了耐磨性。

# 依视路

依视路更新膜致，膜岩后，性能跟上了，可以考虑。膜洁膜御没加硬，不推荐。

膜岩是高清镜片，相对A3，不仅提升耐磨度，在耐高温上也有进步，膜层老化后光学性能衰减也有明显进步。

膜致是前身是X4，UV415~420。非要防蓝光可以考虑。

# 变色片

变色片预期寿命短，主要阻隔可见光，紫外线阻隔还是靠镀膜。 除非是干眼症等眼疾，不然没必要配变色片。 确实有户外需求，可以考虑蔡司悦慕偏光片，注意偏光片看手机屏幕会有色阶。

变色片龙头全视线被依视路收购，最新一代技术用在依视路上，小弟用旧一代技术。 全视线八代的综合变色性能目前领先。蔡司自研变色深度略逊；豪雅自研变色速度慢，褪色后镜片底色偏深。小弟们用的六代在变色速度和变色深度上都有一定差距。

# 自由曲面边缘模糊

友商的双非某种意义上也是“自由曲面”的一种。 侯佳. 基于畸变校正的成像自由曲面光学设计[D].浙江大学,2013: 对于非球面而言，一般可以用公式来描述。在光学系统的优化中，当非球面的优化系数个数大于10时，优化变量的变化对优化的结果提高并没有很大的作用。因为当优化系数的个数大于10时，不同非球面之间的系数会有一个相互制约的作用，非球面优化系数的增加并不能从根本上使光学系统的性能得到最佳。 从非球面到自由曲面的发展是必然的，二者之间并没有实质性的差别。自由曲面相比非球面而言具有更大的自由度，能够根据人为的需要控制每一条光线的出射角度和方向等。光学自由曲面没有严格确切的定义，通常是指无法用球面或者非球面系数来表示或者是无法用确切的表达式来表示的光学曲面，主要是指非旋转对称的曲面或者只能用参数向量来表示的曲面（如NURS曲面）。 光学自由曲面经常是不能用初等解析曲面组成，而是由复杂方式自由变化的曲线曲面即所谓的自由曲线曲面组成，主要包括以下几类： 没有旋转对称轴的复杂非常规连续曲面，包括双曲率面、复曲面、多项式曲面、泽尼克多项式曲面等； 非连续、有面型突变的曲面，例如微透镜阵列、衍射面和二元光学等特殊表面； 非球面很大的曲面，包括旋转对称的自由曲面，如用于描述共性光学整流罩的椭圆形曲面。

# 偏轴误差

大部分光线是倾斜射入眼睛，既存在仰角，而不是理想状态下的平行光。因此镜片子午与弧矢两个方向不同的屈光度会引起误差。同时为了防止产生扭曲畸变，还要尽可能的让镜片符合车尔宁椭圆：

  • 偏轴散光为零，随着入射仰角增大，屈光度变小。
  • 偏轴屈光度为零，随着入射仰角增大，散光变大。

# 设计导向

  • 凯米单非是度数导向设计，联合光度大于光心光度，代价是边缘成像微小形变。
  • 泽锐是散光导向设计，联合光度小于光心光度，边缘光度比光心光度小。

除了光学设计，还有一部分验光不足矫，联合光度更低，叠加偏轴误差，加剧了边缘模糊的现象。

# 验配是系统工程

1、参见前文-折射率与阿贝数，镜片的优质光学区是有限的，以400度，1.6镜片为例，是一个半径0.8cm的圆。验配才能在加工前调整佩戴，确保优质光学区定位在镜片中心，不然很容易优质光学区偏移到镜架外被磨掉。

2、以棱镜效应为例，有很多方式调整：故意做大做小瞳距靠偏心产生棱镜补偿；对镜架进行调整，调整倾斜角，镜架高度；定制片可以棱镜补偿。

3、高度近视试一试用手改变眼镜的倾斜角，看看成像变化。泽锐智锐预设12度倾斜角。

4、点瞳高需要将镜片的参考轴与眼轴对齐，而不是毛估估对准瞳孔点一下，等效斜着戴眼镜。

5、电脑验光有精度范围，而插片验光是被验光者的主观感受，在遇到50度散光，或者加25度度数的时候，往往靠主观感受很难分辨，需要认真负责的验光师通过检影验光来判断到底是散光还是加度数。

6、越是功能性镜片等增加复杂度的系统，越容易出错，越需要验光师对光学，产品的理解，而不是验光师只管验光，全靠售货员推销。

再次重申，归根结底，还是希望找到认真负责的验光师。只是从概率的角度上，越是定位高的店，概率越高。眼镜行业是红海，细分程度非常高，验配质量要求很高的门店，基本都会往专业化方向转型。

# 镜片材料

MR-8成熟，图省事是最不坏的选择，并不代表其他材料没有可取之处。

缺点在于不镀膜的透光率最低。

无框眼镜推荐MR-8。PC拉伸强度差些，开孔易变形松动，做也能做。PMMA（亚克力）太脆，做不了。
