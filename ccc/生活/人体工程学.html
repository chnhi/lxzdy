# 人体工程学·桌椅

  • ergotron推荐舒适工作区范围 (opens new window)

  • 日本网站推荐工作区范围 (opens new window)，公式：

单位：厘米
椅面高度 =（0.25x身高-1 ）四舍五入
键鼠高度 = 身高x0.183 + 椅面高度 - 6
写字高度 = 键鼠高度 + 5

  • 国标GB/T 14776—93，推荐桌面高度：

  • 按键鼠高度买钢木桌。不伏案写字。便宜实惠。注意用显示器原厂底座，显示器会过低。考虑买支架或自行垫高。

  • 钢木桌也可以买宜家的可调节高度桌腿，桌板淘宝有实木的，也可以买宜家夹板。

  • 电动升降桌。

    • 平常保持键鼠高度，伏案写字就升上来一点。

    • 电动升降桌保持伏案高度，键鼠高度另买一个小钢木桌放在升降桌下面。

  • 电子琴桌，托架够大可以放下键盘加鼠标，往往电脑桌只能放下键盘。

# 显示器尺寸与深度

肉眼分辨极限点距 = 眼到屏幕距离 / 3438。（根据人眼可区分的最小视角1分角（1/60度）计算）

推荐屏幕宽度 = 人眼与屏幕的距离 × 1.154 - 6 （27寸16：9推荐眼离屏57cm）根据人眼平均视角计算看到整个屏幕不需要转脖子的最小距离。

# 椅子

网易严选·小蛮腰，凑合能用的里最便宜的。

稍贵些的支家1606，网布支架等用料更好些，性价比就不错。

西昊M57等，跟Herman Miller旗下产品曲线很像的都可以。

和保友金豪曲线很像的都不推荐。故意把腰托做的很突出，营造视觉冲击力，商业噱头。找个人平躺或侧躺在硬板床上，看看放松状态下的人体自然曲线，就明白这种就是在整蛊。

普通靠背椅，可以考虑买个米乔腰靠，不需要气动，效果也是有的，通常能实现健康的中等体型久坐不腰酸背痛。
