# 台灯

推荐买E27接口的台灯，易于维护。

现在技术相当成熟，Ra90以上的LED灯泡满天飞。

目前的高显色指数的LED灯泡主要是蓝光或紫光激发荧光粉，单芯片蓝光激发会有明显的蓝光波峰。双蓝光芯片，用两种波长波峰不同的蓝光LED激发，光谱图上蓝光部分会有两个小波峰。还有一种紫光激发荧光粉。

尽管这些灯泡大吹特吹如何护眼，显色指数也很高，然而一旦荧光粉老化，蓝光，紫光会直接泄露。相对被妖魔化的蓝光，紫光危害反而更大。

温度是荧光粉寿命的一大杀手，金属外壳的灯泡加强散热可能可以延长寿命。现在有很多廉价的高显指灯泡，全新的光谱还可以，但是往往用塑料外壳。

要求高，工作5000小时左右就开始老化。10000小时之后基本就能测到蓝光或紫光的波峰了。包括荧光灯管也是一样。

如果很怕蓝光，紫光，可以网上买个显示器的防蓝光膜贴在灯上，基本都是uv400近乎完全阻隔，410~450nm波长阻隔率50%左右。

# 卤素灯并不护眼

卤素灯虽然蓝光能量较小。但是红外能量却异常的高，红外能量和白内障正相关。光谱连续性较差，存在跳跃波动，很难算是高质量的光源。除非改用diy的DC电源否则存在频闪，而且是50hz低频频闪。

# 荧光灯搭配电子镇流器

荧光灯搭配电子镇流器，至少手机拍不出频闪，可以接受。

荧光灯发光原理是紫光激发三基色荧光粉，同魅族Lipro等相似，至少荧光粉老化前没有紫光泄露。荧光灯正常情况下就直接有uva泄露，灯管玻璃通常不能完全阻隔uva。作为室内全局照明可以考虑。

作为台灯，距离太近。有兴趣可以网上买个显示器的防蓝光膜，基本都是uv400近乎完全阻隔，410~450nm波长蓝光阻隔率50%左右。

# 无可视频闪就够了

所有的灯吹嘘无频闪基本都是手机拍不出频闪，肉眼无法察觉频闪，用仪器测还是有频闪，但只要符合国标就足够。

LED本来就是直流驱动，引起频闪的主要是AC转DC过程中产生的纹波以及调光方式，高频pwm调光相对所谓横流调光，反而频闪的比例有可能更低。
