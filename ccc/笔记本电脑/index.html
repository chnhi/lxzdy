2024.7.19

# 一分价钱一分货

比如YOGA和小新是一样的模具，但是YOGA会特挑严丝合缝的，稍微有点对不齐的会拿去做小新。

# 跑分非常片面

轻薄本的瞬时功耗PL2现在给的很高，跑分都很猛，但是持续重负载，如渲染转码视频，解压文件甚至从U盘拷文件，轻薄本会落回持续功耗PL1。P系列，U系列的处理器通常最多28w，一些机型甚至只有10~15w，表现会很差，常见跑分体现不出。如果用H系列35~45wPL1的处理器，在进行这些任务时，发挥强得多。但是风扇声，机身厚度重量可能就不算理想。

在Lunar Lake之前，我认为完全没必要考虑P,U系列处理器，因为Lunar Lake之前的待机功耗跟苹果M系列完全没法比，轻薄毫无意义，至少H系列能干硬仗。

# 买还是等？

TSMC N3节点提升较小，还涨价，而台积电2nm，导入GAA，恐有巨大提升，有成为钉子户潜力。预计相关产品要等到2025年末才会上市。

  • 英特尔13，14代H系列移动端也有不稳定的现象，极不推荐。

  • amd cpu轻薄本建议等zen5（AI 300系列），可能等到24年末。刚需可入zen3，4。

zen4相对zen3，前端大幅度改进，科学运算性能提升较大，游戏性能提升较小。

zen5将有两个制程4nm，3nm版本。4nm的应该24年就上市，架构有一定提升。

流言：

Granite Ridge（移动端Fire Range），2xCCD，最多16核，X3D，zen5+RDNA3.5，4nm，25年

Strix Point 单片，最多12核，4+8混合zen5c，16CU（RDNA3.5），NPU50T；25年

Kraken Point单片，4+4，8CU。

Strix Halo，也可能叫Sarlak，chiplet，16核+40CU（RDNA3.5）+32M Infinity Cache；功耗允许60~130w，25年

Sonoma Valley 全zen5c。三星4nm。

zen6内核代号“Morpheus”，可能26年末有相关产品上市。升级2.5D增大互连带宽，桌面端可能最多32核。将分为Standard、Dense Classic和Client Dense。前两者可能类似于zen5和zen5c。内存控制器，执行调度器都将从头重新设计。

  • 英特尔全新ultra系列，能效比提升较大，相同功耗性能提升10%以上。
  • 编解码，集显，低负载续航都有大幅度提升。
  • 游戏本等25年初15代Arrow Lake-H。
    • 15代Arrow Lake疑似砍掉超线程，Nova Lake再加回来。流言：15代的IPC提升幅度在20%+。
  • 英特尔Royal Cove将大改性能核，全新升级的超线程Rentable Units届时才会回归。可能要等到26年才会出现第一代。英特尔大核从06年至今都没有重构过，而Royal Cove将迈出一大步，会不会步子太大扯到蛋？

Meteor Lake （移动端14代），INTEL 4+TSMC N5，主打能效，Redwood Cove+Crestmont没有特别大的提升，主要还是为未来铺路。chiplet增加延迟导致IPC反而有一点下降。

全新ultra系列导入NPU主要是相对gpu有更好的能效。我个人认为意义不特别大，本地跑的深度学习应用，主要还是图像处理，轻度的如直播抠像不需要深度学习，重度的推荐购买大显存N卡。

考虑到战未来，NPU支持的是微软的统一API，未来可能连输入法都会运行一个微型chatgpt来改善输入体验。NPU的兼容性稳定性可能会好一些。

  • 轻薄本可以等25年初Lunar Lake-U，单芯片，主打低功耗体验，对标苹果M。

鉴于全新ultra系列的Meteor Lake价格上涨，感觉Lunar Lake-U可能价格可能也不过如此，可以期待一手，对标苹果M，低续航功耗将进一步降低，而且选择成熟路线，可能降低成本不会卖出天价。

Lunar Lake-U将是单芯片，大小核心分两个簇，大核心可以完全关闭。

25年初Arrow Lake（15代），INTEL 20A。SoC Tile可能是TSMC N6。性能能效都可能有一定提升。导入GAA和PowerVia，将供电线路挪到反面。

流言：Arrow Lake-S的计算模块会采用台积电（TSMC）的N3B制造，而移动版本才会采用Intel 20A工艺，英特尔希望基于Lion Cove架构的P-Core能有更好的性能表现，IPC相比Redwood Cove架构的P-Core能有22%至34%的提升幅度。

Arrow Lake-H（15代） - Lion Cove P-Core + Skymont E-Core + Crestmont LP E-Core（SOC）

Lunar Lake-U - 4+4+2 Lion Cove P-Core + Skymont E-Core + Skymont LP E-Core（SOC）

Panther Lake，26年初 - 4+0+4(4Xe)（15W） 4+8+4(12Xe)(25W)

Cougar Cove ，Darkmont E-Core，LP E-Core，更新Soc Tile，可能像14代一样只做移动端。INTEL 18A。Xe3-LPG。

Nova Lake - 16+32+4，26年中。TSMC N2。超线程回归，导入Rentable Unit，Arctic Wolf E核。

流言：Lion Cove的IPC相比Meteor Lake的Redwood Cove架构提高了20%，相比Alder Lake的Golden Cove架构提高了45%。

  • nvidia显卡60系可能换用台积电2nmGAA，甚至英特尔18A，值得期待。预计27年中。

50系流言是三星3nmGAA工艺。三星所谓3nm的技术指标只是略强于台积电N4工艺。我觉得不值得期待。

此处“50系”指Blackwell，TSMC 4NP，晶体管全用来堆Tensor Core，制程没有升级，移动端应该没有明显提升，刚需直接40系。

# 光流插帧主要应用是视频补帧

dlss3由3个功能组成，光流法插帧，超分辨率，低延迟模式。dlss3版本的超分辨率模型40系以外的显卡一样可以启用，40系独占的是光流法插帧。

光流法插帧：利用NVOFA预测光流中间帧，将现有像素根据光流移动，深度学习补全空缺像素。个人认为最大的价值是应用于视频插帧。对于游戏而言，插帧不降低输入延迟，假设30帧插到60帧，还是不跟手。提升最大的场景是CPU单核瓶颈，然而往往是竞技类游戏，不可能开该功能增加输入延迟。更多的是锦上添花。目前最有意义的是微软模拟飞行。

pr的视频光流法插帧，在后一帧是现实存在并非预测的情况下，都不可避免出现 伪影，果冻效应，扭曲和模糊。

超分辨率：以较低分辨率渲染，利用深度学习超分辨率。

低延迟模式：cpu默认等gpu空了才会发送下一帧的数据，然后准备下下帧，在显卡满载的时候会大幅增加输入延迟，开启该功能后，cpu不再等待gpu。

# 没有烂东西只有烂价格

买于2013年之后的电脑只用于上网或处理文档，加内存，换固态硬盘，都很流畅。

  1. 上网，office，轻量ps，简单视频剪辑。硬件性能基本都够用，请关注屏幕颜值做工细节或减预算。

  2. 玩任何游戏最好配鸡肋独立显卡。核显受限于内存带宽(目前主因)，功耗墙，温度墙等原因，不能完全发挥，比如一打团就掉帧。

  3. 善用厂商客服，多花的钱就是买的服务。

  4. 个别机型把数字量化的参数做得很好，但是隐性参数如做工，质感，噪音控制，表面温度等实际体验才能感受到的部分敷衍了事。

  5. 不要有幸存者偏差，坏是概率，性价比高不代表不容易坏。

# 店铺介绍

  • 京东贵，售后快递方便。延保外包给联想售后，屁事多，拆机加内存都不行。
  • 官方旗舰店没有的内存+固态容量配置，由经销商拆包拆机，保新与否全凭商家价值观。
  • 淘宝很多是各地代理商，售后通过店铺返厂，存在邮费出入。拼多多同理。
  • 赠品不如折现。
  • 神舟老船长数码港 (opens new window)贴吧叫好，价格优势不大，店里都是原厂sku，原封。
  • 笔吧评测室经常在拼多多开车，公众号笔吧评测室。

# 避免常见误区:

  1. 重品牌轻型号，只看纸面参数。
  2. 忽略真实需求，被KOL片面强调的需求，消费主义吸引。
  3. 过度在意负面信息。没有完美不坏的产品。
  4. 用发展的眼光看问题，不要有刻板印象。
  5. up一般不会恶意诋毁。
