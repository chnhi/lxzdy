# 游戏本小抄

更新于2021.2.21

  1660ti以上档次显卡机型不是缺货就是涨价，除了机械革命不缺货。急需的先买1650机型过渡。同时即将换代。

# 5000以内唯一选择

  • 神舟tx6-cu5da神舟老船长4699

15.6寸高色域144hzIPS 1920*1080 i5-10400（台式机u可更换） gtx1650 8G内存可扩展 512G 2.5kg

   预算在5000以内，又希望笔记本性能够劲就不要想东想西，直接买神舟就对了。

  1650ti如果只贵100还可以考虑，贵了300完全不考虑。还想再实惠一点，可以考虑把cpu拆了卖个1000块左右，四核的i3-10100大概700块左右。

  对电竞游戏，网游如LOL，csgo，WOW等游戏的帧数不是追求跑很高的话可以考虑六核10500T es（QSRL），大概500块左右，单核睿3.4g，原机10400单核可以睿4.3g。10代由于是马甲，基本没什么bug，10400的es(QN8H)卖的很贵，850左右，性价比不高。

# 主流

  FPS，LOL，DOTA这类游戏，独显直出对体验的提升非常大。联想拯救者全线普及mux显卡切换独显直出，惠普暗影精灵要1660ti及以上档次显卡机型才搭载mux显卡切换。

  搭载锐龙cpu的游戏本务必要买配备了mux显卡切换或独显直出的机型。

  对显卡性能需求比较高的游戏更需要强劲的显卡性能，可以考虑神舟，同价位能能买到更高档次的显卡。

# 12000+

  这个价位的游戏本，性价比偏低，重量3kg以上，配上1斤的电源，太重了。如果是轻薄游戏本，性能释放又非常一般。我更倾向于买个主机或者台式机配个轻薄本。

# 轻薄型游戏本，创意工作者

此类笔记本特点是尽可能轻薄又有不错的性能释放。通常性能释放上的差别不至于达到一个特别显著的地步，同时价格高企，我个人推荐按造型挑选，玩游戏的话尽量挑有mux切换的机型。

  • MSI creator/GS66有最好的性能释放，但是造型不讨喜，支持mux切换，原厂带色域切换软件。

  • hp envy15 不支持mux切换，原厂带有色域切换软件。

  • 机械革命 umi pro 3 ，支持mux切换，便宜实惠。

  • 幻15，高贵的ROG居然不是最贵的就是巨大的恩赐，不支持mux切换，板载内存，原厂带色域切换软件。

  • Razer灵刃具有相对好一点的质感跟手感，不支持mux切换。

  • 技嘉Aero标称屏幕有潘通认证，但更厚，更重，造型有点不讨喜，不支持mux切换。有全高方向键，即想做游戏本又想做设计本，哪一个都没做好。

# 狂热的FPS爱好者

  15.6寸屏幕N156HHE-GA1，笔记本144hz ips屏幕延迟基本都有十几ms，60hz都勉强，只是刷新模块是144hz的。这块tn白到黑响应时间才3.5ms，色彩各方面比以前的tn有很大进步，但还是会泛白，上下可视角度依旧不好。这块面板微星用过，分别是GP63和GE63VR。

  从某种意义上来说，买一台TN显示器也是不错的选择。
