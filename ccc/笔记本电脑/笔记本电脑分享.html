# 笔记本电脑分享

更新于2021.2.21

# 矿工盯上笔记本啦！

  随着美元大量超发，作为避险品的比特币价格一路飙升，游戏本被矿工大量买走。就算是笔记本，按现在的币价，挖矿9个月就回本。这代笔记本基本上不用考虑除了京东自营天猫旗舰店苏宁自营以外的平台买了，谁知道有没有挖过矿呢？

  机械革命惨遭矿工嫌弃，原因不明。

# 没有烂东西只有烂价格

买于13年之后的电脑只用于上网或处理文档，加内存，换成固态硬盘，都还是比较流畅的。

  • 夏蒙乾 (opens new window)写的笔记本导购和普及非常详尽，而且经常更新。

  • 公众号笔吧评测室 评测非常详尽，一贯是讲参数，讲事实。

  • 孤独凤凰战士 (opens new window) 前联想工程师，会分享前瞻讯息与测试，可惜现在入职英特尔，不方便点评。

  1. 上网，office，轻量ps，简单视频剪辑。cpu是i5,i7或者r5,r7，有固态硬盘的机型已经足够流畅，所以要更多的关注屏幕颜值做工细节或者省点钱。

  2. 玩任何游戏，最好配有一块鸡肋独立显卡。核显受限于内存带宽(目前主因)，功耗墙，温度墙等原因，不能完全发挥，还间歇性有顿卡，不在意顿卡也可以考虑集显。

  3. 善用厂商的客服电话，多花的钱就是买的服务。

  4. 买的永远没有卖的精，个别机型把能用数字量化的参数都做得很好，但是一些隐性参数如做工，质感，噪音控制，表面温度等要实际体验才能感受到的部分草草了事。

  5. 不要有幸存者偏差，坏是概率事件，推荐的机器性价比高不代表不容易坏，性价比低的机器不代表容易坏。

# 网络平台和实体店价格

  都知道在网上查价，所以京东天猫平常标价虚高，给实体店留出利润空间。一个月能办七八次活动，这些所谓活动价才是网络日常售价。

  就算卖一样的价格，实体店向来花样百出，有的是办法坑人，厂商有特供机，实体店能换配件。

# AMDcpu短板

列出一些amd不大行的应用，如果涉及不到肯定买实惠的amd。

  1. 黑苹果拉跨，可以kvm虚拟机跑起来但是gpu加速不行。
  2. linux下大部分包，库都是用iu编译的，amd正常用起来是ok的，但不排除小概率事件。
  3. 计算机专业，深度学习，科研专业软件等很容易碰到英特尔写的专用库，尽量选英特尔。
  4. Zen3（5000系）游戏性能出现严重倒退，台式机开启FMAX，定频定压，调整EDC电流至300~400A以上可以解决问题，笔记本还要等厂商更新BIOS。
  5. 尽量买带mux显卡切换独显直出的机型。

# 店铺介绍

  • 京东贵一些，但售后的时候快递方便便宜。京东延保外包给联想售后，要求很高，拆机加内存都不行，意义不大。

  • 淘宝很多是各地代理商，售后通过店铺返厂，存在邮费出入。自己斟酌，拼多多同理。

  • 神舟老船长数码港 (opens new window)贴吧里比较叫好，不排除水军的可能，价格优势不很大，但是他店里一般没有乱七八糟的所谓定制版，也就是说一般机器是原封的。

  • 店里那些破烂赠品都别要，叫店家折现便宜点。

  • 笔吧评测室经常在拼多多开车，公众号笔吧评测室。

# throttlestop

  英特尔平台防降频，降温，某种意义上超频工具。

# 高色域IPSvs低色域IPS

  我自己一直用的都是覆盖srgb100%以上的屏幕，低色域屏幕曾经有两段使用经历，一段是十年前的华为手机， 可能是低色域ips，那个时候对设备没那么高要求，能用就行，要说槽点也是屏幕分辨率太低。一段是6年前的蓝天p151sma，那个屏幕槽点也根本不在色域而在于TN，用了不到3个月我就买了个显示器。曾经我给多位同学将笔记本TN换成低色域IPS，对低色域IPS相对TN的显著观感提升非常满意。

  高色域屏好么？当然好，但现在似乎逐渐成为一个价格歧视的概念，高低色域差价动辄400起，而散屏差价不过一百左右。但高低色域，非AB对比，非设计用途，日常使用能感知么？我不认为值400的差价。
