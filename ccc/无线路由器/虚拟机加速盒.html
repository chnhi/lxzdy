# v2ray机场客串加速盒

本例使用virtualbox虚拟机运行openwrt后，在主机修改网关实现v2ray客串加速盒，注意延迟相对正经加速器可能有50~100ms以上的劣化，且连接品质一般。

开箱即用的openwrt固件有homelede，发布页 (opens new window)，其发布页的google网盘分享 (opens new window)。

本例使用的是其Homelede2023.02.05内核V6.1.9-ESXI-VSphere虚拟机刷机-openwrt-x86-64-generic-squashfs-combined-efi.vmdk。

新建virtualbox虚拟机，类型linux，版本Linw2.6/3.x/4.x/5.x(64-bit)。cpu内存酌情分配，使用已有的虚拟硬盘文件，选中前文下载的openwrt-x86-64-generic-squashfs-combined-efi.vmdk

创建完成后进行配置，在网络中启用网卡2，将网卡1和网卡2都设置为桥接。

桥接网卡列表里没有实际的物理网卡，则win+q搜索网络连接，查看网络连接（控制面板）-右键物理网卡， 属性中启用VirtualBox NDIS6 Bridged Networking Driver服务。

如果路由器默认网关也是1.1，可以把路由器改到1.1以外。也可以在虚拟机修改：

vi /etc/config/network
#修改lan口地址后
service network restart

启动后，homelede默认网关192.168.1.1，账号root，密码password。

win+q搜索网络连接，查看网络连接（控制面板）-右键物理网卡，状态，查看当前路由器的网关地址。

如果路由器地址为1.1以外的地址，则修改电脑ip为192.168.1.2，网关为192.168.1.1，并在网页打开192.168.1.1 (opens new window)，登录后对passwall进行配置。

配置VPN-passwall：基本设置，打开主开关，保存应用。节点订阅-添加-订阅网址，备注随便打几个字，保存应用，手动订阅。之后在基本设置中配置tcp节点和udp节点。

配置完成后，将主机网关改为192.168.1.1，ip改为192.168.1.x（x为除了1以外且不重复的任意数）即可。
