# 穿墙利器mesh

  贵价路由的穿墙能力受限于物理规律与接收设备的功率，提升非常小。而mesh能显著提升覆盖范围，并且无缝漫游，无感知切换。

  mesh基于网络协议802.11s，只需要路由器支持。比较新的快速漫游由802.11k,v,r三个协议组成，需要手机支持。查看苹果关于这三个协议的介绍 (opens new window).

  组mesh更推荐wifi6路由器，普遍优化做的更好，协议支持度也更高。

  天邑TY300以及TY400降价到70左右，组mesh实在划算，不在乎二手首选。

  水星D19g够便宜，也能实现次节点200兆以及无感漫游，但是细节相对竞品略逊一筹。

  华为，小米是相对性价比高，体验也不错的方案。

由acwfi站长整理的路由器mesh协议支持情况 (opens new window)

  商业化固件刷机实现mesh相对便宜，又能获得不错的体验。比如高通方案刷openwrt。其中super-wrt 深耕高通方案的wifi驱动，其无线驱动稳定性明显高于开源驱动。二手的tp 941n等高通方案路由器一般才十几块。

  手头有斐讯k2，k2p等联发科方案矿渣的，可以刷集客AP或者高恪固件。

# mu-mimo

  mu-mimo兼容性暂时很差，除非未来各厂商达成共识，或者你所有的设备都是某家公司全家桶。一旦解决了兼容性问题，mu-mimo能显著改善多用户体验，让4×4产品有用武之地。但是现在4×4产品体验上没有拉开差距。

# 稳定，断流

几乎每款都有人说断流，所谓断流不可尽信

  最重要的必然是稳定，但是显然稳定性很难通过测试量化。每天定时重启一般能一定程度上避免不稳定。

# 干扰

  无线最大的问题是干扰，包括邻居的干扰和家里设备间干扰。解决设备间干扰可以考虑买两个路由器，一个跑在36-64信道上，另一个跑在149-161信道上，哪个信道干扰少就专门用来打游戏。2.4Ghz的干扰太多了，能用5Ghz尽量用5Ghz。

# 64，256，1024QAM提升多大？

  64QAM是11n标准，256QAM是wifi5（11ac）标准，wifi5只包括5g频段，部分芯片在2.4g上也支持256QAM。1024QAM是wifi6（11ax）标准，包括2.4g和5g。

  1024QAM（2^10）有10个二进制位。256QAM(2^8) 有8个二进制位，所以理论最大速率是1.25倍。
