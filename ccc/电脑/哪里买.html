# 整机，淘宝装机最大的问题

代理在找厂家进货的时候都一定要搭售一大堆卖不出去的辣鸡。

网红型号根本不愁卖。整机，装机用的配件都是大牌子在diy市场根本卖不出的辣鸡。如果没有整机，淘宝装机，基本上亏本贱卖都卖不出去。

超级小桀的整机，有华硕中国总经理tony站台，对接华硕一级代理，档位够高了吧，照样安排一水的辣鸡，这就是整机市场的天道，做整机生意不卖辣鸡根本无法生存。尘雨科技 (opens new window)的玄武电源，把利润砍掉一大半，其他品牌干脆直接不准代工厂给他代工。

整机并非完全一无是处，默频使用，当成普通的消费电子产品，两三年一换，也都是可以正常使用的。

非要买整机，可以看看这三家：

中正评测，经典整机路数，跟武汉三雄一个德性，但至少比友商便宜。支持自定义配件装机。

# 整机

警惕洋垃圾，特点是i7级，i9级，xx级类似这样的字眼，cpu以e5,e3为主，显卡矿卡。稳定性很差，除非开工作室多开。

普遍用料差，配件特供，寿命期望短，没有升级余量，超频余量，售后费劲。规模较大的店：武极，京天华盛，攀升。

整机是板卡返修良品的主要出货渠道。固态硬盘是白片黑片的主要出货渠道。内存也是白片黑片拆机片的主要出货渠道，不过内存能点亮一般没事，无非不能超频。

主板显卡本来就有“网吧专供”，基于零售型号阉割。缩供电，接口，RGB。型号通常加后缀，甚至有不改的，看BIOS才能看出来。这种主板实体店也会用。整机板卡还要进一步阉割供电和接口，用档次更低的接插件。

现代cpu和显卡，都有非常高的瞬时功耗，整机的缩水电源很容易顶不住。市网电压稍微有点波动，整机就容易掉电。

# 实体店

网上有价保，实体店进货更便宜，偏爱高利润配件，看人下菜。

# 网购配件哪买？

B站有很多日更搜罗优惠信息的，可以作为价格参考。

# cpu

通常很少坏，找便宜的，闲鱼，淘宝。

魔改,ES最好淘宝，好赖有店铺。魔改普遍稳定的不便宜，便宜的稳定性堪忧。而且卖二手的时候残值率低，我是觉得完全没有折腾的意义。

# 主板

京东自营蹲活动。

  • 主板故障率相对最高，京东售后舒服。那几个出货量大的板u套商家，售后主板有得等了。

  • 所有第三方商家的板u套都会混发良品和退货。

  • 自营活动价加散片CPU，同板U套价格相近。

  • 老主板不推荐二手，修过或电气性能衰减。不如考虑华南金牌，精粤等翻新，七彩虹映泰梅捷昂达等二三线，至少有售后。

# 显卡

以下品牌溢价少，性价比合理。

  • 铭瑄电竞之心，翔升。支持个人送保，一般柏能代工，返修率最低。
  • 同德丐卡推荐在影驰，耕升，铭瑄，盈通里找最便宜的，懒得找推荐耕升，品牌溢价低，而且基本就是同德公版丐，不会再缩水。
  • 电竞判客，栢能系（索泰，映众）最便宜。
  • 以上品牌的丐卡一般代理商送保居多，京东买售后方便。
  • 京东自营可以考虑PNY，比亚迪代工。

性价比低：索泰霹雳毁灭，七彩虹元气，战斧，网驰。

台湾品牌仗着品牌溢价，又贵又丐，甚至为了打价格战，低端系列大量启用矿卡作坊代工。

# 非公

  • 二线非公差价很小才考虑，大都不提升供电，就换个三风扇散热器，甚至往往是负作用。

索泰PGF，天启；七彩虹adoc；影驰星曜；微星魔龙；华硕ROG，KO等非公会升级供电，但是性价比很低。

  • 是否要非公建议看供电相数，drmos每相效率高，温度低的甜蜜点在30w左右。

xx70及以上卡，丐卡都给到10相以上，供电完全足够，非公堆供电，反而增大故障率期望。
