# 固态

更新于2023.4.4

固态跑分只说明爆发力，都有模拟SLC，参考价值非常有限。

挖过奇亚币的都知道，长江存储的原厂颗粒遥遥领先，不是倭国躬匠恺侠白片冒充原厂颗粒，以破坏客户数据为使命的闪迪，人为降低闪存寿命的三棒能碰瓷的。

致态5000，7100性能基本基本看作一个水平，无非7100跑分更好看，爆发性能更强。

致态PC005，7000pro这种带外置缓存的性能更高，满速容量60%。就算掉速了，缓外写入还是有2g左右。无外置缓存固态一般都是剩余容量20%的空间模拟SLC，写入超过开始掉速，1.5g左右，如果盘很脏，很快就掉到700m。

**系统盘最好买带外置缓存固态硬盘，无论原厂副厂。**windows系统盘就算啥也不干，光临时文件就是大量写入。

只要定期擦洗，程序盘用无缓存副厂盘，应该也是够用的。 不定期擦洗，程序盘买副厂盘最好带外置缓存。

关于擦洗请见后文-改善副厂无缓存固态的使用体验。

# 改善副厂无缓存固态的使用体验

固态硬盘健康度下降是坏块被备用块替换的正常现象，不为0就没事，但是主控读取时才知道好坏，超出纠错上限就会有“文件静默损坏”。

用于存储会有大量冷数据处于如好状态，出现静默损坏的概率大增。务必做好定期数据擦洗校验：

  • win11格式化ReFS，开启可选功能-完整性流 (opens new window)，配置任务计划程序Microsoft\Windows\数据完整性扫描（默认每四周，酌情减少间隔），或定期手动校验。通过校验提前完成纠错和冷数据预热。

  • NTFS定期用7zip对整个磁盘计算CRC32也能为冷数据纠错预热，但毕竟没有校验。

# 固态硬盘的外置缓存

可以一力降十会，FTL表，Bitmap表会更宽裕，GC触发更少，GC算法效率好得多，理论寿命更长。带外置缓存的固态最重要的是更低的售后期望。

# 副厂固态硬盘

正片颗粒暴毙概率很小。白片颗粒有较小概率暴毙，而拆机片，黑片，暴毙概率很大。

只要不是颗粒暴毙，即使出现掉盘，健康度下降，一般只要不误操作，并做好擦洗，数据完全损失概率较小。

如梵想，移速，海康，七彩虹，朗科，达墨，金士顿，acer宏碁等贴牌，代工厂自有品牌如阿斯加特，光威，金百达，江波龙雷克沙。

除了白片，还会混有拆机片，黑片。存在送测或前几批用好一点的颗粒，之后图穷匕见的现象。这几个牌子我估计就阿斯加特跟雷克沙因为定位相对高一点，可能会好一点。

截止23年8月，说自己是长江颗粒的好赖是白片。

副厂盘除了文件静默损坏概率大，还有可能读冷数据需要纠错，导致速度变慢，系统盘会导致系统响应很慢。

详细了解请参考【switch·路由器·手机·NAS 】-【truenas】 (opens new window)。

# 离线移动固态硬盘不能用于备份

固态硬盘的特性来看，明显离线移动固态硬盘只能作为大号u盘，进行临时性的文件转移，用作备份是相当危险的。更不要说固态硬盘需要进行GC，TRIM等操作。

我认为，合理的用于存储的固态硬盘盒必须是配电池的在线式简易nas。当下的产品只能用于诸如wintogo等用途。

因此有一个算一个，所有的移动固态硬盘，移动固态硬盘盒产品都是对用户数据极端不负责任。三星，WD闪迪，铠侠热衷于推出此类产品，巴不得用户遇到数据丢失的灾难。

已经爆发血的教训：WD闪迪Extreme移动固态硬盘大面积故障，更令人震惊的是，借着解决问题为由，推送新固件扩大战果，可以说WD闪迪，三星和铠侠三家劣迹企业中WD闪迪最不忘初心，以破坏用户数据为己任。
