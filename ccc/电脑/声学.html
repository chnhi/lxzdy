# 声学（耳机麦克风音箱）

# 耳机

推荐收录eq曲线 (opens new window) 的型号，其中的GraphicEQ.txt可以通过安卓的wavelet软件，windows的equalizer APO软件加载。

ParametricEQ.txt需要支持修改均衡器频率点和Q值的均衡器，FixedBandEQ则是经典的，驱动和音乐播放器都支持的10段式固定频率播放器，这个误差偏大，不推荐。

没有收录曲线也没事，后文介绍了如何根据频响曲线生成补偿EQ。

哈曼曲线的目标是使得耳机听感更接近平直曲线的音响听感。很多耳机的调音热衷于拔高低频，营造强烈的氛围感，以至于很多不明就里的听众会感觉这个feel倍儿棒。

平头塞会因为漏音导致低频测不准，需要辩证看待。

L7audio，解码耳放测评 (opens new window)

dac,adc芯片指标 (opens new window)

# 入门级，仅供参考

# 头戴

  • 得胜hd5800，闷热，原始曲线就非常贴近哈曼曲线。
  • 舒伯乐某些型号头梁是不容易掉皮的硬质材料。
  • 飞利浦shp9500，eq补偿后很贴近哈曼曲线，性价比较高。

头戴会掉皮，热门型号淘宝上都有替换耳罩，头梁掉皮无解。

# 平头

VE和尚(Venture Electronics Monk Plus)，30块，补偿后曲线不错。

相对原道的频响曲线要好很多。原道的生产稳定性较差，网友分享的频响曲线参考价值有限。

水月雨 太空漫游，Moondrop Space Travel。原厂就调的不错。

# 生成频响曲线数据.csv文件

吾等益耳 (opens new window)（只勾选SPL），B站等有很多耳机的频响曲线分享，

频响曲线的图下载之后，在WebPlotDigitizer (opens new window) 网站打开后，坐标系选2D(X-Y) Plot后，会要求选XY轴的起点和终点。完成后会要求输入XY轴各自的范围。

之后选中Add Point，在每一个横坐标的参考线与曲线交点处点一下。如20hz，30hz，40hz，因为频响曲线的横坐标不是等间距的。点完后，按View Data，Download .CSV。excel打开文件，改正X轴的数据。

# AutoEq生成补偿

把AutoEq的Github (opens new window) 文件git拉下来或在网页端下载，要用python3.10，3.12有函数被废除，项目作者23.12月测试没适配。解压后在项目目录下：

python -m pip install -U pip
#安装依赖：
python -m pip install -U -e . 
#验证安装，正常情况会输出提示，依赖安装有问题会报错。
python -m autoeq --help

生成曲线示例：

前文产生的csv文件，本例重命名为diy.csv，放在项目目录的measurements文件夹下。

python -m autoeq --input-file="measurements/diy.csv" --output-dir="my_results" --target="./targets/Harman over-ear 2013 without bass.csv" --convolution-eq --parametric-eq --ten-band-eq --fs=44100,48000
#--input-file既频响曲线生成的数据。
#--output-dir既生成的eq补偿放置位置，示例指令在项目目录的my_results文件夹下。
#--target既目标曲线，参考：
头戴式：./targets/Harman over-ear 2013 without bass.csv
入耳式：./targets/Harman over-ear 2018.csv

鸦雀有声的文章 (opens new window)中介绍，他在头戴式耳机上使用 harman OE(Overear,头戴式) 2013 without bassboost，在入耳式耳机上使用 harman OE 2018 with bassboost。 具体介绍见原文。

# 游戏耳机（耳麦）

通常游戏耳机eq补偿也拉不回他们的怪异调音，自带麦克风往往也很拉跨，我觉得不如搭配30块的ve和尚加eq补偿，用类似于droidcam的软件，手机当麦，效果反而比廉价麦好。

麦也可以买百来块的usb电容麦，至少比耳麦强。如二手的雷蛇·魔音海妖迷你版。

博雅M1领夹麦也可以考虑。

耳麦可以考虑手机用的无线耳机，通常有enc双麦降噪，麦克风效果相对好些。

漫步者hecate部分型号usb耳麦有enc降噪。

# DIY音响

推荐做一对3寸全频，结构简单，要求非常低，不需要调音，不需要分频，功放也不用调音电路。

  • 喇叭单元要求高： 皮亚力士PLS-P830986。有一定低频，高音像模像样，素质非常顶级。
  • 要求不高 曲泉3寸全频喇叭 (opens new window)。素质也不错，整体比上面这款差一点。但是配上55块的价格，我觉得物超所值。
  • 箱体 3寸全频铝合金音响 箱体 (opens new window)，面板开孔可能要定制。铝合金箱体 共振小，低频不散，不调音的影响相对其他材料要小很多。
  • 功放 MA12070 芯片的 数字功放 (opens new window)，100块不到，价廉物美，省电发热低，性能绰绰有余。电源用商家套餐也够用，要求高的可以买倍思的氮化镓充电头用诱骗线，纹波更小。

这种小全频的优势是 声场 定位 和结象，既所谓 脱箱感。

# 解码耳放

高性价比入门：海贝FD1，99块ES9118 DAC，相对小尾巴偏大，耗电大，稳定性差，高灵敏度入耳会有底噪，评论有很多反映要拔插两次才能识别。但同价位好像没有配置更高的。

以前100不到就能买到cs43131方案的小尾巴，现在死贵。可以考虑水月雨破晓，EPZ TP20。（水月雨设计有可取之处，EPZ是规模非常大的代工厂）

# 独立声卡·输出

输出时声卡端的失真通常很小，包括集成声卡，完全不需要独立声卡。

通常只有头戴式高阻低灵敏度耳机需要耳放，一些DIY的高阻平头，入耳，通常灵敏度并不低，不需要耳放。

高灵敏度耳机在集成主板上底噪实在太大无法接受，可以考虑用阻抗棒或购入解码耳放。

音响的底噪用阻抗棒是没用的，可以考虑spdif解码器(”光纤转3.5mm“比较好搜)。

  • 上文介绍的解码耳放就不错。也可以看看xmos 异步usb方案，diy的性价比不错。

# 独立声卡·输入

输入时，独立声卡主要提供低延迟的监听，ASIO驱动，或附加话放，幻象电源，混响等功能。

集成声卡在输入端参数差些，动态范围，TND相对录音声卡，大约有10~20db的差距。由于入门级话放价格同录音声卡差不多，相对声卡自带话放性能也没什么提升，所以通常不如买入门级声卡。

  • 百灵达调音台产品性价比较高。
  • 百灵达UMC22，只录音可以用。
  • 尽量考虑更高价位的福克斯特SOLO3。
  • 以上两款搭配电容麦能用，搭配动圈麦最好再加个小话放，如爱克创MA1，SE DM1。
  • 电容麦必须在房间里贴上吸音棉，不然环境音会非常明显。
  • 此类产品通常是商用，溢价率非常高。福克斯特，雅马哈，玛雅，罗兰等品牌相关产品请自行搜索。 更高价位请参考Jason任的文章 (opens new window)

如果录音底噪很大，可以考虑使用waves-x-noise降噪插件类似方法降噪。

# 音响

  • 百灵达B2030A，更贵的音响只有品牌溢价。
  • 漫步者R2000DB。
  • 二手漫步者2.0随便什么型号的2.0音响
  • 如果追求KTV的压迫感，淘宝上几百块的10寸12寸甚至更大的音响随便买。功放买"伟良音响DIY"的大功率D类功放。

# 家庭影院·环绕声

自己折腾最好是弄4.0方案，两个前音响，两个后置的环绕音响。环绕效果就非常不错，对调音等各方面要求也低。

  • 入门级，随便买成套的5.1方案。不启用中置音箱，组成4.1系统，环绕效果相对更好。（输出设置成5.1系统不启用中置音响，播放器混响5.1输出）
  • DIY方案，按前文介绍的diy四只3寸全频，组成四声道系统，eq的极低频加点增益，补偿全频喇叭。

5.1及更高方案对摆放要求较高，自己折腾可以参考回答摆放 (opens new window)。音响最好买成套方案，最好还是找提供上门调音服务的产品。
