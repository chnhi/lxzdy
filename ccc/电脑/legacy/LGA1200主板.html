# INTEL-LGA1200主板(10,11代)

更新于2021.2.18

LGA1200接口支持10代，11代CPU。如i3-10100，i5-10400f。11代cpu通常只有h510,b560,h570,z490,z590主板支持。
11代相对10代进行了架构大改，购买ES处理器要谨慎。

# 芯片组区别

  Z490，H470在搭配11代处理器的时候DMI总线是X8，搭配10代处理器的时候是X4。B460，H410英特尔确认不支持11代。Z490，H470主板全部支持11代处理器，但个别主板的m.2不支持PCI-E4.0。

  3.0x8总线足以应付3根走南桥的3.0x4固态同时满速。3.0x4只能应付1根半。

  技嘉B460M DS3H V2；H410M DS2V V2和H410M S2H V2由于实际采用的芯片组是H470，所以支持11代处理器。

  接下来的12代即将更换接口LGA1700并引入大小核，快的话能赶上参加今年双十一。

# 500系主板

  微星的b560第一根m.2搭配10代或者11代i3是可以用的，缺点是第二根pcie槽和这个m.2不能共用。而除微星以外的b560搭配10代或者11代i3是不能用第一根m.2的。搭配11代i5及以上处理器都是一样的cpu直出。

  • b560 tomahawk 有外置pcie时钟，可以超外频。z系列从z170开始，cpu外频和pcie频率已经脱钩。
  • B560搭载10代处理器也可以超内存。

# z590

msi-unify 可以讲显卡的pcie 4.0 x16切割成x8+x4+x4，另外的两个x4给m.2。

# B460-十核4.4-主流性能王

  传闻B560将开放内存超频，买B460前要斟酌一下。

  • 微星 MAG B460M MORTAR迫击炮 ￥674（京东史低）

6层菊花链 6相2上2下+1相1上2下+7外置驱动芯片+8pin双散热 2.5G网卡（8125B） ALC1200 mATX

  只有微星B460默认PL1/PL2是255W的b460，PRIME95非K十核全核4.45G，八核全核4.6G，友商默认65wPL1，满载频率严重受限。尽管可以自行修改，但是友商竞品的用料只能支撑125W，硬开255Wmos会过热降频甚至死机。注意微星B460 Bazooka 也是135W水平。

  ROG-F能撑到255W，但是太贵，性价比太低。期待一手华硕的b560，TUF b560应该会绝地反击。

  • 七彩虹CVN B460M GAMING PRO V20 ￥569（京东史低）

6+4Dr.mos+8pin双散热 1G网卡 mATX

  手动调整PL1，在BIOS里也有可能叫cTDP。供电用料理论上是完全可以跑255W的PL1的。

  • 十核3.9 映泰 B460GTA￥648（京东史低）

6相1上2下+2相1上2下+8pin双散热 1G网卡 ALC1150 ATX

  手动调整125wPL1，供电满足六核非K，八核十核全核3.9g。大板备选方案，跑125W的主板里算供电不错且相对便宜的。

# B460-十核3.9-便宜够用

  • 铭瑄 MS-iCraft B460M-F 电竞之心￥619（京东史低）

6相1上2下+1相2上2下 1G网卡 2mimo网卡AX200 mATX

  背板带有一键调整PL1为125W的按钮，可以不在BIOS修改，供电满足六核非K，八核十核全核3.9g。便宜就是硬道理，不过H410更便宜哎。

  • 微星 B460M PRO-VDH WIFI￥579（京东史低）

4相2上2下+2相1上2下 1G网卡 mATX

  默认PL135W，供电满足六核非K，八核全核4g十核全核3.9g。便宜就是硬道理，不过H410更便宜哎。

  • 七彩虹BATTLE-AX B460M-HD PRO V20￥529（京东史低）

4+3相dualFET+8pin裸奔 1G网卡 mATX

  裸奔B460里用料最好的，但毕竟是裸奔的，最好只带到六核。同版型的H410M-PT PRO V20，裸奔里用料好的，但是到处无货。

# ITX

  • 七彩虹CVN B460I GAMING V20￥699（京东史低）

4+2相Dr.mos+8pin裸奔

  有点中庸，供电比上不足比下有余，只能说跑125W但是会比较低温吧，ITX空间紧凑，散热不好，供电还是要好一点的。

  • 映泰 Z490GTN￥999

6+2相Dr.mos+8pin单散热

  性价比比较高的一块主板。非K八核十核火力全开。

  • 华擎 Z490M-ITX￥1359（京东史低）

5+2相Dr.mos+8pin单散热

  非K八核十核火力全开。支持11代，不需要内存超频的话，同版型的H470京东史低879，划算多了。

  • 昂达 H410SD4-ITX

3相2上2下+2相1上1下+8pin裸奔

  最便宜的ITX，除了便宜一无是处。六核顶天。最好要用下压式散热器。

  • 微星（MSI）MEG Z490i UNIFY

10层2DIMM 8相直出Dr.mos+2Dr.mos+8pin双散热 IO装甲

  买顶级Z490 ITX的目的是什么？那肯定是内存超频啊。挑战巅峰 打破记录，就选微星。当然了，技嘉，华硕的Z490I 基本也都是相近的物料水平。

# Z490

  • 七彩虹Z490AK GAMING V20￥899

8+2相dual N(SM7341EHKP)+8+4pin单散热

最便宜的八核5g，供电温度偏高。要注意pcie4.0支持度和支持11代BIOS什么时候能放出。除此之外，映泰GTN也值得参考，缺点就是ITX主板很拥挤，散热片也偏小。

由于Z490已经进入生涯末期，要注意pcie4.0支持度和支持11代BIOS的情况。

如果不追求5G以上频率，入门级z490御三家PRIME Z490M-PLUS相对比较推荐。
