# AMD-AM4主板

更新于2021.2.8

# AM4芯片组区别

  • A520比B550少了1个USB 3gen2，不支持双显卡和CPU倍频调节。

  • A320，A520都是开放内存超频和PBO的。

X570 B550 A520 B450
m.2(cpu)和显卡x16 PCIe4 PCIe4 PCIe3 PCIe3
第二条M.2(南桥) PCIe4 PCIe3 - 3.0x2或2.0x4

  b550第二根m.2能3.0满速，b450第二根m.2一般是3.0x2或者2.0x4，视主板而定。x570各主板配置不同，但至少两个满速4.0m.2。

  B550出的晚，普遍供电强于B450，同价位强于X570。

  目前微星华硕技嘉华擎映泰B450,X470支持ZEN3的BIOS基本都已经发布，华擎的X370/B350支持ZEN3的BETA BIOS正在路上；华擎3系和4系主板的bios是通刷的，直接刷就行了。七彩虹还在打磨Zen2。（zen3既5000系cpu）

  • 铭瑄B550M电竞之心等降价，目前￥1099

8层菊花链 背板清空BIOS+盲刷 mATX

  宣传7+1倍相14+2。但ir35201是6+2控制芯片，情况不明，也不知道mos管是什么，不确定供电水平究竟如何。看起来不错，可能至少迫击炮重炮手以上水平。但是铭瑄很喜欢把12V电感混在cpu供电里装样子。网上也没有拆解，情况不明。磁吸天线细节不错。 背板装甲 数显侦错 板载开机按钮这一般都是旗舰主板才会为裸架超频，液氮超频配备的。期待铭瑄后续能出超频记录。

  大陆主板品牌向中高端进发的努力值得鼓励，赶超映泰技嘉指日可待。但是要经营一个中高端品牌就更应该高标准严要求，类似8层板吹8oz这样的擦边球随处可见。文案中也没有抓住重点，5950X能跑高频，mos温度呢？

# a320,a520-8核默频--便宜够用

  • 映泰 (BIOSTAR)A32M2 ￥299 4相1上2下+2相1上2下8pin裸奔
    • 昂达（ONDA）A320S全固版￥289(淘宝) 4pin裸奔

2DIMM mATX

  入门级a320主板当中唯二供电最好的主板。下压式散热器加良好的风道满足8核开PBO，但裸奔mos温度会偏高，昂达比映泰便宜几十。昂达的cpu供电是4pin接口，一般4pin标准规范极限是192W，3700X够了。

a520目前性价比不高。不如购买主流价位b450。
  • 七彩虹BATTLE-AX A520M-M.2 PRO V14降价到￥349-￥399可以考虑
  • 微星A520M A PRO降价到￥349-￥399可以考虑

4相1上2下+2相2上2下8pin裸奔 2DIMM

  a520主板当中供电比较理想的，期待降价。满足5600X开PBO，下压式散热器加良好的风道满足5800X默频。但裸奔mos温度会偏高。

  • 微星A520M VECTOR WIFI降价到￥400出头可以考虑。

# 旗舰与次旗舰

除了映泰，旗舰都能5950X液氮，次旗舰5900X液氮。

  • 映泰x570gt8全套IR方案，5950X默频。

  • 技嘉 b550AORUS MASTER和x570 Aorus Xtreme cpu核心供电原生14相。

  • msi次旗舰unify朴实，ace炫,unify-X只有俩内存插槽，内存超频会更猛，unify,ace都是6层板。

  • 华硕次旗舰STRIX b550/X570-E，STRIX F不要买，情愿加点买STRIX E。

  • 一般次旗舰6层板，旗舰8层板。微星旗舰godlike，Creation，华硕旗舰c8h，c8f。
