# 小品牌

​ 显示器有三个部分组成：

  • 液晶模组（液晶玻璃加背光）
  • 驱动板
  • 模具

​ 通常大部分耳熟能详的品牌比如AOC冠捷，只进行了模具生产和组装。因此购买同样使用原厂模组的小厂生产的显示器，不仅在显示效果上不会有任何区别，而且价格实惠。

​ 推荐两个品牌，他们在社群当中有一定曝光度，同时承诺保证原厂模组+性能不妥协的驱动板。面板型号和驱动板型号向老板咨询的话会说的很清楚。即使不打算购买小品牌，也可以通过他们在售产品了解到目前最新最流行的面板是什么。

  • IC谷星 天猫柒和数码专营店 (opens new window) 店主老牛微博 (opens new window) 坦然数码科技 (opens new window)
  • 宏夏 宏夏淘宝店 (opens new window)

  IC谷星斥重资买了光度计校色，为消费者提供付费高精度校色服务，在微博上也经常批判显示器友商的营销话术。宏夏主打优化游戏固件，追求极限，也提供付费色域限缩。

  散屏及diy显示器虽然更加便宜，但是有用第三方模组的（原厂液晶玻璃+比较烂的背光），还有卖测试模组的，质量参差不齐，风险较大。

  这两家显示器的售后体验存在一些纠葛，换货换到完美屏为止肯定是不可能了，且对于包无亮点显得非常扭捏，我怀疑他们的面板亮点数恐怕是高于平均水平的。但是便宜就是硬道理。

# 大品牌

​ 大牌当中同一片面板

  • acer宏碁一般最便宜，别人如果比acer还便宜就要小心是不是第三方封装，都是冠捷代工。
  • AOC冠捷收购了飞利浦，这两家比宏碁稍贵，冠捷自建了模组封装厂，要小心他们用自封装模组降低成本。
  • 优派最喜欢用第三方模组打价格战，敬而远之。
  • 华硕，外星人贵在品牌溢价上，贵的没道理。高贵的华硕如果出了平价产品那就要当心了，9成9是第三方封装的模组。
  • 微星路子很野，产品线混乱不堪，有原厂模组有第三方模组。
  • 小米显示器每台显示器都有出厂校色报告，做好色域限缩。但是小米很喜欢用第三方模组，在社群当中他们的解释是原厂模组又贵又不如小米的第三方模组，小米找第三方模组是为了做的比原厂模组更好。
  • 惠科，游戏悍将，联合创新等随着三星停产LCD，应该也会做IPS电竞屏。

# 原厂模组VS第三方模组

  社群中普遍认为原厂模组比第三方模组好，从色域和实测显示参数上确实存在差距，第三方模组普遍都是拉了跨的。我认为这同传统厂商只是代工组装厂有很大的关系，他们费劲向产业链上游发展的目的只会是降低成本，赚更多的钱。但如果贴合一样的背光板，原厂模组同第三方模组为什么会有区别呢？

# 刷新率

  刷新率（如144hz）只和面板用的刷新模块有关，而肉眼可见的拖影只和面板有本身固有的响应延迟有关。只有当面板本身固有的延迟很低的时候，更高刷新率才有意义。

# 广色域

  广色域显示器和电视机已经非常常见，色彩管理这个话题也提上了议程，色影无忌 (opens new window)论坛上有一些讨论。

  • 色域覆盖100%srgb的非广色域显示器，在windows默认设置下就能获得非常不错的色彩表现，不需要折腾。

  • 广色域显示器就会面临映射空间不对，显示过饱和的问题。理论上是很好解决的问题，而windows的色彩管理做的又比较一般，只支持色温和伽马曲线，和很多软件的配合上也有问题。

  • 专业显示器校色后处理参照色影无忌网友何不同的帖子 (opens new window)

  • 游戏显示器，如果不校色，推荐肉眼收货，觉得默认状态看着很舒服就正常使用，觉得太艳了，用显示器上的按钮调整为srgb模式。如果进行校色，全屏应用（切屏会黑一会）是不支持色彩管理的，反倒不如显示器主控改成srgb模式。

# 面板

# IPS

  不二之选，无论是设计，游戏又或是日常使用，色准，色域，可视角度，响应时间相对mva和tn都有提升，相对更新的OLED，micro led又有成本优势。

# MVA

  MVA优点是暗部细节更好，对比度高，但是响应时间非常长。电竞屏加压之后响应时间还是很长，灰阶变烂，反而优势没了。MVA大屏电视配合棋盘控光，播放hdr资源相对IPS大屏电视有明显优势。

# TN

  FPS核心玩家必备。友达新TN色彩各方面已经比以前的tn好很多，但还是会明显泛白，可视角度有很大的改善，但和IPS比还是略逊一筹。

# OLED

  自发光有机物，高亮度，高对比度，延迟比tn还低，缺点是成本高，有烧屏现象，目前已经在中高端笔记本当中逐渐推广，进入显示器市场指日可待。

# MICRO LED

  自发光半导体，除了有OLED的优势外，延迟更低，功耗更低，更耐高温或低温，使用寿命更长。成本暂时比OLED还高。

# 背光

# WLED

  最普通的LED背光，在蓝光LED上覆盖荧光粉变成白光，再透过滤色镜变成红蓝绿三色光。最廉价，蓝光能量最大。一般覆盖srgb100%就顶天了，做不到广色域。

# KSF LED

  在LED的基础上用改进型的荧光粉，Lg nano IPS采用这种背光方式。覆盖95%的DCI-P3色域。

# RGB LED

  很少见，只会在专业显示器出现，价格非常贵，分别用三色led灯珠发出三色光。

# GB-r LED，RB-g LED

  分别把RGB LED中的红色LED或者绿色LED替换成了红色或者绿色的荧光粉。其中GB-r相对rgb还存在红光范围过宽的缺点，RB-g从光谱上看几乎没什么缺点。一般只有对色准要求特别高的时候才会选择，比如设计用途。

# QLED

  用量子点增强膜代替荧光粉，实现广色域的不二之选。

# Mini Led

  micro led演进路线上点距微缩的还不够做显示屏，但是可以做背光，广色域，对比度高，还有精细控光。
