# 主板

# 品牌

没有特殊要求，也不追求极限超频的，也懒得研究，微星迫击炮，华硕重炮手，铭瑄电竞之心（堆料性价比较高）。这三款主板一般绰绰有余，一般都能给足200w以上的PL1供电。

  • 微星华硕，硬件互相跟进，实力相近。
  • 微星华硕针对不同内存颗粒，会自己测一套小参，不会完全沿用公版规范，效能更好。
  • 铭瑄映泰旗舰主打堆料，公版bios，无功无过，清仓拿下，性价比拉满。
  • 台独技嘉，台湾制造=粗制滥造，喜欢营销噱头，技嘉团队只做z主板，别的主板就是通路贴牌，买到就是上当。
  • 华擎花样多，23年国内市场换帅，不知道会不会有大动作，可以看看。

   微星Toppc (opens new window) 频道介绍了很多主板细节。通路厂沿用公版bios问题在于，公版是六层pcb，而中低端主板是四层板，要测试团队对预设值进行调校；产品的稳定基本都是因为测试团队提前踩过坑，这也是大厂主板的优势。

微星在一些设定上会偏保守些，在一定程度上提升了兼容性。华硕很喜欢瞎勾八改名，比如把XMP改名成AI DOCP，跟深度学习没有半毛钱关系硬靠。AI optimized听起来特牛逼，实际上就是给你改成定压定频。

华硕路子很野，x3d烧毁归根结底就是华硕一直在偷偷加压，但是毕竟是最大的板厂，没人敢得罪，没人敢说。

虽然我对华硕观感很差，但也会尊重客观事实，比如华硕b450M PRO-S重炮手，8相drmos Core供电，用料就是很好，我也会推荐。

  通路厂，换架构初代尽量不要买，比如amd b350通路主板别买，等他出马甲了，比如amd b450，踩过坑了才考虑。

通路厂里铭瑄（马甲梅捷）映泰设计相对合理，七彩虹ODM方案太多，层次不齐。昂达影驰便宜。

# 高端主板有什么可感知的提升？

  • 更厚的板层，内存超频更猛。
  • 超频贴心设计，主板集成开关机按钮，液氮模式。
  • 供电部分温度会低很多。
  • 用料更高端。如alc1220集成了数字高通和低通滤波器，参数超过部分独立声卡。
  • PCIE切分

  中端主板目前供电越堆越猛，也有六层板，只要挑对型号，绰绰有余。

# 简易判断主板供电好坏

  数电感有偏差，华擎技嘉会装一堆电感虚张声势。看拆机数MOS就行。drmos推荐每相供电工作在25~30w。上下桥分体的mos。每路下桥推荐15w。

​ 以下方法有效改善供电发热：

  • 主板供电有散热片
  • 改善机箱风道
  • 下压式cpu散热器

# 华硕强化并联VS倍相器倍相，被忽略的驱动IC

pwm控制芯片和mos管之间还需要驱动电路将PWM脉冲放大到足以驱动功率晶体管。驱动电路虽然发热比较小，但也还是有的。

  • 上下桥分离mos：驱动电路集成于pwm控制芯片或倍相芯片
  • 上下桥集成mos：驱动电路集成于pwm控制芯片或倍相芯片
  • 一体式mos（drmos）：驱动电路集成于mos管

drmos用不用倍相芯片已经无所谓了。

内存走线目前主要分为 菊花链（daisy chain）和T型拓扑（T-type）。 家用平台4个内存槽的 菊花链插两根离cpu远的槽，内存超频性能最强。T型拓扑（T-type）插满四根的话比菊花链略强，但是插两根极烂。这种强弱的体现有可能在于频率一样，但是小参可以拉的更紧。ddr4要到4000+频率走线上的差异开始变得明显。 AMD x570公版指导方案是菊花链走线。英特尔公版指导方案还是T型拓扑。

X570和B550三大厂都是菊花链，只有华擎是T型拓扑，但是华擎X570phantom x,creator和aqua是菊花链，B550情况不明，可以自行看板子和华擎以前的X470，B450长的是不是一样的，长的一样就是沿用老的Layout还是T型拓扑，X570之后新开案的才是菊花链。映泰七彩虹那些奇奇怪怪的我推测应该是公版方案菊花链。

x470微星，c8h,prime pro是菊花链，其余基本t拓扑。b450技嘉是t拓扑，微星可能是菊花链，华硕可能是t拓扑。 z370包括z370之前基本都是t type，z370疑似华擎是菊花链，z390开始微星是菊花链，z490疑似三大厂都是菊花链。
