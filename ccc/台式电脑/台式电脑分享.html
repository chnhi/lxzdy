# 组装，整机，实体店？

# 自选配件自行装机

  装机教程视频推荐：远古时代装机猿的装机教程 (opens new window)

优点：按我推荐的买，同价位最优解。或者多看看分享，得出最适合自己的方案。相同价格买到的产品有更高的超频潜力和更好的用料，默认性能倒未必更强。需要性价比也可以追求性价比拉满。

缺点：麻烦，折腾。如果全部上京东买，价格实际上是比较高的，除非等京东活动或者淘宝闲鱼拼多多买，详情参见后文《网购各配件去哪买呢？》

# 花装机服务费

  自行网购配件后买京东148块上门装机 (opens new window)或者找附近的电脑店组装。

缺点：额外付出装机费，附近电脑店有掉包可能性。

# 品牌整机

  警惕洋垃圾，特点是i7级，i9级，xx级类似这样的字眼，cpu以e5,e3为主，显卡基本都是矿卡。稳定性差，经常出现小毛小病。除非开工作室多开。

  武极，京天华盛，攀升等，板卡元器件都用便宜的，基本没有升级空间，设计余量小，元器件基本都是满负荷运作，可见寿命期望也短。包括联想惠普等，当然大牌子售后也许好一点。

  整机价格看起来很划算，但他们用料全是最廉价的货色，如果用这种丐中丐进行组装，很轻松就能比他们还便宜。当然贵价的联想惠普戴尔工作站一般都是好货。但不可否认的是，这些整机都是有售后的，出问题不用自己排摸。

# 淘宝店装机

  鱼龙混杂，他们就是人人喊打的实体店。

  考虑到很多人懒得自行组装，或者懒得了解，所以我找了几家还算凑合的淘宝店整机推荐 (opens new window)，但最推荐的还是自己买自己装，虽然我找的是淘宝店装机里还算不错的，但还是改变不了他们的本性，偏爱利润率更高的配件。

# 实体店

  京东天猫等平台有价保，实体店能拿到比线上更优惠的价格，但商家偏爱利润率更高的配件，还会看人下菜。

# 网络平台和实体店价格

  大家都知道在网上查价格，所以京东天猫平常标价虚高，给实体店留出利润空间，网上一个月能办七八次活动，这些所谓活动价才是网络日常售价。

  就算卖一样的价格，实体店向来花样百出，有的是办法坑你，何必去与人斗呢？

# 杂谈

# 主机

除非是玩电竞游戏，否则ps5或者xbox series x的性价比很高。二手ps4,xbox one大约在1000左右，但是搭配的cpu太烂，cpu参与的游戏运行效能不行，比如mc就会卡。而ps5或者xbox series x整体性能都非常强劲，并且搭配硬件光追，性价比非常可观，毕竟比同性能显卡贵不了多少。

而大屏电视如今也毕竟便宜，比如广色域4k 55寸的荣耀智慧屏x1大概2499，性价比很突出。如果不在乎显示效果，非广色域的55寸电视机1500都能买到。而更贵的索尼x9000h，x9500g电视机具有棋盘式分区控光，对hdr游戏有更好的显示效果。电视机选购参考蓝大仙人的推荐 (opens new window)

# amd

amd在cpu和显卡市场上处于老二的位置，所以性价比会高一些，同时超频策略也非常开放，但经常有对稳定性的抱怨出现，我认为这与开放超频相关联。四相供电的b450m主板超频的用户不在少数，各种杂牌内存也都在乱超。

amd的显卡商有蓝宝石，撼讯，迪兰恒进，讯景，盈通，华擎，华硕，微星，技嘉，都是一二线品牌。nvidia旗下鱼龙混杂，个别品牌为了打价格战，1660ti 竟然有3+1相供电的丐中丐，而盈通rx5500丐中丐都是6+1相。
