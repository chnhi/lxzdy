    # 网购各配件去哪买呢？

    支持个人送保和故障率很低的产品价格优先，除此之外的产品在京东上用更高的价格买相对省心的售后。注意看清楚是京东发货京东提供售后还是供应商提供售后，如果是供应商提供售后，还不如上淘宝天猫买，还便宜。

    # cpu

      cpu通常很少坏，怎么便宜怎么来，闲鱼，淘宝散片安排上。盒装经常有闲鱼淘宝拼多多开车，那些基本都是实体店的。

      魔改,ES cpu最好上淘宝买，好赖有个店铺。魔改的通常普遍稳定的不便宜，特别便宜的稳定性不怎么样。搁闲鱼上卖的有暗病的概率大一点，最好上淘宝买。

    # 主板

      我推荐主板等京东有活动的时候买。同时主板三大厂都是代理商送保，微星支持个人送保，淘宝上卖板u套装或者主板的基本都是各地电脑城的实体店，微星主板怎么便宜怎么来也是可选项。

    • 主板故障率相对最高，上京东售后舒服。

    • 淘宝天猫甚至京东的板u套装混着良品和退货，京东活动价买主板搭配散片cpu价格跟板u套装差不多。

      老旧主板不要碰二手，不是修过就是电气性能差不多了，不如考虑华南金牌之类的翻新主板或者七彩虹，映泰梅捷昂达之类的二三线主板，好赖有售后。

    # 显卡

    • 七彩虹，华硕，盈通，铭瑄，翔升支持个人送保，怎么便宜怎么买，淘宝拼多多买起来。

    • 影驰耕升最便宜的丐卡代理商送保，其余支持个人送保。

    • 其余品牌基本都是代理商送保，代理商送保的上京东买售后方便。

    • 矿卡二手卡尽量上淘宝买，好赖有个店铺。闲鱼现在完全被倒爷和商家占领，真正的个人卖家好价经常被秒，剩下的不是传家宝就是要划半天才能找到，还担心那卡到底行不行。

    # 内存固态

    影驰，雷克沙，威刚，wd，七彩虹，金士顿等品牌线下铺货让利多，所以淘宝能找到更便宜的，除了金士顿别的牌子一般没假的。

    # 机箱

    淘宝天猫拼多多便宜一点，没有标配风扇的一般都有加风扇套餐，京东贵一点售后方便一点。机箱快递费贵，万一要售后京东多舒服。

    # 电源

    整机店里喜欢用的那些都是线下铺货利润率很高的货，所以京东上会显得特别贵。我推荐的电源一般京东比实体店贵不了多少，还不如上京东买，随便用个什么券就跟他们价格差不多。
