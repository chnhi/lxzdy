# 台式电脑超频

# PBO与倍频定压超频

由wjm47196翻译的gamers nexus的 PBO与AUTO OC相关 (opens new window)

PBO能获取更高的单核心频率，但是全核心频率比倍频定压低，且电压给的更高，发热功耗更大。

倍频定压超频能获取一个更高的全核心频率和相对更低电压。

# AMD内存相关

​ ryzen内存超频一定要保证flck频率:内存频率是1:2，这样才能获得最佳的性能。

通常3000系fclk在1800，5000系在2100左右，这是一个大部分U都能超到的频率，对应的内存频率甜蜜点是ddr4-3600和ddr4-4200。

​ 尽管原则上是尽可能放宽时序尽可能让频率跑到甜蜜点，但如果内存体质有限，频率确实上不去的话，适当的降低频率压时序也能提升性能。时序放的很宽无脑提升内存频率有可能实际性能反而更差。

# AMD 自动降压超频工具

   ryzen在跑多核心重度负载如渲染，编解码，计算等应用时，PBO自动电压偏高，全核频率偏低。一般考虑定压定频。软件clock tuner for ryzen可以对体质进行测试，并找到一个电压尽可能低频率尽可能高的稳定过测点，获得比PBO更高的全核性能和更低的功耗发热。在CCX体质不同时，体质更好的ccx频率可以拉的更高。

   ctr2加入了hybird oc功能，可以针对单多线程不同工况得出两个配置自动切换，一个用于全核重度负载，另一个用于单线程负载时尽可能拉高频率。

KENNY肯尼酷制作的教程 (opens new window)bios设置的cppc在advanced/amd cbs/nbio/smu common options当中，软件作者的教程是不开启防掉压的，开启防掉压主板会加压导致测试结果得出的电压偏低，但从某种意义上来说，这样得到的电压反而有更低的待机功耗。如果体质不行，不开防掉压连测试都做不了的话，还是可以考虑开启的。通常微星主板开启防掉压3级，华硕主板3级或者4级。

# 内存超频入门调参顺序

  通常内存超频在参考网友的推荐，或者微星主板的memory try it 定下一个比较保守的值就差不多，超与不超大约是60分到85分，寻找频率极限可能是从85分到95分，压小参可能是95分到99分，却要花费大量时间测试是否稳定。

  但很多时候，为了追求性价比，很多用户买的可能是普条，那他可能就没作业可抄，需要自己摸索一个合适的频率和小参，下面这个调参的顺序仅供参考。

  1. 要注意温度在超频当中是不可忽视的影响因素，温度越高，越有可能会出错，所以有可能冬天稳定的频率小参到了夏天就不稳定了，要注意季节因素。
  2. 先确定内存耐压，调用XMP，但是电压尽可能拉高，1.5V起往上拉，跑memtest出错为止，要是1.5v就出错了就往下拉，拉到不出错为止，这个过程可以下一个主板厂商附赠的软件，在windows下调内存电压。确定内存的极限电压。
  3. 电压拉到不出错的极限电压，通常我会拉到极限电压低一档。频率拉到心动频率，通常我时序会设自动，也可以设22-24-24-44cr2。amd fclk先别超，跑memtest一直到能跑过为止再开始压小参，这里不行的基本是颗粒或内存控制器不行，没必要挣扎直接降频率。
  4. amd 锐龙平台还要超fclk频率，通常5000系可以考虑先设置SOC电压1.25V，VDDP1.2V，VDDG CCD1.1V，VDDG IOD 1.1。然后在windows里用aida64测内存效能，用主板附赠的软件一点一点往下调，调到内存效能突然变低了，就说明不能再低了。用testmem5测试fclk频率能不能稳住，放视频看会不会破音。这几个电压尽量找到一个稳定也更低的点，这几个电压拉太高反而不稳定。
  5. 保证上面四步稳定过测，压小参直接先拉tCL，别的先别动。tCL提升是最大的。找到tCL极限之后，别的小参不动其实也无所谓了。要注意压小参的时候还要参考aida64测试内存效能有没有提升，有时候压的太狠反而效能下降。
  6. 再压小参就是一点一点的试，我一般不会压的那么紧，没什么经验。有一个工具Ryzen DRAM Calculator可以帮助计算小参。
