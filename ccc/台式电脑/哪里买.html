# 整机

警惕洋垃圾，特点是i7级，i9级，xx级类似这样的字眼，cpu以e5,e3为主，显卡矿卡。稳定性很差，除非开工作室多开。

普遍用料差，配件特供，寿命期望短，没有升级余量，超频余量，售后费劲。规模较大的店：武极，京天华盛，攀升。如果把整机当成手机，用个一两年就换，凑合可以用用。

整机是板卡返修良品的主要出货渠道。固态硬盘是白片黑片的主要出货渠道。内存也是白片黑片拆机片的主要出货渠道，不过内存能点亮一般没事，无非不能超频。

主板显卡本来就有“网吧专供”，基于零售型号阉割。缩供电，接口，RGB。型号通常加后缀，甚至有不改的，看BIOS才能看出来。这种主板实体店也会用。整机板卡还要进一步阉割供电和接口，用档次更低的接插件。

现代cpu和显卡，都有非常高的瞬时功耗，整机的缩水电源很容易顶不住。市网电压稍微有点波动，整机就容易掉电。

# 淘宝店装机

鱼龙混杂，一般都经营实体店，偏爱高利润率配件。

淘宝摩西电脑 (opens new window)较合理，偏贵。

装机猿合作·淘宝西安水元素 (opens new window)较合理，主板最好换微星迫击炮。

# 实体店

网上有价保，实体店进货更便宜，偏爱高利润配件，看人下菜。

# 主板

京东自营蹲活动。

  • 主板故障率相对最高，京东售后舒服。

  • 淘宝天猫京东的板u套混发良品和退货。

  • 自营活动价加散片CPU，同板U套价格相近。

  • 老主板不推荐二手，修过或电气性能衰减。不如考虑华南金牌，精粤等翻新，七彩虹映泰梅捷昂达等二三线，至少有售后。

# 显卡

  • 七彩虹，盈通，铭瑄，翔升支持个人送保，找便宜的。
  • 影驰耕升丐卡代理商送保，其余支持个人送保。
  • 其余品牌基本代理商送保，上京东买售后方便。
