# 组装推荐

# 主流

  • 板U套 5600X（散片）+微星b550 mortar（京东自营）

  • 散热器 雅俊b3配防拔出支架。追求性能利民pa120，闲鱼有便宜新货。

  • 内存 ddr4 4000+的，C9BLM。ddr4 3600就好的，长鑫A-die。摸奖，十铨3200的马甲，十层板，tt灯条，十层板；科赋英睿达马甲条。

  • 硬盘首推光威 弈Pro和致钛，长江原厂颗粒。便宜的RC10，性能比SN550强。

  • 机箱 先马，小机箱平头哥M7 Mini；大机箱鲁班1，3。静音黑洞，静音和散热不可兼得。

  • 电源鑫谷金牌直出，或闲鱼矿龙。

# 入门配置

  • CPU 200ge (255)

  • 散热器超频三七星瓢虫V4（39）

  • 主板 映泰 (BIOSTAR)A32M2（299）

    • 昂达（ONDA）A320S全固版（289）

  • 内存 威士奇DDR4 4Gx2 （99x2）

  • 显卡 集成

  • 硬盘致钛 256g M.2接口 PC005 Active系列（京东史低249）

  • 电源首席玩家 火玫瑰400 （京东史低119）

  • 机箱游戏帝国 寂静岭（京东99）

      最好加一把机箱风扇在主板接口旁边那个风扇位，对散热改善很大。

      电源同价位有牌子更大但是是被动式的鑫谷s7，也可以考虑加钱换航嘉450b，以后能加XX50显卡。
