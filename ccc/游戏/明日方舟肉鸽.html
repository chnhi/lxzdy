# 界园

二：洞里【传说】30源石腚换藏品，战斗中出现龙脚，打掉后掉落二结局藏品。第五层结尾打二结局BOSS。龙脚会踩3X3范围内我方干员最多的位置。

三：洞里【杂疑】要5名伺烛客。获得三结局藏品【忘生珍珑】，在指点迷津打特殊作战，进入第六层三结局。

先锋：伊内丝

狙击：维什戴尔、

特种：新约能天使、缄默德克萨斯、麒麟R夜刀

辅助：酒神、电弧

近卫：司霆惊蛰

医疗：凯尔希

术士：荒芜拉普兰德、

13层进洞3烛火，245进洞5烛火。

一层打架+买招募券，进洞发育不打架。

二层券用光，尽量不进洞。

三层优先走【指点迷津】用券。

四五层进洞打架。

# 结局要点

二 (opens new window)：2层【命运所指】，任意【去伪存真】合并纯白花瓣与无字地契。

真伤，妮芙+塑心蛮干；黍/拉人特种拉二阶段特雷西斯；30血放生特雷西斯。

三 (opens new window)：解锁【失与得】，选“一个方盒”（刷新节点变【命运所指】，2个构想进树洞【诡谲断章】）。

树洞中【命运所指】选“时与光”触发。时代会更频繁。打完1或2结局BOSS后进第六层。

  • 尽可能堆高干员血量，+血量灵感。
  • 高台迷彩不会受到BOSS全屏伤害。2x筑墙可合成。（筑墙部署后获得1层护盾）
  • 放在圣盾周围减伤害。120s或自动机碰到圣盾会消失。
  • 带控制法师堵上面的门，清自动机。
  • 最下方悬浮怪，控制干员处理。
  • 快活骗炮兵子弹。
  • 最后一波出“门”，击倒时带走最后一个部署的干员。
  • boss二阶段前先攒技能，留技能清第三波。

45,62,BOSS不进入二阶段90s后出第三波。盾奶/快活打年糕拖到小怪刷完。

30血放生圣爹。

妮芙+回转手，永控。

四 (opens new window)：【失与得】选一片莲瓣，消耗所有源石锭，战斗获得经验源石锭减半。最好第四层拿。

任意【狭路相逢】（第五层必有），2构想。将花瓣掷入故事。然后打船。获得藏品，领袖与精英敌人生命+50%，开启第六层。

刚需全图奶，纯艾，赫默。

妮芙+回转手，仇白1技能+攻速，永控。

五 (opens new window)：【不期而遇】空无前兆，试着接受它，收藏品预言显像。

任意作战都会刷怪“终曲合声”，进蓝门-1生命值，额外获得任意收藏品，怪血量继承。法伤全图普攻，无法阻挡。

击杀得到思绪“绝望”。【去伪存真】+构想可以合成。绝望》拯救》停止，难度依次提升。

【孤注一掷】铸造钥匙，”无终之钥“开启5结局。负面藏品，终结的xx（部署费用+2，敌人减物法伤20%）(敌人攻击力+20%)（5结局关底出现额外敌人）。

135连打延长发育。也可以45连打。

  • 部署干员被封印，球碰一下解封。
  • 魔王普攻伤害很高。
  • 有名的精英(其他结局boss)进蓝门或被消灭就触发清场，被清掉的怪直接扣目标生命。
  • 用控制技能拖住或放生后马上清杂。

多拿快活先锋，快活挡球。

高攻速下，傀影影子抗魔王兔。

休谟斯，前期单守，清杂。石英，需要医疗，前期单守+清杂；

银灰；芳汀；宴；刻刀：打刺。

维娜，真伤+吃文明存续，二五结局。

号角；

安洁莉娜（特限模组）+抠挠手，灵知也可以吃。

梅，小控制；

华法琳，拐。

玛露西尔，寒檀，蹭波纹手（澄闪）。

深靛，小控制。夜烟，小减抗。

# 特种

阿斯卡伦；锡人；**艾拉；**砾；伊桑；

缄默德克萨斯；麒麟R夜刀；傀影；

歌蕾蒂亚；
