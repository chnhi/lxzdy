显示同步Sync设置 ​

设置参考；RTSS Async原理；刷新率换算最大帧速率公式；

  • 缩写：

  • NVCP = NVIDIA Control Panel（控制面板）

  • RTSS = RivaTunerStatisticsServer

通用设置 ​

画面不撕裂，输入延迟尽可能低。

NVIDIA ​

  • 游戏设置关闭垂直同步（V-sync），启用 nvidia reflex+enhanced 增强，启用低延迟模式。

  • NVIDIA APP 或 NVCP 最大帧速率设置根据显示器刷新率设置： 120-116；144-139；165-158；180-172；240-226；360-328；480Hz-424 FPS；

最大帧速率=刷新率 -（刷新率 x 刷新率/4096）

  • NVIDIA APP 或 NVCP内垂直同步（V-sync）设置为启用On，显示器技术开启G-SYNC Compatible。

1、可能需要显示器上的按钮调出显示器设置面板来启用Adaptive Sync。

2、NVIDIA APP或 NVCP 中有一个单独的设置页面，控制是否开启G-SYNC Compatible。

G-SYNC Compatible可以设置全屏或窗口+全屏。通常优先只启用全屏，假如游戏并没有使用全屏独占模式，而是用的窗口化全屏，通常也是生效的。

对于一些老游戏，只要画面上没有显示overlay，就算调用的是老的非翻转DXGI模型，并且没有禁用全屏优化，通常也生效，但是一旦出现overlay就失效，比如音量调节界面之类的。

AMD ​

Freesync = G-SYNC Compatible

Vsync = 垂直同步；Wait for Vertical Refresh: Always On；一样在游戏设置中关闭。

Radeon Chill = 最大帧速率，只需要最大最小值设置成一样的值。

AMD Anti-Lag 2 ≈ NVIDIA Reflex

好像也可以用RTSS强开。

为什么启用垂直同步V-Sync？ ​

单纯启用G-Sync并不能保证画面完全不撕裂，根据测试显示，启用G-Sync的前提下，启用V-Sync与否，不增加最大输入延迟，但可以保证画面完全不撕裂。对平均输入延迟的影响，随着帧数上升越来越小，在200Hz之后几乎没有影响。

G-Sync + 启用V-Sync将以可以实现的最快速度向显示器提供无撕裂帧，禁用V-Sync获得的最小输入延迟的提升，其本质是一个撕裂帧，不如参考后文的发骚级设置，完全无视画面撕裂追求极限低输入延迟。

发骚级：极限低输入延迟 ​

可能出现画面撕裂。但假如帧数达到500+，则人眼几乎无法感知到画面撕裂。

游戏设置、驱动设置不启用G-Sync，垂直同步（V-sync）。

游戏设置启用 nvidia reflex +enhanced 增强，启用低延迟模式。

NVIDIA APP 或 NVCP 设置最大帧速率为无限制或2~3倍的刷新率差不多500左右。如240hz显示器设置最大帧速率480。

考虑启用游戏内显示，查看GPU占用率，通过控制最大帧速率，使得占用率在约70%左右，尽量留一定余量。

此处70%的数据不绝对，是我自己毛估估。理论上可能90%~95%其实可能也可以，既只留一点点余量。

画面平滑度而不是输入延迟 ​

RTSS Async限帧会缓冲1帧来使画面平滑，但是会增加1帧的输入延迟，对于竞技游戏完全不考虑。但是对于非竞技向游戏，我个人认为理论上可能RTSS+Async限帧也是一个不错的选择。假设更看重画面平滑度而不是输入延迟。

  • 游戏设置关闭垂直同步（V-sync）。

  • RTSS Async 最大帧速率设置根据显示器刷新率设置： 120-116；144-139；165-158；180-172；240-226；360-328；480Hz-424 FPS；

最大帧速率=刷新率 -（刷新率 x 刷新率/4096）

  • NVIDIA APP 或 NVCP 垂直同步（V-sync）设置为启用On，显示器技术开启G-SYNC Compatible。

杂 ​

强开NVIDIA reflex ​

不能启用NVIDIA reflex的游戏，RTSS设置Enable framerate limiter为NVIDIA reflex就可以强开，不过只有游戏内无法开启NVIDIA reflex才需要强开，强开后需要用RTSS设置最大帧速率，驱动和游戏内不需要设置最大帧速率。

显示器不支持Adaptive Sync ​

显示器不能启用G-SYNC Compatible，则尽量不要开启垂直同步（V-sync）。只启用NVIDIA reflex，并按通用设置使用NVIDIA APP 或 NVCP 设置最大帧速率。

FastSync ​

垂直同步+三重缓冲，但是用最新帧刷新，最新帧没赶上趟就用缓冲帧，缓冲帧用不上就丢弃。必须要GPU余量很足，也就是至少能稳定跑288帧以上，才可能在144Hz显示器上取得好的效果。
