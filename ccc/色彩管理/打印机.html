# 喷墨

只考虑墨仓式或喷头墨盒分体机型；喷头墨盒一体的虽然机器便宜，但是耗材贵，加墨麻烦，故障率非常高。

  • 打印速度比激光慢很多，商用机型800喷孔一色速度接近激光。

  • 打印效果

    • 黑度相对激光打印机不够。
    • 爱普生四色颜料都能达到118%cmyk色域容积，喷墨以外打印方式一般都是100%cmyk。

  • 家用打印机一般都是RGB打印机，打印机驱动接受RGB编码，不要画蛇添足改成CMYK模式调色。

# 颜料与染料

  • 颜料墨水，原厂墨都可能堵，不会褪色。至少每周打一次，可以是喷嘴检查图。
    • 不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹）。
    • 堆墨在高光纸上（包括钡基艺术纸）由于光线漫射，会产生烫金效应（反光泛白、泛青铜色、呈彩虹状），观感不如染料自然。主要产生于灰阶、青色墨水（包括浅青），尤其是CMY混合而成的灰阶。有多级黑灰的机型用兼容颜料，烫金效应也相对会好一点。
  • 染料墨水，兼容墨都不会堵，但会褪色。
    • 染料饱和度高，通透感，亮光感是颜料无法实现的。观感更吸睛，但色不稳定，色准会差一点。
  • 断线清洗就通的是打印头干了，堵是自己清不通的。

颜料墨水曝晒也会褪色，但是由于色料外包裹的树脂颗粒先被破坏，使得整体褪色程度相近，只是看起来显旧，颜色间相对联系不变，这也是人眼最敏感的部分。

干式彩扩机染料

爱普生L8168的原装014墨水，耐候性和色域比原厂的普通染料都要好很多。

根据爱普生海外披露的墨水大概的成分来看，和干式彩扩机的活的色彩D6r-S成分相似，具体某色的显色性可能有区别。富士干扩机也是类似的墨水。

爱普生活的色彩DS6、NDD大幅面染料抽墨，耐候性同兼容耐光抗UV染料相近。

注意区分干式彩扩墨水是活的色彩D6r-S。

# 耐候性

这些测试都聚焦于见光的耐候性，染料避光无保护的耐候性测试中。

据wilhelm-research实测加数学推测：

激光银盐怕UV怕空气；热升华怕高温；染料怕高温，怕潮湿，怕空气；

wilhelm-research的标准可能是某个墨水的色密度衰减35%，我的结论基于打印84个色块中挑个别色块计算色密度衰减35%，平均dE00基本在4以上。 (opens new window)

  • 光照强度也是非常关键的因素，颜料墨水室内光照的预设是450lux以下。比如博物馆通常就比较昏暗。
  • 铸涂相纸、喷墨铜版纸通常都含荧光增白剂，在阳光下曝晒2~3周就会降解黄变，比不加还黄得多。
  • 非无酸介质避光保存一般三四十年就可能酸化。无酸介质如硫化钡纸张就要贵很多。
  • 爱普生大幅面染料抽墨，活的色彩DS6、NDD耐候性同兼容耐光抗UV染料差不多。

染料褪色第一难题，灰平衡。

染料墨水机型灰阶主要由CMY墨水混合，而CMY墨水褪色速率不一，这往往导致灰平衡破坏的非常快，DS6墨水短短3天，中性灰块的dE00就超过2.5。

但是灰平衡如果没有参照对比，不一定明显感知。

染料C墨水褪色速率最快

以色密度计算，C墨水褪色速率达到M墨水的2.5~4倍。把C墨水替换成耐候性更好的墨水，可以明显改善整体的耐候性。

染料黑色墨水褪色发红

不仅堆黑显眼，而且会参与到彩色增加色密度，跟C墨水一起劣化整体发红的趋势。干式彩扩黑色墨水褪色更偏向饱和度下降显旧，不至于那么显眼。

如果打算用染料墨水，最好做保护，覆膜、塑封、淋膜，保护后可以不避光，压在玻璃、亚克力等材料下也可以。

  • 填充墨盒稳定性，寿命好。颜料墨水只推荐用填充墨盒，连供要自己用上出墨的uv机用的墨瓶改装，下出墨的连供很容易堵管线影响流动性。
  • 墨仓机型墨囊结构不一样，运行清洗就行，稳定性好很多。

# 兼容染料色盐被卡脖子

爱普生旗舰染料既干扩染料的色料是向Fujifilm买断独占的，大墨水厂如墨库、天威、鸿盛一般不合成色盐，而是采购色盐搅合配成墨水，采购的色盐通常就是纺织行业色盐原型。

  • 相纸的二氧化硅涂层，是物理吸附，墨水被毛细作用吸入孔隙中。
  • 染料色料同纺织品纤维是化学吸附，会形成较强的氢键或离子键。
  • 染料墨水在粗面纸、普通纸上耐候性通常显著强于相纸。

而干扩染料色盐本身针对喷墨影像开发改性，不仅针对性加强耐候性，还要保证溶解度。研发目的不同，从纺织行业的色盐碰运气不太现实。专门研发而市场太小，海外InkThrift兼容染料对标干扩染料，已研发的产品都因为市场太小而停产。

国内有色盐研发能力，唯一聚焦量产合成喷墨色盐的是色如丹，但他是爱普生供应商。完全可以招安，甚至不排除必须依托爱普生供应链才能生产高纯度色盐的可能性。

# 兼容成膜染料

颜料墨水是成膜树脂喷墨后成膜，把颜料颗粒粘在纸上，既堵头往往是树脂干了成膜板结，混加墨水引起反应凝絮，甚至自己莫名其妙凝絮。

兼容染料也可以加成膜树脂，喷墨后成膜，一定程度上减少染料分子同空气、光线的接触。根据我之前的草测，目测见光可能可以多坚持2~3倍时间。详细的测试进行中。 (opens new window)

但是由于色盐本身用于影像的耐候性较为羸弱，而染料合理的使用场景是避光保存。其长期避光保存的耐候性是否会受限于色盐本身，犹未可知。

在染料墨水中添加成膜树脂，其堵头的风险是否同颜料墨水相近，亦不知。

# 照片纸

  • 染料用铸涂相纸（又叫光面或高光相纸）成本较低。
  • 太便宜的铸涂会掉粉，考虑高光喷墨铜版纸，效果相近。要200g以上，不然要调低质量，防止洇墨。
  • 颜料用RC相纸或艺术纸。

泛太克，柯达。汇美有珍珠面，明瑞号称代工爱普生相纸。艺美佳。

染料也可以用RC相纸

RC相纸就是纸基正反面都有PE淋膜，相对更硬挺不易卷曲。RC相纸不适合热裱或塑封，所以商家用铸涂高光相纸，成本也低。个人用户打染料也可以用RC相纸。

有些间隙性RC相纸的涂层有双层，最外层是纳米级二氧化硅颗粒，然后是微米级二氧化硅颗粒。通常纳米级涂层吸纳颜料颗粒固色，水分和溶剂被第二层的微米级颗粒吸收，然后慢慢挥发。这样的RC相纸打染料墨水之后，几分钟内色密度就急剧衰减，因为染料分子跟着墨水一起流到下层微米级涂层里。

**不过目前市面上大部分RC相纸一般不存在这样的现象。**我推测可能是固色涂层里有阳离子树脂之类的成分，会和阴离子染料结合固色。也可能是只有一层纳米级涂层？如果是阳离子树脂结合固色，应该可以加强耐候性。

如果是色密度会急剧衰减的RC相纸，往往褪色也会更快。

颜料分照片黑和粗面黑，粗面黑更不容易渗入介质，适合艺术纸。照片黑颜料在艺术纸上容易渗入介质，导致看起来泛纸白不够黑。但是在照片纸上，因为有涂层，粗面黑墨水往往显得死黑一片，损失暗部色阶，原装粗面黑会好一点，但原装粗面黑在照片纸上往往不耐刮擦。

相纸由涂层固色，色密度相对高一些，色域广一些。而白卡纸，普通纸则几乎完全靠纸基吸墨，色密度受限，而艺术纸则在白卡纸的基础上，涂布有涂层，由涂层与纸基吸墨。从校色文件的色域容积来看，通常白卡纸普通纸不到相纸的四分之一，艺术纸大概是三分之二左右。

那相纸岂不是无敌了吗？也不尽然。人眼对颜色间相对联系更敏感，既除非AB对比，不然对饱和色的差距其实感知不明显。艺术纸哑光的特性看起来更自然，往往具有一种温暖的感觉。

尽管高光喷墨铜版纸，铸涂高光相纸一般也都可以打水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，静置15分钟左右干燥后才附着。干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。

**RC细绒相纸，艺术纸不易感知走纸印和棘轮印，**高光相纸包括RC高光，高光喷墨铜版纸，反光能感知有走纸印和棘轮印。

自行分切卷纸

据网友@逸 实测分析分享，校色后对比平滑度和色域空间大小，佳能旗舰相纸是消费级领域比较容易买到的顶级相纸，但是性价比实在太低，淘宝·华林兴性价比高，但sku多，有好有坏。泛太克代工的柯达可圈可点，国内纸品牌NO.1，但是贴泛太克品牌的不如贴柯达品牌的。

佳能相纸可以考虑买卷纸，自己先反卷，放置两三天后（根据实际情况判断反卷时间）自行分切，分切可以考虑买滚切刀。之后平放一段时间，等其自然整平。

如果打算自行分切，可以考虑在想要的尺寸上，头和尾都留出4厘米，优先纸尾，纸头可以不预留，在打印机驱动中自定义介质大小。

爱普生打印机的纸头尾4cm打印质量可能会下降，可能是因为头尾4cm不是完全压住。

# 荧光增白剂

荧光增白剂曝晒约2~3周降解，一旦降解会导致纸变得比不加荧光增白剂的还黄得多，不过只要不受到光照就不会降解。

要检测是否不含荧光增白剂，用365nm的紫外线手电筒照射即可，注意必须要365nm，有荧光增白剂的纸会发出幽幽的蓝光。395nm的不易分辨。

# 灰阶不够中性

除了色准、灰阶、耐候性存在短板，6色染料的效果相对10色颜料机还是相当可以的。

4~6机型最大的短板是灰阶靠彩墨混合；P708不需要无关墨水增加色密度，靠对应颜色配合黑灰墨水就可以。而6色机型，往往需要无关墨水中和增加色密度，更容易因为同色异谱而产生色偏。

即使都用光度计校色，如果对样张非常熟悉，还是看几眼能感知色准上的差距。

但是**色准高并不代表一切，**CMOS拍照的过程就是用数学算法“猜色”，市面上绝大部分手机厂商的直出，严格来说都过饱和。色彩更多的是一种主观上的感觉，即使色准上差一点，染料墨水的低成本+亮光感和通透感还是有可取之处。

如果换用兼容颜料墨水，彩墨混合的灰阶会表现得更糟，染料墨水存在一定程度的晕染，而颜料墨水CMY混合而成的灰阶反光有严重的烫金效应，泛青铜色。但是染料墨水的CMY褪色速率不一，往往导致兼容染料的灰平衡破坏的很快。

L8168驱动选大部分介质都只用014墨水通道既5色染料墨水，灰阶是彩墨加灰墨混合。但是介质VVE【爱普生天鹅绒美术纸】和【普通纸】，灰阶会主要用013颜料黑和灰墨，明显要中性非常多，但喷墨程序针对粗面纸，在RC相纸上会有颗粒感。

# 换颜料墨水务必审慎

普通染料换来换去一般影响不大。颜料各家的分散剂成分可能不同，更换墨水存在堵头风险，务必审慎，颜料换染料，要特别审慎。

墨仓式打印机，加墨在墨仓中一般约50ml，打印头上有个墨囊（内墨盒），约3ml，打印头内还会残余一些墨水。

染料间或染料换颜料按该视频 (opens new window)操作，抽走旧墨水后直接加新墨水即可。最好用100ml针筒+抽墨针头抽墨仓，比较省事，针筒老化滑动很费劲的时候，往针筒的橡胶头上抹点洗洁精。

**希望最保险的话清洗墨仓。**用清洗液稀释液，10ml清洗液，60ml纯净水即可，加进墨仓，再抽走，直到抽出来是清澈的。必须加满清洗液稀释液等24~48小时后，抽出来是清澈的才行，尤其是颜料。墨库、天威的清洗液就可以，20多块1升，成分其实就是清墨水加“洗洁精”。

考虑用网盘 (opens new window)里售后工具的initial ink charge取代“深度清洗或清洗3~5次”，据老外说清洗会用电信号和超声波刺激震荡墨水通道，连续进行有巨大风险，而initial ink charge只用废墨泵抽墨。

  • 利用initial ink charge

颜料间，或颜料换其他墨水，必须清洗墨仓。

不介意成本，清洗墨仓后，在墨仓内加注每色约30ml清洗液，运行initial ink charge。再清洗墨仓，再加清洗液运行initial ink charge，或多加点清洗液连续运行两次initial ink charge。注意观察墨仓液面是否全部下降。

加满稀释液清洗液，24小时后再次清洗墨仓，再运行一次initial ink charge。再清洗墨仓，换上新墨水，运行initial ink charge后，打打印头检查图，有断线运行清洗。

进墨路的清洗液最好不要稀释，稀释液粘度太低，有可能从打印头上流出去，存在风险。

参考视频 (opens new window)把墨囊拆下来清洗。最好用清洗液中转运行一次initial ink charge或深度清洗后，再上墨水。

清洗后如果没有好转迹象，不要不停清洗，先观察分析。如果固定位置断线，可能是打印头堵了，可能最好找售后，或自己死马当活马医，用那种强效清洗液泡一下，或试试下文的终极一战看看能不能冲开。

如果不是固定位置断线，说明供墨不畅，先检查墨路供墨是否正常。运行清洗观察管线内是否有墨水流动，或墨仓液面是否下降，只要有下降，一般墨路应该是没有大问题。墨仓机型改过打印头结构，微压电晶体之前有一个很细的过滤网，这个过滤网堵的可能性相对大一些。那么就是终极一战，用10ml针筒打6~8ml清洗液回去，针筒轻轻拉满往回抽，会有黑墨被抽回来，是墨栈里的废墨，这样重复换新的清洗液推拉2~3次，可能可以冲开过滤网，最后再打4ml左右清洗液回去。

爱普生的标准8色CMYK+LCLM+LK+LLK就标称潘通93%；添加紫色后标称潘通97%；添加橙绿后标称99%；

**个人用户尽量考虑爱普生：**商用打印量大，打印头是耗材，热发泡打印头相对便宜是优势。爱普生维护得当，十几年前的机器打印头还凑合能用的都有。而热发泡打印头不管你打不打，上颜料墨水放在那里运气不好也是2~3年断线报废。

但不可否认的是，佳能Pro500以一个较低的价格提供了更好的进纸系统，更能处理不平整的纸张，亦有其可取之处，爱普生要P5380贵很多。大部分微喷用的纸张不提供预切纸，都是卷纸。如果用兼容颜料墨水，或过期、拆机墨水也不是完全不能考虑，毕竟PF-10打印头大概2500左右，当成一个两年换一次的耗材也不是完全不行，只怕万一点背。

# 晶亮色墨水

如果打算全部用兼容墨水，需要打有光泽度的纸张，考虑选佳能pro系列机型。晶亮色墨水可以改善兼容颜料的反光泛青铜色观感怪异。

如果有一定要求，Y，RED，BLUE，CO最好都用原装墨。可以说成也CO，败也CO，因为CO墨消耗量很大。据我测试观察，原装CO墨基本不改变原来的颜色，但是兼容CO墨像蒙了一层轻微的灰色滤镜，其中墨库的CO墨反光的光泽度相对比较接近原装CO墨，但是味道很大很冲，而且比过期墨盒便宜的不多。

爱普生系列则考虑，专色、深色墨水CMYK用原装墨水可以扩展色域，浅色墨水LC、LM、LK、LLK用兼容墨水。

其中Y墨水是除了专色兼容相对原装差距最大的，兼容墨水的专色很大概率是用CMYK勾兑。

原装粗面黑墨水更不易渗进介质中，兼容粗面黑要么是浓度更高的照片黑，要么就是高速图文机用的，会死黑。

照片黑墨水，爱普生PRO10,PRO12一代，原装墨水会大幅度减轻反光泛白，但是喷墨量大了反光会泛青铜色。

据我测试观察，墨库兼容颜料用m3偏振测量黑位并不差，既兼容颜料的黑位观感差可能是因为树脂。

# 喷墨与特殊纸

# 热转印

热转印分为普通转印纸（几毛一张），水转印纸(3块起/A4)，纯棉T恤转印纸（烫画纸）（1.5￥起/A4）。

普通转印纸只能用热升华墨水，加热后墨水升华，染到印刷物上，手机壳，键盘键帽等常用该方法。热升华墨水一样会褪色。

水转印纸和纯棉T恤转印纸兼容水性墨水，转印后是把“纸”粘在被印物上。

  • 染料遇水会晕染，也容易褪色，尽量用颜料墨水。
  • 烫画纸有热熔胶和喷墨涂层，加热后胶水融化黏住，有胶质感，不透气。
  • 水转印纸有一层水溶胶。

纯棉T恤转印纸需要和烫画膜（PET离型膜）区分，烫画膜需要搭配数码烫画涂料墨水(DTF)，墨水中含有聚氨酯与异氰酸酯，溶剂蒸发后会固化成膜。再用热熔胶把图案粘到被印物上，烫画膜是拿走的。印有图案的地方不透气。

直喷涂料墨水(DTG)原理类似，只是会涂预处理液，预处理液既交联面料又吸附墨水。

涂料墨水具有腐蚀性，非工业打印头寿命可能是用周来计量，白墨更短。

# 涂层不匹配或无涂层纸

水性墨水需要被打印物有涂层，不然不吸墨，或者普通纸那样墨水渗入纸张但是分辨率较低。这些情况下，会有3种表现。洇墨，墨水有流动痕迹，一摸就花。

  • 洇墨，既墨水有轻微化开的感觉。通常考虑调整打印机驱动中介质选择改变喷墨程序，调低喷墨浓度来调整。

常见于介质选择不适配，或菲林纸，PET喷墨防水透明胶片 (水性防水菲林片)。

  • 一摸就花，不耐刮擦。

最好静置15~30分钟干燥，如果静置后依旧不耐刮擦，说明墨水不适合该介质。

尽管高光喷墨铜版纸，铸涂相纸都可以正常打印水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。而染料墨水在高光纸上的走纸印需要反光才能感知。

  • 墨水有流动的痕迹，常见于无涂层纸，如胶版纸（道林纸）没有涂层。

无涂层，即使调整到可以打印了，效果也是很差的，只能考虑快干颜料墨水或工业墨水。

印刷厂俗称的铜版纸是涂布印刷纸coated art paper。没有喷墨涂层，喷墨打印机必须用铜版纸颜料墨水（快干颜料墨水）搭配加热装置。快干颜料墨水掺了弱溶剂墨水的溶剂，低温下粘度会变高，易出现斜喷等异常现象。

据墨库介绍，没有加热装置最好室温30度以上。

# 热发泡微压电墨水互用性

热发泡打印头用微压电墨水，通常正常工作，但是染料墨水加热后黑色会变红褐色。颜料墨水恐怕堵头概率会很大。

微压电打印头用热发泡墨水也能正常工作。

# 工业墨水

都是通过不同原理能固化成膜，极易堵头，往往具有强腐蚀性，非工业打印头寿命按天计算。

  • 弱溶剂墨水

非工业打印头用溶剂墨水可能3个月甚至更短就报废。

爱普生SC-S40680等机型的墨盒是溶剂墨水，可以抽墨。墨盒260块/700ml，国产溶剂墨水，便宜的80块/L。

  • 乳胶墨水（latex ink）

乳胶墨水也是水性墨水，在50~60度左右树脂分散乳液会固化成膜。HP用这个墨水的打印机用的是热发泡喷头，我推测腐蚀性可能较弱，堵头概率可能相对较小，但毕竟是会成膜的墨水，也不好说。

1688上东莞奥彩的兼容墨水卖325/L，可以零售。

鸿盛有适配微压电喷头的乳胶墨水，官网介绍打印介质加热到40~50度即可。咨询后，不零售，必须对接他们区域业务员，只接待商用客户，可能要买他们的机器或改装方案。他们好像用到空气干燥加热，可能必须要蒸发水分。

其介绍的工艺非常繁琐，不过可能是用在特殊介质上，先用不同喷头打一遍无色墨水，再数码打印，之后还要涂后处理液固色剂140度加热5分钟固色。

乳胶墨水好像要100多度才能完全充分反应，达到最强的黏着力，但是如果不是打印纺织品，可能自然风干也可以，只是黏着力弱些。

后处理液固色剂好像有二氧化硅使其摸起来滑，有会固化成膜的树脂材料等其他成分。

无色墨水据介绍有点像光油，但是只能打底不能盖面，主要起桥接作用，加强介质适应性，还可以平整提高精度，好像还可以提升交联性。好像之所以能实现吸墨涂层是因为会和白彩墨水产生成膜的反应，喷头要保持一定距离。

涂料墨水（DTF烫画墨水），UV墨水，都是通过不同原理固化成膜的墨水，都具有强腐蚀性，还极易堵头。

# 工业打印机

工业用墨水，包括UV墨水，涂料墨水，溶剂墨水都具有强腐蚀性，且极易堵头，喷头完全是消耗品，爱普生民用打印头往往一两个月就报废，几千上万的工业打印头维护得当也基本就一两年寿命，要靠商用分摊成本。

维护不得当甚至可能新打印头都马上报废。

就细节上稍微改改，打水性墨水就是写真机，打溶剂墨水就是喷绘机，打uv墨水就是uv打印机。

厂家都是联想式攒电脑的组装厂，基本都不掌握核心竞争力，同价位机器都差不多，主要还是看当地售后师傅是否热心。前提是正经厂家。

机型往往会用幅面代称，比如6090就是支持60mmX90mm，1610既支持1.6米x1米。

# 打印头

工业头基本都有加热功能，uv墨水低温下粘度会变高，爱普生I3200没有加热，T3200有加热功能。

工业头比较耐压，可以负压供墨，而家用头都是正压供墨，工业用墨粘度高，很容易供不上墨。

工业打印头基本都能应付溶剂墨水两年左右，应付uv墨水则层次不齐：

  • 爱普生I3200（1寸） 没有加热，据称寿命大概能有6个月甚至更长，但是跑的机器都是工业大机，不如上工业大头。

  • 小理光gh2220，1头2通道，可2色。360喷孔。

寿命比g5短很多，g5平均能顶2年，这个头则只能顶3个月。

一样是小头，爱普生I3200一般用低粘度墨。小理光却用工业大头的高粘度墨。

工业小头比民用头贵了很多，寿命也不算理想，整套机器也不便宜，有点尴尬。

  • 爱普生T3200（2.65寸），T3200有加热，价格小贵。

  • 理光G5/G6，1头2色。

G5标称7PL，G6标称5PL，G5大概1万2一个头。

# 专色好像不参与cmyk

我浅浅了解后发现，工业打印机曲线似乎只兼容CMYK和LC，LM，LK，LLK。爱普生微喷导入紫色，橙色，RGB等墨水会调整喷墨程序用于扩展色域。但是工业打印机好像只能把新导入的颜色作为专色，既指定具体几组RGB色值用这个墨打，或在图像文件中新建一个通道，这个通道同cmyk不产生联系，既用于打金属银，白墨，光油之类的特殊墨水。
