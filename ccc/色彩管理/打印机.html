# 喷墨

只考虑墨仓式或喷头墨盒分体机型；喷头墨盒一体的虽然机器便宜，但是耗材贵，加墨麻烦，故障率非常高。

  • 打印速度比激光慢很多，商用机型800喷孔一色速度接近激光。

  • 打印效果

    • 黑度相对激光打印机不够。
    • 爱普生四色颜料都能达到118%cmyk色域容积，喷墨以外打印方式一般都是100%cmyk。
    • 颜料优点是不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹），大面积深色墨水在相纸上由于光线漫射，会产生烫金效应（深色墨水反光会泛白泛青铜色），观感不如染料自然。
    • 染料饱和度高，通透感，亮光感是颜料无法实现的。观感更吸睛，但色不稳定导致色准和色阶都受到限制，有层次上的损失。

  • 家用打印机一般都是RGB打印机，打印机驱动接受RGB编码，不要画蛇添足改成CMYK模式调色。

# 耐候性

  • 颜料非常好。
  • 染料很差，易褪色，原厂染料覆膜基本不褪色，但还是怕高温，怕潮湿。
    • 染料黑褪色会发红，导致明显色差，其他方式暴晒后通常也会褪色，但是整体褪色，不易感知。
  • L8168的原装墨水，同爱普生的干式影像输出设备用的活的色彩D6r-S墨水的成分相似，以及富士的干式打印机的墨水，可能具体的显色性不同，但他们是成分相似的同类墨水。
  • 佳能G580的染料墨水是ChromaLife100，比爱普生普通染料强，要pro100的chromalife100+才能和干扩机染料PK。

据老外实测加数学推测：普通原厂染料覆膜只有室内光照，约17年不褪色，激光银盐约40年，热升华约17年，颜料约60~200年，claria染料约98年。

普通原厂染料怕臭氧，怕潮湿，怕高温。激光银盐怕UV，怕空气。热升华怕高温（自带覆膜所以不怕别的）。

标准是noticeable fading or changes in color balance occur。

干式彩扩机墨水相对原厂普通染料，耐候性上最大的提升是，普通染料黑褪色速度过快，一褪色就发红，破坏颜色间相对联系，而干式彩扩机墨水在褪色初中期，黑色不会明显发红，更偏向于整体褪色而不破坏颜色间相对联系。

兼容染料包括所谓抗UV墨水，不覆膜，避光3~6个月会褪色，见光1个月，阳光直射基本3天就褪色发红。原装普通染料，能坚持几倍时间，但最终一样会褪色。

颜料墨水曝晒也会褪色，但是由于色料外包裹的树脂颗粒先被破坏，使得整体褪色程度相近，只是看起来显旧，颜色间相对联系不变，这也是人眼最敏感的部分。而染料墨水中黑色的褪色速度显著的快于其他颜色，很快褪色泛红，使得染料的褪色特别明显。

  • 堵头

    • 染料就算是兼容墨水都不会堵，即使断线了也很容易泡开冲开。
    • 颜料原厂墨都有可能堵，特别是气候干燥地区。长期不打存在堵死冲不开的可能。至少每周打一次，可以是喷头检查图。

  • 连供

    • 填充墨盒稳定性，寿命好。
    • 官方连供，墨仓相对寿命长一些，但是墨囊还是会进空气，导致气堵不出墨，L805用注射器给墨囊排气视频 (opens new window)；
    • 墨仓机型运行深度清洗，墨囊结构不一样，断线概率已经低了很多。

  • 佳能惠普喷墨不要考虑。热发泡喷头寿命短，墨水稀，用量大，性价比低，兼容墨贵，颜料极其容易堵死，喷头直接报废。打印前还要预热，速度慢。

  • 爱普生机型，用兼容染料，约4万张A4后喷头状态下降，标准打印质量下降，需要调高打印质量增大喷墨量掩盖瑕疵，其寿命可见一斑。

  • 爱普生机型打一张A4照片大约共消耗1ml墨水，清洗喷头每色就要1~2ml墨水。墨仓机型深度清洗，我估算每色3~5ml左右，差不多墨囊的容积。

# 照片纸

  • 染料用铸涂相纸（又叫光面或高光相纸）成本较低。
  • 有些铸涂会掉粉，不如买高光喷墨铜版纸，效果相近。要200g以上，不然要调低质量，防止洇墨。
  • 颜料用RC相纸或艺术纸。

颜料分照片黑和粗面黑，粗面黑通常更不容易渗入介质，适合艺术纸。照片黑颜料在艺术纸上容易渗入介质，导致看起来泛纸白不够黑。但是在照片纸上，因为有涂层，粗面黑墨水往往显得死黑一片，损失暗部色阶，原装粗面黑会好一点，但原装粗面黑在照片纸上往往不耐刮擦。

需要有喷墨涂层的介质才能吸收更多墨水而不洇墨，进而增加色密度。通常相纸纸基上有一层PE膜，基本完全靠涂层吸墨，所以相纸往往有最高的色密度，通常有最广的色域，色域广了，自然也更容易表现更多的色阶。

而白卡纸，普通纸则几乎完全靠纸基吸墨，表现力严重受限，而艺术纸则在白卡纸的基础上，涂布有涂层，由涂层与纸基吸墨，色域远胜于白卡纸普通纸，相对相纸还是要差一些。从校色文件的色域容积来看，通常白卡纸普通纸不到相纸的四分之一，艺术纸大概是三分之二左右。

那相纸岂不是无敌了吗？也不尽然。人眼对颜色间相对联系更敏感，既除非AB对比，不然对饱和色的差距其实感知不明显。艺术纸哑光的特性看起来更自然，往往具有一种温暖的感觉。

尽管高光喷墨铜版纸，铸涂相纸在调低质量后都可以打水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，静置15分钟左右干燥后才附着。干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。

RC细绒相纸，艺术纸不易感知走纸印和棘轮印，高光相纸包括RC高光，高光喷墨铜版纸，反光能感知有走纸印和棘轮印。

据网友@逸 实测分析分享，校色后对比平滑度和色域空间大小，佳能旗舰相纸是消费级领域比较容易买到的顶级相纸，但是性价比实在太低，淘宝·华林兴性价比高，但sku多，有好有坏。泛太克代工的柯达可圈可点，国内纸品牌NO.1，但是贴泛太克品牌的不如贴柯达品牌的。

佳能相纸可以考虑买卷纸，自己先反卷，放置两三天后（根据实际情况判断反卷时间）自行分切，分切可以考虑买滚切刀。之后平放一段时间，等其自然整平。

如果打算自行分切，可以考虑在想要的尺寸上，头和尾都留出4厘米，优先纸尾，纸头可以不预留，在打印机驱动中自定义介质大小。

爱普生打印机的纸头尾4cm打印质量可能会下降，可能是因为头尾4cm不是完全压住。

# 墨水推荐

  • 换墨水必须校色。

染料机换颜料包括原厂颜料，也必须校色，染料黑660nm以上波长光反射率会上扬，既看起来是红褐色，必须用C和Y墨水混合绿色来中和，换颜料墨水不校色黑色会发绿。

兼容墨水强烈推荐墨库·百度西，价格实惠，商用数码墨水中国销量第一，至少杂质过滤合格。实际适配的机型很多，但是不重视零售，可以向墨库淘宝客服 (opens new window)索要产品型号表。

天威也是大牌，相对比较重视零售，需要小瓶，易加墨瓶可以考虑天威，性价比差些。

其他品牌，包括所谓南朝鲜墨水inkmate往往杂质过滤都不合格，必须用漏斗过滤。

  • 墨库染料，32￥/1L。校色icc色域容积跟原厂普通染料自行校色的icc差不多，能达到原厂icc的90%，但耐候性差距非常大，包括所谓抗UV墨水。

  • 墨库颜料，43￥/500ml。校色icc色域容积达到爱普生原装颜料自行校色的75%~80%，原厂ICC的70%。

    • 兼容颜料低温流动性不好，低温下可能斜喷。
    • 颜料墨水在高光相纸上有烫金效应（深色墨水反光会泛白泛青铜色），观感不如染料自然。黑墨酌情用大幅面颜料墨盒抽墨。P6080的HD系列墨水，比K3黑，K3如果便宜也可以考虑，大面积堆黑反光会泛白。P7580的Pro10，12系列黑墨反光不泛白，但是反光可能呈现出青铜色，泛白可能还自然一点。

  • 爱普生大幅面机器的墨盒抽墨。原厂颜料，1.5~2￥/ml。

  • 爱普生活的色彩NDD，p10080d机器的9色袋装染料墨水230￥/1L。耐候性跟L8058的墨水差不多，同干扩机染料有差距。现在更新为P11080D，墨盒型号T53Yx，闲鱼我见过一盒两支不拆盒，既1色3.2L，600￥。（不清楚爱普生中文官网为什么叫活的色彩DS6，海外的Ultrachrome DS6是热升华墨水，理论上P11080D的墨水应该还叫活的色彩NDD，就加了大红色R专色）

我抽墨分装的闲鱼链接 (opens new window)，分享的校色文件详见色彩管理-打印机校色文件分享 (opens new window)。

  • 爱普生009 原厂颜料4色，99￥/黑129ml，彩70ml。

  • 爱普生T49L墨水，T3180D机型的墨水。原厂染料4色。过期的25块1瓶140ml，新鲜的40多块一瓶。

  • 干扩机墨水：爱普生D580墨瓶。D880，D1080等机型是墨盒，另有富士DX100,DE100等。

# 热发泡

佳能，hp热发泡，可以买HP-gt52抽墨，或G580墨水抽墨。G580墨水是佳能的chromalife100染料，耐候性相当不错，比爱普生普通染料强。

# 换墨水小提示

普通染料换来换去一般影响不大。颜料各家的分散剂成分可能不同，更换墨水存在堵头风险，务必审慎，颜料换染料，尤其是干扩机同类染料，也要审慎。

墨仓式打印机，加墨在墨仓中一般约50ml，打印头上有个墨囊（内墨盒），约3ml，打印头内还会残余一些墨水。

染料间或染料换颜料按该视频 (opens new window)操作即可，务必要深度清洗。

颜料间，或颜料换其他墨水，墨仓内旧墨水抽走后，最好在墨仓内加50ml清洗液，抽走清洗液后，加纯净水（买娃哈哈纯净水），抽走纯净水，直到抽出来是清水。

参考该视频 (opens new window)把墨囊内旧墨水轻轻抽走，看抽出来多少，轻轻的打纯净水或清洗液回去，一般2ml。加完墨水后**务必运行深度清洗。**要求高按该视频前3分钟操作，清洗墨囊和打印头内残余墨水。

浅色墨水LC,LM,Y可能可以稍微敷衍一点，但是BK,C,M浓度高，尤其是BK，务必审慎。

# 爱普生过期墨盒抽墨

大部分就是改了个名，过期K3VM，HDR的墨盒完全可以考虑，只是注意

  • 黑色墨水是改动的重头戏，尽量不要混用。
  • Y黄色不要和后面出的混用。
  • M和LM注意区分鲜洋红和洋红，鲜洋红理论上可以和后出的混用。

**混用有改动的墨水堵头概率很大。**这些都是根据表现和宣传推测的，不排除没提到的有改动的可能性。

  • 世纪虹彩；4600,7600机型。
  • 世纪虹彩K3；3800,7800机型，增加浅灰色。

# 家用喷墨·影像输出

  染料的通透感，亮光感是颜料无法实现的。观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。

缺点一：易褪色，色不稳定，色准和色阶都受到限制，有层次上的损失。

缺点二：染料黑光谱有明显的红色偏移（看起来泛红），既660nm以上波长的可见光反射率上升。

  颜料优点是不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹），输出大面积黑色时，由于光线漫射，会产生烫金效应，反光泛白泛青铜色。

相对P708，6色机型最大的短板是灰阶靠彩墨LUT；P708不需要无关墨水增加色密度，靠对应颜色配合黑灰墨水就可以。而6色机型，往往需要无关墨水中和增加色密度，更容易因为同色异谱而产生色偏。

# L8058

  • 不在乎成本，干式彩扩机墨水。
  • 在乎成本，大幅面染料抽墨。黑色酌情考虑干扩机墨水。
  • 可以做到每周至少1打，不担心颜料堵头，墨库·百度西 兼容颜料。只限光线充足的照片。

L8058灰阶由彩墨混合，不够中性，校色作用较小，包括i1 pro在内的光度计在暗部都有一定的本底噪声，而且用彩墨混合，怎么样都会有点偏色。

如果换用兼容颜料墨水，彩墨混合的灰阶会表现得更糟，因为染料墨水存在一定程度的晕染。

8058的速度明显更快，300g左右厚纸进纸稳定性更高。8168还能进一步调慢速度延长墨水干燥时间，8058没有提供类似功能。8168速度慢，可能更倾向于喷小墨滴。

8058可能为了速度而降低了质量，也因此，8058在打印头状态下降后往往很容易出横纹，特别是标准质量。

# L8168

缺浅色墨水，在浅色阶过渡上细看有明显颗粒感。原装墨水耐候性显著提升，在色域上也有明显扩展，既不需要专色也可以打出还算可以接受的大红色和蔚蓝色。

L8168驱动选大部分介质都只用014墨水通道既5色染料墨水，灰阶是彩墨加灰墨混合。但是介质VVE【爱普生天鹅绒美术纸】和【普通纸】，灰阶是用013颜料黑和014灰墨混合。

爱普生L8168搭配原装墨水，提供了黑白模式，算是一个在艺术纸上入门级可用的黑白输出，但不用原装墨水可用性较低。

灰度模式则只用黑灰墨水，但是因为浅色阶会泛纸张本色，反而观感不中性。

原装013墨水是粗面黑，在任何纸上都不耐擦。

# L8168不推荐兼容颜料

只用014墨水通道喷墨的介质选项，灰阶由彩墨混合，颜料墨水对灰阶的劣化在我看来浪费了导入的灰色墨水，同L8058半斤八两。尽管如此，L8168在色准上除了绿色系以外都会比8058更高。

使用013通道的两种介质，VVE和普通纸，灰阶明显要中性非常多，但问题在于喷墨程序是针对粗面纸，显著减少喷墨量，在RC相纸上会有明显颗粒感。

L8168机型建议尽量用原装墨水，最多用大幅面染料抽墨。

# 佳能PRO500 vs 爱普生P908

pro500海外叫pro1000。

# 喷墨与特殊纸

# 热转印

热转印分为普通转印纸（几毛一张），水转印纸(3块起/A4)，纯棉T恤转印纸（烫画纸）（1.5￥起/A4）。

普通转印纸只能用热升华墨水，加热后墨水升华，染到印刷物上，手机壳，键盘键帽等常用该方法。热升华墨水一样会褪色。

水转印纸和纯棉T恤转印纸兼容水性墨水，转印后是把“纸”粘在被印物上。

  • 染料遇水会晕染，也容易褪色，尽量用颜料墨水。
  • 烫画纸有热熔胶和喷墨涂层，加热后胶水融化黏住，有胶质感，不透气。
  • 水转印纸有一层水溶胶。

纯棉T恤转印纸需要和烫画膜（PET离型膜）区分，烫画膜需要搭配数码烫画涂料墨水(DTF)，墨水中含有聚氨酯与异氰酸酯，溶剂蒸发后会固化成膜。再用热熔胶把图案粘到被印物上，烫画膜是拿走的。印有图案的地方不透气。

直喷涂料墨水(DTG)原理类似，只是会涂预处理液，预处理液既交联面料又吸附墨水。

涂料墨水具有腐蚀性，非工业打印头寿命可能是用周来计量，白墨更短。

# 涂层不匹配或无涂层纸

水性墨水需要被打印物有涂层，不然不吸墨，或者普通纸那样墨水渗入纸张但是分辨率较低。这些情况下，会有3种表现。洇墨，墨水有流动痕迹，一摸就花。

  • 洇墨，既墨水有轻微化开的感觉。通常考虑调整打印机驱动中介质选择改变喷墨程序，调低喷墨浓度来调整。

常见于介质选择不适配，或菲林纸，PET喷墨防水透明胶片 (水性防水菲林片)。

  • 一摸就花，不耐刮擦。

最好静置15~30分钟干燥，如果静置后依旧不耐刮擦，说明墨水不适合该介质。

尽管高光喷墨铜版纸，铸涂相纸都可以正常打印水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。而染料墨水在高光纸上的走纸印需要反光才能感知。

  • 墨水有流动的痕迹，常见于无涂层纸，如胶版纸（道林纸）没有涂层。

无涂层，即使调整到可以打印了，效果也是很差的，只能考虑快干颜料墨水或工业墨水。

印刷厂俗称的铜版纸是涂布印刷纸coated art paper。没有喷墨涂层，喷墨打印机必须用铜版纸颜料墨水（快干颜料墨水）搭配加热装置。快干颜料墨水掺了弱溶剂墨水的溶剂，低温下粘度会变高，易出现斜喷等异常现象。

据墨库介绍，没有加热装置最好室温30度以上。

# 热发泡微压电墨水互用性

热发泡打印头用微压电墨水，正常工作，但是染料墨水加热后黑色会变红褐色。颜料墨水恐怕堵头概率会很大，热发泡墨水里会掺溶剂，水蒸发后还会有一点溶剂。

微压电打印头用热发泡墨水也能正常工作。

# 工业墨水

都是通过不同原理能固化成膜，极易堵头，往往具有强腐蚀性，非工业打印头寿命按天计算。

  • 弱溶剂墨水

非工业打印头用溶剂墨水可能3个月甚至更短就报废。

爱普生SC-S40680等机型的墨盒是溶剂墨水，可以抽墨。墨盒260块/700ml，国产溶剂墨水，便宜的80块/L。

  • 乳胶墨水（latex ink）

乳胶墨水也是水性墨水，在50~60度左右树脂分散乳液会固化成膜。HP用这个墨水的打印机用的是热发泡喷头，我推测腐蚀性可能较弱，堵头概率可能相对较小，但毕竟是会成膜的墨水，也不好说。

1688上东莞奥彩的兼容墨水卖325/L，可以零售。

鸿盛有适配微压电喷头的乳胶墨水，官网介绍打印介质加热到40~50度即可。咨询后，不零售，必须对接他们区域业务员，只接待商用客户，可能要买他们的机器或改装方案。他们好像用到空气干燥加热，可能必须要蒸发水分。

其介绍的工艺非常繁琐，不过可能是用在特殊介质上，先用不同喷头打一遍无色墨水，再数码打印，之后还要涂后处理液固色剂140度加热5分钟固色。

乳胶墨水好像要100多度才能完全充分反应，达到最强的黏着力，但是如果不是打印纺织品，可能自然风干也可以，只是黏着力弱些。

后处理液固色剂好像有二氧化硅使其摸起来滑，有会固化成膜的树脂材料等其他成分。

无色墨水据介绍有点像光油，但是只能打底不能盖面，主要起桥接作用，加强介质适应性，还可以平整提高精度，好像还可以提升交联性。好像之所以能实现吸墨涂层是因为会和白彩墨水产生成膜的反应，喷头要保持一定距离。

涂料墨水（DTF烫画墨水），UV墨水，都是通过不同原理固化成膜的墨水，都具有强腐蚀性，还极易堵头。

# 工业打印机

工业用墨水，包括UV墨水，涂料墨水，溶剂墨水都具有强腐蚀性，且极易堵头，喷头完全是消耗品，爱普生民用打印头往往一两个月就报废，几千上万的工业打印头维护得当也基本就一两年寿命，要靠商用分摊成本。

维护不得当甚至可能新打印头都马上报废。

就细节上稍微改改，打水性墨水就是写真机，打溶剂墨水就是喷绘机，打uv墨水就是uv打印机。

厂家都是联想式攒电脑的组装厂，基本都不掌握核心竞争力，同价位机器都差不多，主要还是看当地售后师傅是否热心。前提是正经厂家。

机型往往会用幅面代称，比如6090就是支持60mmX90mm，1610既支持1.6米x1米。

# 打印头

工业头基本都有加热功能，uv墨水低温下粘度会变高，爱普生I3200没有加热，T3200有加热功能。

工业头比较耐压，可以负压供墨，而家用头都是正压供墨，工业用墨粘度高，很容易供不上墨。

工业打印头基本都能应付溶剂墨水两年左右，应付uv墨水则层次不齐：

  • 爱普生I3200（1寸） 没有加热，据称寿命大概能有6个月甚至更长，但是跑的机器都是工业大机，不如上工业大头。

  • 小理光gh2220，1头2通道，可2色。360喷孔。

寿命比g5短很多，g5平均能顶2年，这个头则只能顶3个月。

一样是小头，爱普生I3200一般用低粘度墨。小理光却用工业大头的高粘度墨。

工业小头比民用头贵了很多，寿命也不算理想，整套机器也不便宜，有点尴尬。

  • 爱普生T3200（2.65寸），T3200有加热，价格小贵。

  • 理光G5/G6，1头2色。

G5标称7PL，G6标称5PL，G5大概1万2一个头。

  • 东芝CE4，1头1色。

据称东芝，柯尼卡技术授权来自于XAAR。

东芝CE4比理光G5可能寿命短一点，速度慢一点，价格稍微便宜点，4000块。

  • 精工508gs，1024gs，1头1色。

508gs，12PL，1024gs，7pl。价格较高，但是12PL的头我感觉寿命应该会更长。

精工和柯尼卡好像产量不高，用量也不大的样子，不知道会不会买不到喷头，或者坐地起价。

# 民用机魔改

改装机就是练练手交学费，看看怎么个事，想省心必须工业机。

改装机厂家的工业机型完全不要考虑，都是骗一个算一个，机器耗材都卖出天价。

  • 改装机品牌没意义，都毫无技术可言。
  • 一样交学费，不如去二手机贩子那里交学费，少交点。

机器故障率非常高，**做好喷头1月1换的心理准备。**商家把耗材也卖出天价，不如买墨库的墨水。

  • 小型机有用民用头开发的，幅面4050，3360，6090，喷头用爱普生XP600之类的。二手，翻新xp600喷头淘宝卖几百块，还算便宜。

这种机器可以做cmyk线性化校色，也可以调整RIP。还凑合可以用用看怎么个事。

  • 幅面A4，A3的改装机干脆就是爱普生办公打印机魔改，还是爱普生主板和驱动，喷头是所谓TX800，R1390(L805)，所谓的AcroRIP实质上还是发送的RGB数据给爱普生驱动。

这种uv打印机毫无用处纯粹是垃圾；

# 专色好像不参与cmyk

我浅浅了解后发现，工业打印机曲线似乎只兼容CMYK和LC，LM，LK，LLK。爱普生微喷导入紫色，橙色，RGB等墨水会调整喷墨程序用于扩展色域。但是工业打印机好像只能把新导入的颜色作为专色，既指定具体几组RGB色值用这个墨打，或在图像文件中新建一个通道，这个通道同cmyk不产生联系，既用于打金属银，白墨，光油之类的特殊墨水。
