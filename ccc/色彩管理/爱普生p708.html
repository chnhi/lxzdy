# 相对6色染料的提升：

1、颜料墨水受纸白的影响比染料小，颜料不透光而染料透光。配合染料墨水的廉价纸张往往含荧光增白剂，在不同光源下表现有差异，导入误差。

2、多级黑灰+颜料墨水，才实现较为中性的灰阶，10色兼容颜料不校色灰阶相对6色兼容染料就中性很多。

3、黑灰墨水避免了使用CMY混合色阶在彩色中增加色密度，减少同色异谱引起的偏色。

4、染料墨水的色不稳定往往在校色中导入不可控的细微偏差。

6色染料机RGB校色，以上介绍的点叠加之下，如果对样张非常熟悉，看几眼就能感知色准上的差距。

但是**色准高并不代表一切，**CMOS拍照的过程就是用数学算法“猜色”，市面上绝大部分手机厂商的直出，严格来说都过饱和。色彩更多的是一种主观上的感觉，即使色准上差一点，染料墨水的低成本+亮光感和通透感还是有可取之处。

10色机的驱动和4~6色机的驱动明显是两帮人开发，同样开5760dpi，不清楚是MicroWeave技术还是驱动的处理，颜料墨水应该锐度更高，但p708纹理细节反而感觉更肉，L8058明显更锐，凑近细看颗粒感更明显。当然这属于比较细微的差异，不AB对比不易感知，说不上来哪种更好。

兼容墨水不匹配喷墨程序

除非商用大幅面配RIP软件，不然驱动只接受RGB数据，转换为实际喷墨点由打印机固件进行，而兼容墨水对喷墨程序的匹配不如原装墨水，由于只能进行RGB校色，一样校色相对原装墨水在色准上还是会有细微差距，尤其是灰阶。

都用兼容颜料墨水RGB校色，受益于多级黑灰墨水，色准明显提升，尤其是灰阶，相对6色机型CMY混合的灰阶显著改善烫金效应反光泛青铜色观感怪异。

如果爱普生放开4~6色机型的线性化，可能缩小差距。但不太可能放开，而且还要驱动或RIP软件适配靠谱的抖动算法，不然会有明显颗粒感，这方面驱动最靠谱，追求效果无疑原装墨水最省心。

染料王L8168+原装014墨水

最好配合稍微贵一点不含荧光增白剂的RC相纸，成本不低，而且还是最好避光保存。适合不计成本追求染料墨水的通透感，亮光感。

理论上染料墨水的深色墨水墨点相对浅色墨水墨点更不易褪色，L8168+014的CMYK都是深色墨点，而且匹配喷墨程序。

10色颜料机型用干扩墨水，CMYK、LCLM干扩抽墨、GY用014，自行稀释GY为LGY，一份C两份M勾兑出V墨水。用ColorBase 2软件线性化，该软件调整幅度应该不会很大，恐怕变动过大了。

富士有专门的灰色色浆，可能是用不同的CMY发色基团拼配，并且平衡了褪色速率，配合聚合反应，组装成聚集体染料，稀释可能破坏平衡，不够中性，耐候性也下降。稀释K墨水同理。

10色颜料机型用兼容染料墨水，浅色墨水不太可能专门研发色浆，应该就是稀释，本来耐候性就差，稀释更完蛋，除非做保护。理论上效果应该比4~6色机好，但我觉得不如用L8168，10色机多的颜色用染料墨水感觉意义不是很大，机器的差价都能买六七组014墨水了。

# 墨水折腾

爱普生P708的零售墨盒25ml，P908的零售墨盒50ml。

但是巨变态的是P708随机附带的初始化墨盒是34ml，P708的初始化既维护工具的initial ink charge要消耗29ml墨水，P908消耗31ml墨水，P908随机墨盒应该是50ml。

P708如果用零售墨盒初始化，中途要替换墨盒。

P708的墨盒小的变态，更换墨盒后会自动运行清洗浪费约15~20ml墨水。

爱普生P708设计的这么拧巴，不是和P900共用墨盒，我推测是面对佳能Pro300的竞争，机身不能卖太贵，但爱普生又觉得太亏。

# 改用微喷墨盒抽墨

大幅面微喷的墨盒墨水单价要便宜很多，新鲜的同系列ProX大概每毫升2块左右。淘宝商家HD系列大概1块2，过期、拆机墨盒更便宜，1块左右。闲鱼有些抽墨的更便宜，7毛~1块不等。

空填充墨盒，可以加注60ml墨水。加墨后注意加墨口堵住，气孔拔掉，出墨口朝上往外抽空气。

“ 墨量重置器 “ 重置墨盒芯片，0%芯片会锁死不能重置。初始化墨盒芯片也可以重置，墨量计数只要不是满的都可以重置，并没有低到15%才能重置的限制。重置的时候，可以顺便把低于50%的全重置了，避免机器因换墨盒触发清洗，导致清洗下又有墨盒芯片空了。

废墨维护箱芯片计数不能低于15%。

P708零售墨盒重置后好像是25ml计数，P708初始化墨盒重置后好像是34ml计数。理论上计数更大可能可以减少墨盒更换而浪费墨水清洗。

这类兼容填充墨盒不清楚是什么原因，尽量控制墨盒内至少保证有一定量余墨，或最好高于出墨口，不然很容易莫名其妙掉链子进空气反而浪费墨水清洗。所以从这个角度来看，25ml的芯片可能还勉强可以一次加墨两次重置，34ml的芯片就要悠着点了。

简单进行了测试好像是这样，但不能完全确定，后续将进一步测试确定。

换言之，理论上用初始化墨盒芯片重置满了之后可能可以运行维护工具的initial ink charge，而不用担心芯片计数变0重置不了。

墨水参考后文【微喷颜料墨盒抽墨】。

如果用了原装初始化墨盒进行充墨：

  • Y黄色墨水必须用HD系列（P6080）之后机型的墨盒，和以前的版本配方不同，存在巨大的凝絮风险。

  • 照片黑墨水，保险起见最好用ProX系列的墨盒抽墨，不过有网友直接加HD系列照片黑，一切正常，但不保证一定正常。

  • GY、LGY，保险起见最好用HD、ProX系列的墨盒抽墨，不过有网友直接加K3系列LK、LLK，一切正常，但不保证一定正常。

如果要用ProX系列照片黑，最好只用3档质量（1440x1440dpi），质量开高了喷墨量大了，黑墨反光反而可能泛青铜色，观感怪异。

重置芯片每次开机、打印都会提示非原装墨盒，要按确定

用维护工具看用过的墨盒计数（Ink replacement counter），用重置过的墨盒芯片不增加计数。

我大胆猜测机器可能只记录上一个墨盒硬件码，墨量计数跳变识别到非原装墨盒。用两套芯片交替重置上机，有可能不跳非原装墨盒，不过没有实测，也可能没用。

# 改用兼容墨水

兼容墨水的CMYK、专色Violet紫色，色域相对原装墨水有较大差距。如果全部替换为兼容墨水，色域容积只有原装墨水的70%左右。折中的话，考虑LC、LM、GY、LGY用兼容墨水。

原装墨水用最少程度，黄Y、Violet紫。紫色用量非常少，兼容墨水往往用CMY勾兑，不扩展色域。黄色本身是浅色墨水，但又是CMYK基色，相对容易成为木桶效应的短板，兼容墨水的黄色要么色密度较差，要么为了色密度，结果稳定性不佳，容易断线斜喷。

根据红河纸业的测试 (opens new window)，打印Bill Atkinson测试图，3档质量的墨水用量，该测试图网盘有。第一列粗绒相纸，启用碳黑，第二列为粗面纸，不启用碳黑，粗面纸墨水消耗量增加了10%。P708开3档墨水用量每平米16毫升不到。

根据该测试显示，CMYK+V用量约36%，LCLM用量约24%，GY、LGY用量约40%。

新机最好用原装初始化墨盒初始化防止机器买来就有故障。但是更替颜料墨水存在风险。

更替墨水最好另外准备一套满计数芯片和空填充墨盒，加注清洗液，或参考【稀释墨水】一文，自行配置爱普生同成分清墨水。如果旧墨水是兼容墨水，也可以考虑对应厂家的清洗液，一般每升20多块。

大部分清洗液就是清墨水加表面活性剂，有颜色的一般就是稍微加了点染料。

  • 运行Head ink charge而不是清洗/深度清洗

清洗/深度清洗会刺激震荡墨水通道，绝对不能连续运行，可能导致打印头烧毁。而ink charge只用废墨泵抽墨，且墨水泵可能协同加压。

P708零售墨盒的芯片不能运行initial ink charge，计数只有25ml，但是跑一下要29ml墨水，跑一下芯片计数就变0锁死了。还好有head ink charge。

最好每色充墨90ml清洗液后，静置48小时，再充墨30ml清洗液，之后上新墨水。

我用天威适配大幅面颜料抽墨

天威专利CN112552746A显示，天威用了科莱恩有机颜料，苯乙烯丙烯酸分散树脂。而且不同于友商，天威黑色墨水用的不是碳黑，而是有机颜料黑。不清楚天威该专利是否用在实际零售的墨水上。

大模型说爱普生用的也是苯乙烯丙烯酸树脂分散，科莱恩颜料据流言说，已经是兼容墨水厂家能弄到的比较好的颜料了。不过爱普生的粘结树脂应该是独门秘方且用到界面聚合反应，天威就是再加丙烯酸树脂搅合，反光的烫金效应上应该还是有差距。

根据天威招股书，齐彩、畅阳（耀阳）应该就是贴牌天威，如果更便宜可以考虑，但不一定全是贴牌天威。

烫金效应

颜料墨水在高光纸上（包括钡基艺术纸），反光会泛白、泛青铜色、呈彩虹状，观感不如染料自然。主要产生于灰阶、青色墨水（包括浅青），尤其是CMY混合而成的灰阶。有多级黑灰的机型用兼容颜料，烫金效应也会好一点。

  • 酌情调低浓度减少喷墨量一定程度上可以改善烫金效应，但对CMY混合的灰阶作用很小。

  • 用微喷墨盒抽墨相对兼容颜料可以显著改善烫金效应泛青铜色。

# 微喷颜料墨盒抽墨

爱普生墨盒抽墨，通常我用2.5mm缺口针捅进去往外抽就行。佳能大幅面墨盒抽墨，要额外备1个正常长度的针头，有两个洞，先插额外的针头，再插针筒+针头抽墨，需要额外的那个针头通气。

爱普生微喷墨水体系分代

大部分就是改了个名，过期K3VM，HDR的墨盒完全可以考虑，只是注意

  • 黑色墨水是改动的重头戏，尽量不要混用。
  • Y黄色HD之前和HD之后不要混用。
  • M和LM注意区分鲜洋红和洋红，鲜洋红理论上可以和后出的混用。

**混用有改动的墨水堵头概率很大。**这些都是根据表现和宣传推测的，不排除没提到的有改动的可能性。

  • 世纪虹彩；4600,7600机型。

世纪虹彩K3墨水相对世纪虹彩有巨大改进，从爱普生的披露来看，K3相对世纪虹彩有配方上的大改，据称色密度和烫金效应都有显著提升。后续几代的改动幅度相对要小很多。

价格也可见一斑，K3的过期墨盒一样卖得很贵。

  • 世纪虹彩K3；3800,7800机型，增加浅灰色。

此处改进烫金效应指的是减少反光泛青铜色或彩虹色，同ProX系列的改善反光泛白区分。
