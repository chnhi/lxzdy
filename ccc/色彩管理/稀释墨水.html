# 稀释墨水

喷墨打印机 浅色稀释墨水 爱普生 佳能 清墨水 清洗墨水 空墨水 空白墨水 清洗液 epson canon dilute ink clear ink refill ink

爱普生欧洲安全信息表 (opens new window)；爱普生霉国安全信息表 (opens new window)；佳能安全信息表 (opens new window)披露了墨水大概的成分。

包括颜料墨水在内的浅色墨水色料和深色墨水一样，只是含量不同。理论上浅色墨水可以通过稀释深色墨水来获得。

稀释颜料墨水，可能存在风险

成分没有完全披露，比如色料的披露是XX供应商的商品名，比如颜料墨水有写东方化学的XX颜料商品的，分散剂树脂颜料颗粒等都算进这个商品里。或分散剂和树脂可能写诸如毕克化学的商品名。分散剂受到ph值，墨水成分等的影响很大，到底还能不能发挥作用，并不明晰。是不是有关键成分没有披露，也不清楚。

不过我看东大和老外都有用第三方清洗液稀释颜料墨水嘎嘎打的，应该是可行的。

tips：

  • 文中提到的占比都是按重量算。
  • 稀释颜料墨水最好把深色墨水往稀释墨水里加，边加边搅拌。

# 复刻爱普生清墨水

我根据爱普生披露的主要成分复刻了清墨水，没有加色料。29包邮/1KG，鱼同名“不晓自在”。

只测试过稀释原厂大幅面染料墨水在L8058上正常使用。非常疯狂的是，我按深色墨水占比25%稀释，做出来校色文件相对P10080D的浅色抽墨，色域容积还要大10%，只替换了浅色墨水，深色墨水没换。后续进一步测试深色墨水占比。

披露其给出的含量占比是大概范围，取其中间值。具体参考文首链接或后文的总结概括。

试剂都是淘宝买的，都号称是99%纯度的分析纯，不清楚其中杂质是否有影响。

娃哈哈纯净水

保湿剂-甘油（丙三醇）10%。

界面活性剂-三乙二醇单丁醚10%。

防腐剂1,2-苯并异噻唑-3-酮0.025%。

  • 干扩机染料里有2-吡咯烷酮（CAS 616-45-5）

理论上该成分起到成膜助剂的作用，不清楚是否对干扩机染料有特别作用。

添加2-吡咯烷酮2%，减少三乙二醇单丁醚为8%。

网友测试稀释014墨水正常。

2-吡咯烷酮的熔点是25℃，冬天可以考虑用热水水浴后变成液态，用纯净水稀释成20%浓度的溶液。

  • 稀释颜料墨水

加墨瓶的墨水都会加2,4,7,9-四甲基-5-癸炔-4,7-二醇 。非离子型表面活性剂，可能用来消泡。墨盒产品一般都没有这个成分。蜡状固体，溶解需要60度水温且使用分散机搅拌。

从披露信息来看，颜料墨水基本就是甘油+三乙醇胺，三乙醇胺添加量比染料大，可能需要保持弱碱性保证乳化分散剂发挥作用。披露的浅色和深色墨水成分基本一致，我推测深色墨水内带的乳化分散剂可能够用。

没有测试过，不清楚稀释后不用乳化机搅拌乳化，颜料颗粒是否能正常乳化分散。

根据发明专利的披露，颜料的乳化分散剂通常是不带电荷的高分子化合物，使其吸附于颜料表面，利用大分子链的空间位阻作用隔离颜料粒子，从而实现分散稳定状态。

除了LLK（LGY）会额外添加2,4,7,9-四甲基-5-癸炔-4,7-二醇。可能太稀了需要添加该成分增加分散乳化稳定性。

  • 爱普生水性墨水清洗剂，空气化学104E表面活性剂1.5%，成分类似于上文的2,4,7,9-四甲基-5-癸炔-4,7-二醇。

根据爱普生的发明专利，其打印头清洗液就是不加色料的清墨水加表面活性剂。不过意义不大，墨库的水性墨水清洗液22块1升。

爱普生清洗液配方还加了二甲基亚砜，三乙二醇二甲醚，其中二甲基亚砜具有腐蚀性，三乙二醇二甲醚对呼吸和中枢神经系统有刺激性。该配方还适用于弱溶剂墨水，根据友商的成分，我推测用于水性墨水的清洗液，应该是可以不加这些成分的。加了这些成分可能更猛。

# 复刻佳能清墨水

普通染料，甘油5~10，乙二醇15~20，水60~80。

chromalife染料，甘油5~10，乙二醇10~15，尿素5~10，水60~80。

颜料墨水，甘油1~5，乙二醇5~10，尿素1~5，水60~80。

墨水成分跟墨库的水性墨水清洗液很像，再加点尿素就一样了，不过清洗液里有表面活性剂。

尿素利用电荷排斥效应来达到分散目的，尿素官能团存在孤电子对，颜料表面存在的电荷促使间电位势对抗产生能量阻隔。大分子丙烯酸树脂则是采用立体障碍效应来达成粒子间的分散效果,树脂将颜料包围于内部,利用立体结构原理,产生立体阻碍,而避免颜料粒子间的结块与凝聚,从而提高分散稳定性。

可能chromalife染料的耐候性核心也是成膜树脂，所以要加尿素。

# 猜想

# 干式彩扩染料的核心是成膜树脂

干式彩扩染料墨水中2-吡咯烷酮，可能起到成膜助剂的作用。老外有一种已停产的inkthrift cl ink染料墨水，据称耐候性可以和干式彩扩机墨水媲美，据其介绍，其使用改性聚氨酯树脂乳化后同染料色料一起搅拌，最终使得树脂包裹色料，喷墨后会成膜。

所以我猜测干式彩扩墨水一方面色料耐候性更强，另一方面是墨水里有成膜树脂，喷墨后会成膜。

# 市场太小撑不起高质量兼容颜料墨水

根据爱普生的披露和发明专利显示，分散剂和包裹颜料颗粒的树脂用了很多毕克化学的产品。

无独有偶，老外的ConeColor Pro墨水号称也用了毕克化学的产品，其吹嘘可以和原厂墨水PK，特别是烫金效应。但根据其ICC显示，色域容积只能达到原厂墨水的80%，价格却要原厂墨水的70%，比过期墨盒抽墨还贵。
