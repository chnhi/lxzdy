# 稀释墨水

喷墨打印机 浅色稀释墨水 爱普生 佳能 清墨水 清洗墨水 空墨水 空白墨水 清洗液 epson canon dilute ink clear ink refill ink

爱普生欧洲安全信息表 (opens new window)；爱普生霉国安全信息表 (opens new window)；佳能安全信息表 (opens new window)披露了墨水大概的成分。

包括颜料墨水在内，浅色墨水色料和深色墨水一样，只是含量不同。理论上浅色墨水可以通过稀释深色墨水来获得。

  • 稀释染料墨水，稀释后摇墨瓶一段时间，就可以。包括干式彩扩墨水 (opens new window)在内都正常。
  • 染料清墨水29包邮/600g，鱼同名“不晓自在”。

稀释颜料，补充分散剂后纯色块凑近了细看能感知有水平细条纹，超声波处理后，扫描后放大能看到水平细条纹。

我用过期6年的原装颜料墨盒抽墨，一样指定某个墨水通道打纯色块，也会有水平细条纹，也就是说在微观上，严重过期的墨水可能发生了弱团聚，只是打照片不易感知。

稀释014墨水的GY，供应商Fujifilm有专门的灰色色浆，可能是用不同的CMY发色基团拼配，并且平衡了褪色速率，配合聚合反应，组装成聚集体染料。稀释K墨水得到的灰色墨水可能破坏平衡，不够中性，耐候性肯定会下降，但不清楚具体会下降多少。不够中性通常再掺点彩色墨水可以调整，但整体的耐候性不好说。

# 爱普生染料清墨水

稀释原厂大幅面染料墨水在L8058上正常使用。非常疯狂的是，我初步按深色墨水占比25%稀释，做出来校色文件相对P10080D的浅色抽墨，色域容积直接大了将近四分之一。

我推测主要原因是056墨水可能出于耐候性考虑，明显色域更小，适配大幅面的墨水明显色域广很多。通常号称适配056的墨水从色密度观察也是受限的，如果056墨水机型打算用大幅面墨水，直接用大幅面浅色会由于喷墨程序不匹配而损失色域容积，大幅面浅色对于056墨水机型过饱和，由于据我测试耐候性相近，所以056墨水机型如果打算用大幅面染料抽墨，浅色墨水最好考虑用号称适配056的兼容耐光染料。

披露其给出的含量占比是大概范围，具体参考文首链接或后文的总结概括。

试剂都是淘宝买的，都号称是99%纯度的分析纯，不清楚其中杂质是否有影响。

加墨瓶的墨水都会加炔二醇，可能用来消泡。墨盒产品一般不加，我估计不加也没事。

  • 蜡状固体，溶解需要60度水温且分散搅拌。

  • Surfynol 465，乙氧基炔二醇，提升在水里的溶解度，可能起表面活性剂作用。

# 调节粘度

按MSDS披露配方配置后，在28.5度室温下，用品氏粘度计测量，清墨水粘度约2.35mPa·s，低于爱普生F1080打印头的优选粘度3~5mPa·s。

根据爱普生发明专利，友商专利，我推测可能MSDS中没有披露三乙二醇（三甘醇）。

甘油12%；三乙二醇：10%，约2.8mPa·s。

或使用甘油、PEG-400调节粘度：

甘油21%；约3mPa·s。冷藏一晚后测量，约3.47mPa·s。

甘油10%；PEG-400：5%，约2.5mPa·s。

甘油12%；PEG-400：5%，约2.7mPa·s。

甘油12%；PEG-400：8%，约2.9mPa·s。冷藏一晚后测量，约3.5mPa·s。

平氏粘度计，品氏粘度计，建议买垒固贴牌，上海宝山启航玻璃仪器厂代工的。 宝应哲晶玻璃仪器厂代工、翌哲贴牌环光代工的粘度计，其常数究竟是否是实际测量的我持强烈怀疑态度。用其测量去离子水，并对比水的粘度表，存在巨大偏差，而启航玻璃仪器厂代工的则偏差很小。

# 稀释爱普生颜料墨水

从披露信息来看，颜料墨水基本就是甘油+三乙醇胺，三乙醇胺添加量比染料大，可能需要保持弱碱性保证乳化分散剂发挥作用。

三乙二醇单丁醚在染料墨水中可能起到助溶作用，微喷原装VM颜料墨水会添加1%~3%的三乙二醇单丁醚，可能起到改善喷出性能或界面活性的作用，而VLM通常没有加。

我倾向于按染料清墨水配方，不加三乙二醇单丁醚，三乙二醇加到20%，或甘油加到20%。

兼容颜料墨水、佳能颜料墨水的专利来看，浅色墨水和深色墨水成膜树脂是等量的。

佳能的原装墨水生产路线和兼容墨水相近，而且有CO墨，可以明显改善烫金效益，所以佳能机型完全可以考虑浅色颜料墨水直接用兼容颜料墨水即可。要求高的话专色RGB，深色CMYK直接用原装墨水即可。 我的观点是佳能机型，不如全部用兼容颜料墨水。用原装墨水的成本，不如上爱普生微喷、甚至HP Z9，颜料墨水生产路线不同，且得到GMG支持，佳能没有被支持，我推测是技术上可能有潜在的短板，达不到GMG的要求。

成分没有完全披露，比如色料的披露是XX供应商的商品名，比如颜料墨水有写东方化学的XX颜料商品的，分散剂树脂颜料颗粒等都算进这个商品里。或分散剂和树脂可能写诸如毕克化学的商品名。分散剂受到ph值，墨水成分等影响，到底还能不能发挥作用，并不明晰。是不是有关键成分没有披露，也不清楚。

K3VM导入鲜洋红，HD更新黄色墨水配方，据称黑位更黑。PRO12改善大面积堆黑的反光泛白。

# 稀释颜料墨水可能要补充树脂

据我在L8058上测试，稀释爱普生微喷墨盒抽墨，如果按染料墨水的清墨水配方稀释并搅拌的浅色墨水正常喷墨，但是打纯色块有明显的深浅条纹、断线。

补充同供应商毕克化学的分散剂DISPERBYK-190(有机颜料1.5%~4.5%炭黑2.25%~7.5%)后先把清墨水用磁力搅拌器搅拌半小时以上，最好静置12小时以上后加深色墨水，再搅拌半小时。用浅色墨打纯色块和色阶过渡30cm以上观看距离无明显瑕疵，凑近了细看能察觉有水平细条纹。

样张无明显瑕疵，要特地打凑近了看能感知到，离远点就基本无感。

处理后要敞开静置24~48小时，可以用塑料袋罩住挡灰尘。不清楚具体原理，效果更好。

用磁力搅拌器边搅拌，边用超声波细胞破碎仪超声后，样张基本无感，纯色块感觉好像若隐若现，但定睛一看又看不出来，不知道是不是心理作用。扫描后放大观察，不能算是完美。

扫描图烦请跳转语雀查看 (opens new window)。

我用过期6年的原装颜料墨盒抽墨，一样指定某个墨水通道打纯色块，也会有水平细条纹，也就是说在微观上，严重过期的墨水可能发生了弱团聚，只是打照片不易感知。

从专利看，颜料色浆研磨到100nm左右粒度后，配置成墨水不需要二次研磨。但稀释后疑似弱团聚，我测试补充分散剂无果。我推测可能**要补充树脂。**但这是原装墨水的核心技术，树脂可能是兼容墨水反光怪异的主要原因。

不排除不添加树脂用行星球磨机+0.1mm锆珠研磨可以打散弱团聚，实现完美的可能性。

贸然添加树脂，存在巨大的凝絮风险。

测试过效果不佳的方法：

  • 墨水稀释在溶剂，搅拌后，再边搅拌边一滴滴加水，效果比直接加清墨水搅拌好一点，但还是不能实现完美。
  • 超声波清洗机改善较小，可能是功率密度太低。
  • 装配锯齿状分散搅拌盘的电动搅拌机，1500RPM搅拌跟磁力搅拌器效果半斤八两。
  • 单独用标称20khz的超声波细胞破碎仪+8mm探头，据我测试分散不均匀，必须用磁力搅拌器边搅拌边超声。
  • 高速匀浆机，12000RPM效果不如超声。
  • 15ml离心管+3000rpm漩涡混匀器+0.3mm氧化锆珠，锆珠大部分沉底带不起来。
  • 3D打印简易球磨机+0.3mm氧化锆珠。压根没用。

超声波细胞破碎仪能做到最好的情况是边用磁力搅拌器边搅拌，边用超声波细胞破碎仪超声，样张基本无感，纯色块感觉好像若隐若现，但定睛一看又看不出来，不知道是不是心理作用。扫描后放大观察，能察觉有水平细条纹。

工业上应用超声波都是在线式流量很大，超声功率也很大确保降低粒径，用于小批量处理，不排除局部功率过大的可能性。

超声波细胞破碎仪的标称功率都是吹牛，我这款没有标识，商家吹牛500w，我估测实际声功率约20w。

估测实际声功率，可以用一定量的纯净水，连续超声一段时间测量温升计算。 P=（mCΔT）/t；P声功率，m水的质量，C比热容，水是4.18，ΔT温升，t超声时间。 P/探头面积=功率密度，大模型说功率密度最好控制在20~30w/平方厘米左右。

必须严控温度，猛猛超声，温度太高反而效果更差，最好用冰水水浴。毕克分散剂的pdf说储存温度要低于50度。

据我测试，冰水水浴控制温升，猛猛超声30~40w/平方厘米效果没有4w/平方厘米功率密度效果好。

# 分散剂

大模型说爱普生UltraChrome系列墨水用的树脂是聚氨酯-丙烯酸杂化树脂（空间位阻型）。

根据发明专利的披露，颜料的乳化分散剂通常是不带电荷的高分子化合物，使其吸附于颜料表面，利用大分子链的空间位阻作用隔离颜料粒子，从而实现分散稳定状态。 可能是阴离子丙烯酸树脂。

分散剂同树脂应该是要协同作用的，光补充分散剂，从我的测试来看不够。

DISPERBYK-190 是 聚醚改性苯乙烯-马来酸酐共聚物。

DISPERBYK-2012 也是 聚醚改性苯乙烯-马来酸酐共聚物，但是用了CPT技术。

DISPERBYK-2014 是 环氧胺类。

BYKJET 9151、9152应该和DISPERBYK-190是同类的，9171、9175是聚丙烯酸酯类，用了CPT技术。

大模型推荐9152用于墨水稀释，9171适用于高透明红、黄，C不能用。

毕克化学的printing资料册指印刷，喷墨资料册是inkjet。毕克化学针对喷墨产品有BYKJET系列，但是买不到少量样品，只有工业包装25kg一桶的卖。

我测试过DISPERBYK-190、2012、2014，BYKJET 9152，只能说喷墨上机的表现都差不多，可惜没有粒径仪，可能“微观”上有差异。

毕克化学的inkjet资料册里，DISPERBYK系列适合水性墨水的主要就是190和2014，大模型说环氧胺类分散剂和聚氨酯-丙烯酸分散体系有冲突，所以可能只有DISPERBYK-190是适合的。DISPERBYK-2012是一样的成分，还用了CPT技术好像更先进，但不知道为什么不在inkjet资料册里列举。

毕克化学的分散剂好像是按分子量划分的，这几款就已经是分子量最高的产品了。

大模型还提到了路博润Solsperse 22000/24000/27000分散剂。

# 爱普生水性墨水清洗液

普通清洗液就是不加色料的清墨水，多加点表面活性剂。可以用爱普生墨水披露里用的炔二醇，加1.5%的空气化学104E或空气化学465，在水里溶解度更高。

根据爱普生专利，爱普生有剂猛药，30%三乙二醇单丁醚、30%1,2-己二醇、20%γ-丁内酯、20%纯净水。

还有一个猛上加猛的配方，1,2-己二醇40%+二甲基亚砜40%+20%纯净水，二甲基亚砜具有腐蚀性，风险较大，但更猛。

这种很猛的配方应该需要保持流动，不能长时间在墨路留存，或手动打进打印头，并尽快用清墨水冲洗。

# 复刻佳能清墨水

  • 热发泡甘油最好用liponic eg-1，务必不用醇醚类、少用2-吡咯烷酮等溶剂，避免溶胀风险。

  • ph调节剂可能用二甲基乙醇胺1%。

  • 热发泡室温25度优选粘度可能在2mPa·s左右。

普通染料，甘油5~10，乙二醇15~20，水60~80。

chromalife染料，甘油5~10，乙二醇10~15，尿素5~10，水60~80。

颜料墨水，甘油1~5，乙二醇5~10，尿素1~5，水60~80。

墨水成分跟墨库的水性墨水清洗液很像，再加点尿素就一样了。

尿素利用电荷排斥效应来达到分散目的，尿素官能团存在孤电子对，颜料表面存在的电荷促使间电位势对抗产生能量阻隔。大分子丙烯酸树脂则是采用立体障碍效应来达成粒子间的分散效果,树脂将颜料包围于内部,利用立体结构原理,产生立体阻碍,而避免颜料粒子间的结块与凝聚,从而提高分散稳定性。

稀释兼容颜料墨水、佳能颜料墨水的专利来看，浅色墨水和深色墨水成膜树脂是等量的。稀释不太可能，贸然添加树脂有很大的凝絮风险。

佳能的原装墨水生产路线和兼容墨水相近，而且有CO墨，可以明显改善烫金效益，所以佳能机型完全可以考虑浅色颜料墨水直接用兼容颜料墨水即可。要求高的话专色RGB，深色CMYK用原装墨水。

佳能是兼容颜料墨水的难兄难弟。用原装墨水的成本，不如上爱普生微喷、甚至HP Z9，颜料墨水生产路线不同，且得到GMG支持，佳能没有被支持，我推测是技术上可能有潜在的短板，达不到GMG的要求。

# 佳能清洗液

根据艺美佳的专利 (opens new window)，热发泡打印头的清洗可能需要除去发热体上的结垢、焦烧物、附着物。

其使用的表面活性剂是陶氏15-s-9，设计了三种清洗液。

第一种成本低一点，10%乙二醇+1%表面活性剂，用于清洁墨路、打印头内残留墨水，洗到废墨清澈。不在乎成本可以用清墨水。

第二种是功能性清洗液，高浓度三乙醇胺40%，辅以二乙二醇丁醚20%。

第三种既清墨水10%乙二醇+5%甘油+0.5%表面活性剂+2%尿素+0.5%三乙醇胺+防腐剂0.05%。

# 猜想

# 干式彩扩染料的核心是成膜树脂

干式彩扩染料墨水中2-吡咯烷酮，可能起到成膜助剂的作用。老外有一种已停产的inkthrift染料墨水，据称耐候性可以和干扩染料媲美，据其介绍，其使用改性聚氨酯树脂乳化后同染料色料一起搅拌，最终使得树脂包裹色料，喷墨后会成膜。

所以我猜测干式彩扩墨水一方面色料耐候性更强，另一方面是墨水里有成膜树脂，喷墨后会成膜。

# 市场太小撑不起高质量兼容颜料墨水

老外的ConeColor Pro墨水吹嘘用了原厂墨水同类树脂技术，特别是烫金效应可以和原厂墨水PK。但根据其ICC显示，色域容积只能达到原厂墨水的80%，价格却要原厂墨水的70%，比过期墨盒抽墨还贵。

# DURABrite与UltraChrome系列墨水的差异

不清楚是否是否是为了优化烫金效应，DURABrite系列墨水可能是类似于兼容颜料墨水，成膜树脂与颜料一起搅合。UltraChrome系列墨水可能是界面聚合，既两种单体分别溶于水、“油”中，在交界面发生聚合反应，形成壳层包覆颜料颗粒，构成自分散“颜料胶囊”。
