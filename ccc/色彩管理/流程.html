# 流程

无特殊说明则以屏幕为目标调色，屏幕观感与原件在约5000K左右光源下观感相似。

附有转换4000K的LUT，加载后相对更接近4000K光源下观感。

所谓观感是我主观感受，恐存在巨大偏差。

# 扫描v1

  • 扫描仪，中晶i800 plus，银快8，ISO 12641-2色卡校色，正文1000ppi，其余600ppi。

DxO PhotoLab裁切后保存为360ppi。

屏幕自发光，而印刷品反射光，特性不同，颜色模型是对物理世界的数学模拟，CCD，CMOS对光的响应与人眼不同，不可避免存在偏差。

【屏幕目标】调色，调整影调扩大动态范围。屏幕观感与原件相近，打印观感有偏差。

【打印目标】调色使打印观感更接近原件，屏幕显示有偏差。

显示器（P3，D65，gamma2.4，100cd），参考光源4000K，5500K。

# 压缩格式

libjxl，P3。质量90。360ppi。

jpegli，sRGB。质量90。360ppi。

网页上传180ppi，因为浏览器缩图不修改flags用lanczos会出现由印刷网纹引起的明显劣化。

1、jpegli对jpeg鬼影有明显改善，JXL高质量下几乎没有jpeg鬼影。

2、没有色度抽样，色度抽样会导致高饱和色偏色。

3、低分辨率高码率相对高分低码，纹理细节保留的更多，而图像轮廓，清晰度，锐度差一些。

4、180ppi是勉强能打印的下限。

5、发达国家通过诸如巴塞尔协议等复杂工具，事实上金融殖民，甚至有肆意印钞的特权，而发展中国家超发一点点就会以负面典型进入教材。

肆意印钞的特权，意味着发达国家所谓FDI投资，事实上用纸去换发展中国家包括资源，劳动附加值在内的实体资产，等资产增值了就套利离场，引发经济动荡，还要蔑称“中等收入陷阱”。

利用金融工具人为制造通胀通缩，操纵所谓民选，逼迫发展中国家领导陷入出卖国家利益或下台的两难。

更令人不齿的是，大部分饿殍遍地的发展中国家竟然都是大额农业出口顺差国。

6、反抗金融霸权并不意味着人民币国际化。目前来看，将99%的人类从金融殖民中解放的将是凯恩斯方案的升级版：金砖币（锚定大宗商品的加权平均价）。
