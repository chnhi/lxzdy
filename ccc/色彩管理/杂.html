# 杂

# 颜料与染料

染料的通透感，亮光感是颜料无法实现的。观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。缺点是易褪色。

同时染料由于色不稳定，色准和色阶都受到限制，有层次上的损失。

染料黑光谱有明显的红色偏移（看起来泛红），既660nm以上波长的可见光反射率上升。

颜料优点是不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹），输出大面积黑色时，由于光线漫射，会产生烫金效应，显得不够黑。 (opens new window)

染料机型的灰阶需要彩墨中和，由于同色异谱现象，在不同光源下，容易一会发青一会泛红。没有多级黑灰的机型换装颜料墨水会凸显灰阶的短板，暗部层次非常差，死黑。

# 染料墨为什么易褪色

染料墨水的着色剂直接暴露在空气中，在光线和臭氧等的作用下，染料着色剂的化学链可能断裂，造成着色剂变色等现象，使打印稿褪色。一旦打印稿遇水，染料着色剂将再次溶解，并随着水的流动向四周扩散，造成严重的晕染现象。

颜料墨水的着色剂是一些固体小颗粒。为了使这些固体着色剂在墨水中均匀分布，还要加入乳化剂，使着色剂外部包裹一层树脂，借助树脂层上少量的亲水粒子，使墨水乳化，着色剂悬浮在墨水中。由于着色剂并不溶解于水，当打印稿遇水，着色剂并不随水迁移，其抗水性能自然有很大的提高。着色剂一般为高度粉碎的矿物质，其自身的稳定性就非常不错，加上外层包裹有树脂层，当遇上阳光和臭氧等时，首先破坏的是外面的树脂层，而内部的着色剂不受影响，自然不容易产生褪色现象。

# 颜料墨为什么饱和度低

颜料墨水的着色剂不溶于水，在水中借助于分散剂悬浮，颗粒直径比染料墨水中的着色剂分子大50～100倍。与染料墨水渗入到介质再形成色彩的原理完全相反，它是通过着色剂附在介质表面发色，这导致其混色时并不能充分混合，不能充分发挥对可见光波选择吸收的特性，导致其出现同色异谱现象和烫金效应，光线漫反射现象。

# 天威专利

天威专利CN112552746A显示，天威用了科莱恩有机颜料，苯乙烯丙烯酸分散树脂。而且不同于友商，天威黑色墨水用的不是碳黑，而是有机颜料黑。

大模型说爱普生用的也是苯乙烯丙烯酸树脂分散，科莱恩颜料据流言说，已经是兼容墨水厂家能弄到的比较好的颜料了。不过爱普生的粘结树脂应该是独门秘方且用到界面聚合反应，天威就是再加丙烯酸树脂搅合，反光的烫金效应上应该还是有差距。

天威24年招股书显示，24年1~6月天威采购颜料，珠海云旦66%，6%的大坑逼苏州中徽。广东中庸商贸占比24%，应该是颜料颗粒。DIC颜料1%，富士0.4%，上海创舟0.6%不知道卖什么。

不知道广东中庸跟上海创舟卖不卖科莱恩有机颜料。

# 爱普生大幅面染料抽墨

P10080D（T42V）叫活的色彩NDD。

P11080D（T53Y），T5680D（T54A）改名叫活的色彩DS6，意义不明，海外的UltraChrome DS6是热升华墨水。

其中T42V需要直接倒出来，另两款需要抽墨工具抽墨，就是套在针筒上，变得更细的一个转接头。

P11080D和T5680D的墨水应该是一样的，因为这三个机型说明书是同一个pdf文件：SC-P11080D_SC-T7680D_SC-T5680D。

P11080D机型介绍页面的html源码，对墨水介绍的备注是活的色彩NDD，第一版宣传PDF也是活的色彩NDD，后面悄悄改名DS6。

染料墨水的特性是一样的，但是原厂染料的耐候性，色域还是比兼容染料好很多。

  • 耐候性跟爱普生T674，056墨水差不多，跟L8168用的014-claria墨水有差距。
  • 不覆膜暴露于空气会褪色泛红，覆膜后曝晒还是会褪色泛红。
  • 染料黑都有的红色偏移，既660nm以上光线反射率上扬。

兼容染料包括所谓抗UV墨水，避光3~6个月褪色，见光1个月，曝晒3天就褪色发红；覆膜避光1年褪色。

原装染料，能坚持两三倍或更长时间，但最终一样褪色；覆膜避光基本不褪色。
