# 年化收益低于10%就是贬值

  现代货币体系是无锚的，遇到危机会疯狂印钱。08年至今美联储资产负债表年复合增速18%左右，美元资产年化收益率达不到18%是在贬值。

   中国资产负债表年复合增速也在18%左右，但减去外汇占款，中国净主权信用货币的年复合增速在10%上下，人民币资产没有10%的年化收益率是在贬值。

  任何产品年化收益率比10%还高，就意味着风险很高，比印钱还来钱。

# 基金101

  • 基金除了有申购费和赎回费，还有管理费，托管费，销售服务费的成本，每天扣除。
  • 主动型基金买入尽量挑熊市。牛市熊市可以参考果仁市场因子 (opens new window)
  • 投资股票的基金的有仓位限制，即使市场有泡沫也不能全部撤出，要自己赎回避免损失。
  • 尽量分散投资。

# 稳健收益

  由于潜在的超发引发的严重通胀风险，务必避免长期持有较低利率的固定收益产品特别是理财型保险，5年定期等。

通常年化利率 推荐买入时机
定期存款 1.5%(1年)2.75%(3年)2.75%(5年) 血亏，想都不用想。
宝宝类如余额宝，货币基金 2%~3% 日常开销
短期理财产品(半年以内) 3.5%~4% 日常开销
中期理财产品(半年~一年半) 4%~5% 稳健
短债基金 2%~5%(要扣去0.4%-0.6%的成本) 日常开销
中长债（纯债） 1%~20%(要扣去0.4%-1%的成本) 十年期国债收益率历史高位
黄金，比特币 短期波动大，投机成分大。 美国佬降息或有通胀预期

  短债基金收益率略强于宝宝但是比理财产品差一点。中长债上限高，恰好在十年期国债收益率历史高位，推荐买入，缺点是要长期持有才会有一定收益，稳健型基金中唯一有持平通胀或者小赢通胀的可能性。

  积极型债券相对中长债风险提升率从历史战绩看是比收益提升率小的。

  对冲基金从过往数据分析，风险和长债差不多，收益不如，成本还高，除非股债双熊的时候求稳才考虑。但这个时候买主动基金的收益期望我认为是大于风险期望的，再跌也跌不到哪里去。

  黄金和比特币等在美国佬降息或有通胀预期的时候用于避险，短期波动大，投机成分大，降到历史低位也可以考虑买入长期持有。

# 主动基金

通常标准差(%) 通常夏普比率
可转债 12-18 0.85-1.1
积极债券 8-12 0.9-1.3
积极配置 16-30 1.2-1.6
指数基金 15-32 0.8-0.95
  • 标准差反映基金波动幅度，越大代表波动越大。
  • 夏普比率代表基金承担单位风险得到的超额回报率越高。

可转债波动较大，短期收益是相对比较高的。从数据上看，指数基金的风险跟积极配置差不多，唯一优势就是持有成本低一点，收益却不如积极配置。

  • 注意尽管货币超发，但不一定会流入股市，市场有很多，比如大宗商品，汇市，楼市。
  • 常年年化收益率20%上下的都是大C。

# 借钱加杠杆

  互联网上那些借钱的软件看起来每日利息似乎没多少，务必要换算成年化利率仔细对比，通常要5.5%到22%。
