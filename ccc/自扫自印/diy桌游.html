# 硬挺度

官正诸如珍藏版典藏版那些之所以更硬挺，是因为涂布了上光浆料，会放硬挺剂，所以350g相对DIY的300g纸张硬挺很多，DIY想要实现较高的硬挺度，可行的方法是覆加厚的PET膜，不过据z佬说观感会更暗。

# 切卡

三国杀：62x87mm-R3mm
标准扑克尺寸： 63.5x88.9mm，（2.5寸x3.5寸）
有一些网站标注万智是63.5x88.9mm，但据老外分享，DIY做63x88mm正好。

  • 铡刀型切纸刀+省力型倒角钳，成本低，就是费事，而且往往明显尺寸不一。

  • 定制重型冲卡机， 淘宝·印后 相册 广告设备专销店 (opens new window)，是我找到报价最低的，200+12邮费，可能还有更便宜的。

重型冲卡机，出血留大了目视对齐容易歪斜，想要避免歪斜，目视对齐的一面出血要小，靠白边的大小来判断是否歪斜，然后平移消弭白边。缺点是要注意力集中，也是比较费事的。

  • 定制电动名片切卡机+手动圆角机相对省力很多，主要是不需要集中注意力。只是机器成本要900+200。

小型机器只能切不覆膜的350g纸。能切覆膜纸的定制7.8K，切出来是直角，需要手动圆角机切圆角，由于R3圆角太小，稍微有点误差视觉上会比较明显。主要表现是倒角不对称。

切卡机最好搭配印刷彩印。只能切不覆膜的纸张，喷墨自印必须要颜料墨水保证耐候性，搭配颜料墨水可以不覆膜的纸张表现力会差一些。

7.8K的切卡机我觉得没意义，不如2~3万左右定制的自动冲卡机，人工上料，需要人工对准1个激光参考锚点，冲卡是倒好角的。鱼“光信制卡厂”。

  • 印刷厂用模切机

核心原理是刀模，然后机器施加压力，有直上直下的，有滚筒施加压力的。进纸有人工上料对准的，有人工上料后自动对准的，有自动上料的。

搜索关键词：扑克自动冲卡机，早教卡自动冲卡机。

扑克牌印刷厂看视频有些先切成出血大小，然后叠一起用同一个模具模切，不知道是不是为了一致性。

搜索关键词：商标液压模切机

# 覆膜·塑封

  • 冷裱膜是pvc，特别厚，而且很软，手感像套了个手机软壳。
  • 覆膜（热裱膜）普通bopp预涂膜不防刮，指甲可以划出白痕，有些膜不顺滑。
    • 光膜是镜面反射，会成像，和原卡差异较大。
    • 哑膜几乎没有反光，同原卡有差异。
    • 防刮BOPP哑膜手感会有一点粘腻，会有一点新的银行卡上面贴的保护膜那种感觉，没有质感。
    • 阿里巴巴天宝印刷，一款21柔光膜，一款微米30哑膜。光泽度介于哑膜亮膜之间，柔光膜会成像偏漫反射，观感和过光油原卡相对接近些。哑膜则是有一定光泽度的漫反射，相对比较接近覆膜原卡。
    • **DIY建议PET哑膜，可以改善走纸印和棘轮印。**相对稍微更硬挺点，也不怕刮。
    • 淘宝 艺都千帆塑封工厂，直接联系客服买PET哑膜。有加厚的，更硬挺。不过据z佬说观感会更暗。
  • 印刷厂液压覆膜压力大，还会过压纹机，切割后还是很服帖，diy覆膜可能翘边，起膜。
  • 印刷厂过压纹机可以做凹凸手感和纹路；diy定制十字布纹预涂膜，只有纹理没有凹凸手感。

小型台式覆膜机，新机器看看淘宝同一数码科技的机器便宜点。没有液压机劲大，尽量买可以调速的，可以把速度调慢点来改善效果。

# 塑封膜

  • 无论是覆膜还是塑封，都是热熔胶合膜层和打印面，开胶撕掉会把打印的图案撕走。

    我定制了【72x99】10丝在售，用于【无损塑封】，既套牌套塑封，牌套的封口对准塑封膜的封口。卡牌可以无损拆出，但是可能有气泡，拿在手里，或轻捏卡牌都能明显感知到有空洞感，牌和牌套之间是空的，而牌套已经黏住塑封膜。

  • 塑封膜不足8丝留边像照片纸的硬度，10丝更硬一点，8丝折中。

    • 商家塑封套餐一般用6~8丝。

  • 推荐大厂艺都千帆定制（淘宝或阿里巴巴），10包起订，算上邮费，10丝7块多一包，8丝5块多一包。

大厂倒角圆润，热熔胶涂布均匀，收缩率低，透明度高。

  1. 把卡牌推到封边底，塑封后约4mm留边。
  2. 封短边则上下留边居中比较稳定，左右目视对齐，无法避免偏差。封长边则左右留边比较稳定，我更倾向于封长边。
  3. 塑封膜外径主要影响留边边距，我定制一般四边都留边4mm。
  4. 三国杀官正出血四边都是2~3mm，所以我倾向于留边四边一致。
  5. 通常留边至少3mm，但有一边是目视对齐，稳妥起见预留7mm，避免误差，留8mm。

三国杀卡牌【62x87】，St尺寸（万智）【63x88】。我会定制三国杀【70x95】，St【71x96】。

# 塑封机

几十块塑料壳的塑封机只有两个辊不如铁壳的4辊平整。

加热方式分为石英管和云母片，石英管是红外灯管。石英管分为内加热和外加热，云母片暂时只有外加热，内加热是把加热管装在胶辊里。外加热是加热管在胶辊旁边加热胶辊。

  • 内加热更均匀，但是加热速度慢，胶辊贵。

  • 云母片加热速度快，预热时间短，传热效率高，连续作业效果好，但是成本高。

    • 200￥雷盛LM320-S(T)内加热，塑料齿轮，19mm胶辊。

    • 350￥平达330C，云母片，金属齿轮（跟客服要云母片的）。

    • 500￥平达330CN是内加热，金属齿轮，25mm胶辊。

    • 720￥平达330TD，云母片+可调高速，适合量很大。

量小不在乎速度慢尽量买内加热，量大尽量买云母片+可调高速，石英管外加热温度不均匀，容易出瑕疵。

塑封机并不是绿灯亮了就可以塑封了，绿灯亮只代表温度传感器检测的点达到设定温度，并不代表整根胶辊的温度都够了。所以使用中可以考虑先调到80~90度，绿灯亮起后预热5分钟左右，再调到需要的温度，再次亮起再酌情开始塑封。温度尽量不要猛猛调高。

# 常见塑封异常

雾状白点，不粘，温度不够（设定过低或胶辊温度不均匀或绿灯没亮）。

泡点（一点一点，点之间有一定间距）温度过高或胶辊需要清洁。

起皱不平整，温度过高或放置不平整。

留边不粘或起泡（类似于手机贴膜有气泡），温度不够或胶辊压力不够，有两个有弹簧的螺丝拧一下。

# 打印机

喷墨打印前要进行输出锐化，qimage ultimate软件打印的自带锐化对白字的处理比较好。PS下可以用pixelgenius的PK滤镜，这个输出锐化滤镜是识别高对比反差，白字描黑反而劣化效果。网盘-软件内有。

家用打印机都是RGB打印机，驱动接收RGB编码，不需要改成CMYK。由喷墨程序将RGB编码转换为CMYK喷墨量。要效果好应进行色彩管理，请参考 (opens new window)用扫描仪校色打印机。

喷墨打印机校色后色域容积远大于CMYK。

  印刷厂如胶印是CMYK四色油墨分别刷一遍，调整CMYK通道可以优化效果，比如把四色黑改成单色黑可以避免重影。

  • 原装颜料太久不打都可能堵死喷头，清洗清不通，至少每周打一次可以是喷头检查图。
  • 颜料最大的优点就是色稳定，不褪色，色准高。
  • 但可能有凹凸感（高光纸表面明显墨水颗粒的痕迹）
  • 由于光线漫射反光会产生烫金效应，反光泛白，泛青铜色，观感怪异。尤其是C、LC墨水以及CMY混合而成的灰阶。
  • 但是对于自印没有特别合适的纸张，详见后文。
  • ---
  • 染料就算是兼容墨水都不会堵死，只会遇到打印头干了，清洗能清通。
  • 染料色不稳定，易褪色，色准上有细微差距。
  • 覆膜或塑封可以明显改善其耐候性。
  • 染料观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。
  • 具有通透感，亮光感是颜料无法实现的。

  • A4单面打印，爱普生机型通常共消耗墨水约0.75ml左右，佳能热发泡可能会多消耗一些。
  • 爱普生大幅面染料抽墨，活的色彩DS6、NDD耐候性同兼容耐光染料差不多。

染料褪色第一难题，灰平衡。

染料墨水机型灰阶主要由CMY墨水混合，而CMY墨水褪色速率不一，这往往导致灰平衡破坏的非常快，DS6墨水短短3天，中性灰块的dE00就超过2.5。

但是灰平衡如果没有参照对比，不一定明显感知。

染料C墨水褪色速率最快

以色密度计算，C墨水褪色速率达到M墨水的2.5~4倍。如果无视灰平衡，把C墨水替换成耐候性更好的墨水，可以明显改善整体的耐候性。

染料黑色墨水褪色发红

不仅堆黑显眼，而且会参与到彩色增加色密度，跟C墨水一起劣化整体发红的趋势。干式彩扩黑色墨水褪色更偏向饱和度下降显旧，不至于那么显眼。

如果打算用染料墨水，最好做保护，覆膜、塑封、淋膜，保护后可以不避光。

我更倾向爱普生机型，折腾空间大，详见后文墨水介绍。

爱普生一般4万张左右打印头状态不佳，4万张之后基本要用最高质量加大喷墨量掩盖瑕疵。惠普对热发泡商用大幅面的建议是每色1.5L墨水换打印头，既4500~6000张左右A4幅面照片更换打印头。一般diy很难用掉那么多墨水。但是佳能打印头容易买还便宜，爱普生打印头坏了，不如买新机。

爱普生机壳不值钱，打印头值钱，尽量避免二手爱普生。爱普生的二手机，大都是商用的，打了几万张，干烂的机器，打印量写在主板上，看似打印量很少的机器，打印头有没有被掉包都不好说。商家收机器，打印量很少的好头，一般600~800可以卖给白墨烫画，UV打印之类的商家。跟着机壳一起卖的往往都是烂泥扶不上墙的打印头，基本上寿命都很短。

爱普生尽量买L8058、L8168新机，其他机型连续打厚纸可能不够稳定，L426X机壳打不了300克厚纸。

折腾墨水考虑L8058。原装墨水选L8168，耐候性显著强于L8058的原装墨水，色域也明显更广。

L8168机型的走纸印在中间，避不开，L8058的走纸印在两边约1cm左右，可以避开。新出货的L8168有可能也改了，他们机壳很多设计都是通用的。

**佳能机型的走纸印，棘轮印要比爱普生好很多。**不过覆PET哑膜后浅色基本感知不到，深色可能感知到。

佳能有三类，E468、TS5380墨盒喷嘴一体，TS8380墨盒喷嘴分体，G580墨盒喷嘴分体+墨仓。还有很多停产的二手机，TS9020、TS8020、TS5080是墨盒喷嘴分体。

佳能优先全新G580，1000出头就导入了灰色墨水，比爱普生便宜，还有墨仓，加墨省事。

佳能这几个TS的机壳同一个前身，能进厚纸。E478贼便宜，也是后进纸，但机壳前身mg3080打不了厚纸。便宜机身考虑TS5380，喷嘴折腾坏了买个新墨盒打洞就行，墨盒容量实在太小，最好配合三仓连供，按说明处理气压平衡，不然稳定性不太理想，不过总归没有墨仓稳定省心。TS5380高质量彩打既选光泽照片纸，黑墨不喷墨，是CMY混合的黑色，相对不黑，而且因为CMY褪色速度不一样，打出来的黑色褪色更快更显眼。

TS9020、TS8020、TS5080等几个二手机比较便宜，但能不能买到靠谱的打印头非常看运气，往往可能买来是好的，但连续打印会掉链子。

# 折腾墨水：

  • 不打算用原装墨水可以买拆墨空机。

换任何墨水都必须校色。扫描仪校色打印机 (opens new window)；校色仪校色打印机 (opens new window)；

如果墨水更匹配喷墨程序，往往灰阶相对更中性，校色事半功倍。天威、另色鬼号称适配XX型号具体适配度如何，我也不清楚，通常墨仓机型另色鬼影像墨水耐候性比天威稍微强点，天威对墨仓墨水都不定位“影像级”。

L8058的原装056墨水，耐候性相对014墨水有较大差距，但是价格却不便宜，我觉得有点小坑。

折腾墨水的话用扫描仪校色的灰阶相对可能更差，用光度计也要色块多一点，至少TC918。网盘 (opens new window)/校色色靶/色靶参考/下有我提供的色块集，有限个色块但着重灰平衡，更适合折腾墨水。

**买任何所谓原装墨水，拆机墨水，**务必在爱普生或佳能官网能查到的店铺购买。真正卖拆机墨水的很不好搜到，而且倾向于整套一起卖。一搜就搜到，出货量很大，贼便宜的拆机墨水99%是假货。

  • 不在乎成本用干式彩扩机墨水，耐候性好很多。
  • 兼容“耐光抗UV”染料避光或保护，见光无保护2~4周褪色。
  • C墨水褪色速率最快，用好的可以改善耐候性。

保护既覆膜、塑封、淋膜，压在玻璃、亚克力等材料下也可以，保护后可以不避光，可能是光子激发反应时，还需要氧气参与反应。

**干式彩扩机墨水：**L8168(014)，D580（墨瓶）；D700，D880，D1080（墨盒）。富士墨盒。浅色墨水（GY，LC，LM）可以自行稀释 (opens new window)。

兼容染料色域不差，但是耐候性太差。必须买抗UV耐光染料，普通染料一天就变色。

兼容“耐光抗UV”染料据我测试 (opens new window)，所有品牌都没有质变，哪个便宜买哪个。适配056墨水以外的墨水通常色域明显更广，如适配大幅面或老型号R330，原装056可能为了耐候性限制了色域。深色墨水扩展色域，浅色墨水用适配机型的显色性匹配更好可能可以减少校色后色域容积的损失，但也不绝对。

偏重耐候性，天威R330影像级染料，兼容成膜染料。颜料墨水是成膜树脂喷墨后成膜，把颜料颗粒粘在纸上，既堵头往往是树脂干了成膜板结或意外凝絮。兼容染料也可以加成膜树脂，喷墨后成膜，一定程度上减少染料分子同空气、光线的接触，见光的耐候性相对普通兼容耐光染料有明显改善。避光保存的耐候性是否会受限于色盐本身，犹未可知。其堵头的风险是否同颜料墨水相近，亦不知。

L8058用天威R330影像级染料可以测试看看我提供的ICC (opens new window)。

偏重打印头寿命，我考虑色如丹色盐的染料，色如丹是爱普生供应商，色盐纯度应该可以。色如丹目前也建设了墨水产线，但不清楚哪些品牌贴牌。艺美佳同色如丹交叉持股，我推测艺美佳染料可能是色如丹的色盐。根据色如丹2022年的招股书，另色鬼、天威、墨库也都买过色如丹色盐。等色如丹下次IPO再看看。

黑色墨水最好考虑另色鬼影像染料、鸿盛EverNew、天威R330影像级染料。另色鬼，鸿盛褪色发黄显旧，天威这款褪色偏发青，饱和度下降，变色不明显，但是成膜染料。

兼容颜料校色icc色域容积达到爱普生原装颜料自行校色的75%左右，但是自印卡用颜料墨水，没有特别合适的纸，具体参考后文，各有各的短板。

兼容墨水没什么好折腾的，厂家没有做好墨水的市场空间，买大厂便宜的就行，如天威、鸿盛、墨库（百度西）。墨库出货量大还便宜，但为了商用连续作业的稳定性加挥发性溶剂，味道很大。

  • 颜料墨水有烫金效应（反光泛白泛青铜色），观感不如染料自然，尤其是C墨水和CMY混和的灰阶。

  • 原厂颜料可以改善反光泛青铜色，1~2￥/ml。

  • 覆膜或塑封，注意热熔胶寿命不算理想，说不定还没褪色先开胶了。

  • 颜料至少每周打1次可以是打印头检查图，太久不打可能堵死。

  • 兼容颜料低温粘度变高，易斜喷。

  • 热发泡尽量不要用颜料墨水，堵死没得修。爱普生打印头的堵头存在维修可能性，但不要因为能修就存在侥幸心理。

  • 原装粗面黑在哑光纸上黑得多，但是不耐刮擦，不覆膜不建议用原装粗面黑墨水。

相对爱普生10色机，4~6机型最大的短板是灰阶靠彩墨混合，P708不需要无关墨水增加色密度，靠对应颜色配合黑灰墨水就可以。而6色机型，往往需要无关墨水中和增加色密度，更容易因为同色异谱而产生色偏。即使都用光度计校色，如果对样张非常熟悉，还是看几眼能感知色准上的差距。

染料王就是L8168+014墨水，匹配喷墨程序，而且染料深色墨点耐候性相对浅色墨点更好。

详细了解请参考【色彩管理-打印机】 (opens new window)

# 纸

  • 印刷厂俗称铜版纸的纸张，准确来说是 印刷涂布纸coated art paper。

没有喷墨涂层，喷墨打印机要用铜版纸颜料墨水（快干颜料墨水），最好改装加热装置。墨水掺了溶剂有腐蚀性，不仅影响打印头寿命，而且低温下粘度会变高，打印容易斜喷或断线。

  • 喷墨用的” 高光铜版纸 “，准确来说是“喷墨铜版纸”inkjet coated paper。

有喷墨涂层，可以吸收更多墨水增加色密度，色彩表现力相当不错。但是：

  1. 涂层也计入克重，相同克重下更薄更软。
  2. 反光能感知走纸印和棘轮印，两张卡摩擦有白痕，必须要保护。
  3. 覆PET哑膜可以改善走纸印和棘轮印。
  4. 只建议用染料墨水，颜料墨水的烫金效应会导致深色反光光感与其他颜色不同，显得怪异。
  • ”白卡纸“。有350g，400g。

白卡纸克重就是纯纸，最硬挺。但是靠纸基吸墨，色密度严重受限，影响表现力，由于墨水被纸张吸收，会泛纸张本色，显得色密度不够。颜料要原装粗面黑才比较黑，但是不耐刮擦，手一摸就变黑。

  • 哑光喷墨铜版纸，matte inkjet coated paper。

所谓“哑光铜版纸”有两类，一类实质就是白卡纸。另一类有涂层。有涂层的我买到过两种：

1、骏马纸业代工，很多小牌子的所谓哑光铜版纸就是贴牌。这个纸我认为可以称为“丝光彩喷纸”。说是哑光，实际上还是有轻微的反光。相对高光铜版纸，走纸印和棘轮印有很大的改善，基本要反光细品才能感知到有走纸印。但是表现力相对高光铜版纸要稍差些。

这个纸经常本身涂层上就有斑驳的痕迹，覆膜后会很明显。

必须覆膜，不覆膜切割后，切口处涂层会有皲裂。覆膜后染料颜料都可以用。

2、我称为”哑光彩喷纸“，有涂层但是不会反光，表现力相对“丝光彩喷纸”要差。我买到过力武的“哑光铜版纸”是这类。

走纸印和棘轮印相对“丝光彩喷纸”改善一点，基本要浅色部分反光细品才能感知。搭配颜料墨水，黑色墨水堆积的地方会有反光，其他地方又没有反光，反光就会有一点斑驳。这款纸如果天气潮湿，颜料墨水干的慢了，会有明显走纸印。

  • 增强粗面艺术纸

走纸印和棘轮印相对哑光铜版纸还要进一步改善，不怎么能感知到，但是由于是涂层和纸基一起吸墨，所以精度锐度相对会差一些，成本还高。

搭配颜料墨水没有烫金效应，观感比较“自然”温暖，但是要写实的作品才能体现，自印卡基本没有。

  • RC细绒相纸搭配颜料墨水有非常强的表现力，但又软又薄，可以考虑白条卡或塑封。

RC相纸色域最广，广很多。纸张和涂层之间有PE膜隔绝，完全是涂层吸墨，不易感知走纸印，细品可能感知棘轮印。

  • 莱妮纹 珠光纸，布纹效果。
  • diy局部镭射闪卡请参考白羽ことり (opens new window)的分享 (opens new window)。其用到的镭射透明不干胶，透明pet等可以打颜料墨水。

工业墨水：弱溶剂墨水，乳胶墨水，涂料墨水（DTF烫画墨水），UV墨水等，腐蚀打印介质或通过不同原理固化成膜，极易堵头，还具有强腐蚀性，非工业打印头寿命按天计算。

# DIY-PVC卡

DIY可行的PVC卡有两种，一种是PVC免层压，另一种是喷墨PVC白卡。

PVC免层压是PVC卡上涂有热熔胶，打印在PET透明膜上（有喷墨涂层）然后过塑封机，把PET透明膜和PVC卡胶合，然后用重型冲卡器冲卡。热熔胶寿命不长会脱胶，甚至刚做完就起泡。

L8058机型有PVC证卡托架，可以打印在PVC白卡上，必须是喷墨PVC白卡，涂布了喷墨涂层。缺点是只有86x54尺寸。最好用颜料墨水。

88x63尺寸我估计应该是可以打的，只不过喷墨白卡和托架都要定制，不知道在哪定制托架。

L8168机型有一个后部进纸槽，把屁股掀开可以平行进纸。网上搜 L8168 打印证卡看看视频，自己3d打印一个类似的托盘，自己在InDesign软件微调出适配的模板文件。

# 测试不适合diy的

# 切卡

  手动模切机配刀模，挨得太近压力不够会撕扯打印面，而且免不了会有一点位移刮伤打印面，感觉电动模切机也够呛。

  刻字机，要打标记点给机器巡边，A4只能切6张卡，而且机器不能校正角度偏斜，要自己对齐摆放，会有刀口毛刺，而且切割要有垫板保护刀头，垫板有粘性，所以必须要给纸覆膜，或本身就是不干胶有背纸，不然打印面会粘在垫板上。

# 保护

  • 双面RC细绒相纸+水性光油。

  水光油用丝印尖口刮刀在RC相纸的打印面刷一遍，不需要丝印网板，就能实现不错的光泽度和顺滑的手感。裸卡重叠在一起相互摩擦，分离非常顺滑，但是相对原卡明显薄软很多。RC绒面观感更有质感，但与原卡明显不同。

但手动涂布免不了出现瑕疵。

RC相纸表面有一层淋膜，防水，水光油不会马上成膜，而且是孔隙结构，用水光油中的树脂颗粒填充孔隙后，光泽度好，手感顺滑。铜版纸是一层蜡，手动涂布不够均匀，成膜效果不佳。

  • uv光油

  uv打印机打光油，手感不润。手动涂布光油墨水会有油性物质渗进纸张，不清楚是溶剂还是稀释剂之类的成分。

  UVC光油，丝印尖口刮刀不需要丝网，可以薄薄的挂一层，要用高压汞灯，对皮肤，人眼都有伤害，有刺鼻的刺激性气味。LED-UV灯，365或395nm波长的不行。

  原卡一般是过水光油，硬度高，手感顺滑，还会压光（磨光）磨薄涂层。UV光油硬度差很多，顺滑程度明显不如。

橡胶滚筒瑕疵多一些，较厚，明显感知。

丝印网板缺点是没有喷枪清洗不干净，效果比滚筒好，但是从网板拿下来的时候会有痕迹。

  • 亮色墨水

  爱普生P408和佳能pro-300有亮光色墨水通道，但都只是薄薄的覆一层，不能在哑面纸上实现光泽，但是可以实现卡变得很顺滑。

  • 水性光油

  工业用的水性光油，线棒涂布不能流平有纹路，丝网会黏住，塑封机会留下胶辊上的纹理，还会黏到灰尘。喷漆枪要喷的厚一点，但还是免不了会有麻子，往往喷涂量过大，导致纸张吸水，干燥后纸张会翘曲变形。橡胶滚筒不能均匀涂抹。

铜版纸只有表面的蜡有一定防水性能，纸芯不防水。

  • 油墨光油

  油墨光油，黄黄的，不是很透。橡胶滚筒效果凑合，太厚，干的太慢，要几十个小时，免不了会沾到灰尘。油墨要用天拿水清洗，用水冲没用。油墨本身味道较淡，但是天拿水味道相当大，相当上头。
