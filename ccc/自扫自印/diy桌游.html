# 切卡

三国杀：62x87mm-R3mm
St（万智牌；一般桌游都用这个尺寸）：63.5x88.9mm-R3mm

切纸刀+R3圆角器，成本低，就是费事。

定制重型冲卡机， 淘宝·印后 相册 广告设备专销店 (opens new window)，是我找到报价最低的，200+12邮费，可能还有更便宜的。

# 双面打印对齐

软件用Indesign，排好版，链接图片，让内容缩放适配框架。打印机上限纸器装纸以后推紧。打印测试用尺量偏移再微调，但还是会有一点误差。

A4排St尺寸能塞下9张（3x3），但是我倾向于只排8张（2x4），这样遇到两面打印没对齐的情况，会有一边的出血显得更窄，排8张就是牌的上下的出血对不齐，比左右不对齐感觉更不明显。

# 覆膜

  • 对于diy，我更倾向于牌套或塑封。
  • 冷裱膜是pvc，特别厚，而且很软，手感像套了个手机软壳。
  • 普通bopp预涂膜不防刮，指甲可以划出白痕，有些膜不顺滑。
  • 光膜是镜面反射，会成像，和原卡差异较大。
  • 哑膜是光泽度很低的漫反射，同原卡有差异。
  • 防刮BOPP哑膜是diy效果和成本折中的选择。

我在阿里巴巴天宝印刷买的光泽度30GU，半哑半光，是有一定光泽度的漫反射。观感相对接近原卡一点。

  • 印刷厂覆膜后还会过压纹机，切割后还是很服帖，diy覆膜切割后可能翘边。

  • 印刷厂过压纹机可以做凹凸手感和纹路；diy定制十字布纹预涂膜，只有纹理没有凹凸手感。

  • 塑封7丝留边像纸一样软，8丝的留边会硬一点但还是偏软，10丝比较硬挺。

  • 无论是覆膜还是塑封，都是热熔胶合膜层和打印涂层，开胶撕掉会把打印的图案撕走。

常见能买到的塑封膜通常都是针对三国杀尺寸，【68x95】和【69x95】。

我定制的【72x99】塑封膜在售，既可用于St尺寸，也可用于【无损塑封】，既套牌套塑封，牌套的封口对准塑封膜的封口。卡牌可以无损拆出，但是可能有气泡，拿在手里，或者轻捏卡牌都能明显感知到有空洞感，牌和牌套之间是空的，而牌套已经黏住塑封膜。

# 打印机

  家用打印机都是RGB打印机，驱动接收RGB编码，不需要改成CMYK。由喷墨程序将RGB编码转换为CMYK喷墨量。即使改成CMYK文件也会被转换成RGB编码传输。

  要效果好应进行色彩管理，请参考 (opens new window)。附扫描仪校色打印机分享，一体机可以自己校自己。

喷墨打印机校色后色域容积通常大于CMYK。

  印刷厂如胶印是CMYK四色油墨分别刷一遍，调整CMYK通道可以优化效果，比如把四色黑改成单色黑可以避免重影。

# 堵头

  • 原装颜料就算天天打，气候干燥地区都有概率堵头，太久不打甚至会堵死喷头。
  • 染料就算是兼容墨水都不会堵，断线清洗一次就正常。

# 效果

  • 染料的通透感，亮光感是颜料无法实现的，
  • 染料观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。

    • 缺点一：易褪色，色不稳定，色准和色阶都受到限制，有层次上的损失。

    • 缺点二：染料黑光谱有明显的红色偏移（看起来泛红），既660nm以上波长的可见光反射率上升。

  • 颜料墨水不易褪色，色准高。
    • 如果没有灰色墨水暗部很不理想。
    • 有凹凸感（高光纸表面明显墨水颗粒的痕迹）
    • 输出大面积黑色，由于光线漫射会产生烫金效应，显得不够黑。

# 耐候性

  • 覆膜避光，原厂染料不易感知褪色，兼容染料撑不过一年。

  • 兼容染料包括所谓抗UV墨水，不覆膜，避光3~6个月褪色，见光1个月，阳光直射3天就褪色泛红。

  • 原厂染料，能坚持两三倍时间，但最终一样会褪色。

  • 颜料是整体褪色，显旧，不易感知，而染料黑褪色发红，特别显眼。

# 打印机

打厚纸就一点，后进纸。

爱普生打印机设置，维护-扩展选项，“厚纸与信封纸”，可以尝试勾选。

  • 爱普生微压电，运气好能打10万张以上。
  • 佳能热发泡，打5千张以上都算运气好，墨水稀，用量大，成本高。换颜料墨水很容易堵死喷头。

佳能优点：

  • 打印头容易买还便宜，爱普生打印头坏了，不如买新机。

  • 佳能g580机型1300块就有灰色墨水，爱普生L8168不要原装墨水也要2500块才有灰色墨水。

    • 导入灰色墨水，灰阶好很多。

# 开箱即用：

爱普生L8058，L8168，L805官方驱动带校色文件，原装墨水直接打印不用折腾效果就不错。

爱普生L8168是claria染料墨水，耐候性显著强于普通原厂染料，色域也明显更广。

佳能六色g580/g680电脑驱动不带校色文件，手机app却有。原装墨水可以用红河纸业提供的 (opens new window)ICC，马甲有G650，G680。ICC对应红河纸张可以选Pecos Gloss既高光相纸。

校色请参考 (opens new window)，对原装墨水相对屏幕偏色，也提供了土办法。

极不推荐爱普生四色机的原装墨水

  四色机的原装墨价格不便宜，但是耐候性甚至连L8058的原装墨都不如，而且官方驱动也不提供校色文件。L8168的原装claria墨水贵不了多少，色域，耐候性都要好很多。不如买这个墨水自行校色。

L8168的013墨水是颜料黑，有堵头风险。014照片黑是染料黑。

  如果已经有爱普生四色机原装墨水，可以考虑用红河纸业提供的校色文件。打印机校色文件使用方法。 (opens new window)

  • 红河-ET-2800 (opens new window)，4色染料，海外叫522墨水，可能是004墨水。
  • 红河-502墨水机型 (opens new window)，黑颜料+3色染料，可能是002墨水。
  • 红河ET-8500 (opens new window)，L8168的马甲。

红河打印设置推荐 (opens new window)；选对光面，亚光一般大差不差。

红河的照片纸基本都是RC相纸，只有Pecos River是铸涂相纸，与常见零售铸涂相纸不同，兼容颜料墨水，涂层成分可能不一样。

glossy是高光；satin和luster国内都叫RC绒面，好像satin会叫细绒，luster叫粗绒；在爱普生驱动里都用高质量亚光纸（luster）。粗绒的光泽度会比细绒高一些。

# 兼容墨水：

只要不是原机墨水都要重新校色。请参考 (opens new window)。

A4单面打印，爱普生机型通常共消耗墨水约1~1.5ml左右，佳能热发泡可能会多消耗一些。

如果不打算用原装墨水可以买拆墨空机。

强烈推荐爱普生大幅面的染料墨水，p10080d机型，活的色彩NDD原装染料卖230￥/1L(1袋)。耐候性应该跟L8058的056墨水普通染料差不多，同L8168的claria染料有差距。

我抽墨分装的闲鱼链接 (opens new window)，16.9/50ml，分享有校色文件。详见色彩管理-打印机校色文件分享 (opens new window)。未涵盖机型可免费校色。

兼容墨水强烈推荐墨库·百度西，价格实惠，商用数码墨水中国出货量第一，至少杂质过滤合格。不重视零售，可以向墨库淘宝客服 (opens new window)索要产品型号表。

天威也是大牌，相对比较重视零售，需要小瓶，易加墨瓶可以考虑天威，性价比差些。

  • 墨库染料32￥/1L，校色icc色域容积跟原厂普通染料自行校色的icc差不太多，能达到原厂icc的90%，但耐候性差距非常大，包括所谓抗UV墨水。
  • 墨库颜料43￥/500ml。色准高，不褪色。颜料黑可以考虑博艺，据称解决了烫金效应。
    • 覆膜或塑封，没必要颜料，热熔胶寿命不算理想，说不定还没褪色先开胶了。
    • 校色icc色域容积达到爱普生原装颜料自行校色的75%左右，原厂ICC的65%~70%。
    • 颜料必须常打防堵。气候干燥地区务必审慎。
    • 兼容颜料低温粘度变高，易斜喷。
    • 颜料黑也可以考虑爱普生大幅面墨盒抽墨，黑位黑很多。
  • 爱普生大幅面P5080，P7580，P9580的大容量墨盒抽墨。原厂颜料，1.5~2￥/ml。最新墨水系统改进了树脂，大面积堆黑没有烫金效应。

详细了解请参考【色彩管理-打印机】 (opens new window)

# 纸

  数码墨水和印刷厂油墨成分不一样，存在同色异谱现象，机器看来都是黑，人眼看起来有区别。印刷厂的黑色未必有多黑，但是观感比较正，通透。

  染料墨水打喷墨铜版纸效果很好。但打印机走纸会在高光纸上留下走纸印，棘轮印，哑面纸走纸印不易感知，但是表现力差些。

  300g铜版纸做卡不如原卡硬挺，裸卡重叠在一起摩擦有粘滞感。印刷厂用的纸是蓝芯纸等，更硬挺，侧面是芯的颜色。尽管更硬挺，但没有喷墨涂层，喷墨打印机的表现力严重受限，不适合diy。

  • 水性颜料墨水常用的RC相纸，明显更软，更薄。绒面虽然观感更有质感，但有差异。
  • diy局部镭射闪卡请参考白羽ことり (opens new window)的分享 (opens new window)。其用到的镭射透明不干胶，透明pet等可以打颜料墨水。
  • 淘宝华林兴有150g高光相纸A4幅面卖12块100张，价格实惠，白条卡可以考虑。

  “喷墨铜版纸”inkjet coated paper涂层是蜡没有孔隙给颜料颗粒留存，颜料颗粒是纸表面的一层凸起物，不耐刮擦，但不会洇墨。静置一段时间干燥后是附着的，只是摸起来粗糙。

  印刷厂俗称的铜版纸是印刷涂布纸coated art paper，更硬挺，没有喷墨涂层，喷墨打印机必须用铜版纸颜料墨水（快干颜料墨水）搭配加热装置。快干颜料墨水掺了弱溶剂墨水的溶剂，低温下粘度会变高，打印容易异常。

据墨库介绍，室温30度以上才可能不搭配加热装置。

简单介绍工业用墨水请参考。 (opens new window)

工业墨水：弱溶剂墨水，乳胶墨水，涂料墨水（DTF烫画墨水），UV墨水等，腐蚀打印介质或通过不同原理固化成膜，极易堵头，还具有强腐蚀性，非工业打印头寿命按天计算。

# 测试效果不好的方案

# 切卡

  手动模切机配刀模，挨得太近压力不够会撕扯打印面，而且免不了会有一点位移刮伤打印面，感觉电动模切机也够呛。

  刻字机，要打标记点给机器巡边，A4只能切6张卡，而且机器不能校正角度偏斜，要自己对齐摆放，会有刀口毛刺，而且切割要有垫板保护刀头，垫板有粘性，所以必须要给纸覆膜，或本身就是不干胶有背纸，不然打印面会粘在垫板上。

# 保护

  • 双面RC细绒相纸+水性光油。

  水光油用丝印尖口刮刀在RC相纸的打印面刷一遍，不需要丝印网板，就能实现不错的光泽度和顺滑的手感。裸卡重叠在一起相互摩擦，分离非常顺滑，但是相对原卡明显薄软很多。RC绒面观感更有质感，但与原卡明显不同。

但手动涂布免不了出现瑕疵。

RC相纸表面有一层淋膜，防水，水光油不会马上成膜，而且是孔隙结构，用水光油中的树脂颗粒填充孔隙后，光泽度好，手感顺滑。铜版纸是一层蜡，手动涂布不够均匀，成膜效果不佳。

  • uv光油

  uv打印机打光油，手感非常粗糙，不润。手动涂布光油墨水会有油性物质渗进纸张，不清楚是溶剂还是稀释剂之类的成分。

  UVC光油，丝印尖口刮刀不需要丝网，可以薄薄的挂一层，要用高压汞灯，对皮肤，人眼都有伤害，有刺鼻的刺激性气味。LED-UV灯，365或395nm波长的不行。

  原卡一般是过水光油，硬度高，手感顺滑，还会压光（磨光）磨薄涂层。UV光油硬度差很多，顺滑程度明显不如。

橡胶滚筒瑕疵多一些，较厚，明显感知。

丝印网板缺点是没有喷枪清洗不干净，效果比滚筒好，但是从网板拿下来的时候会有痕迹。

  • 亮色墨水

  爱普生P408和佳能pro-300有亮光色墨水通道，但都只是薄薄的覆一层，改善颜料墨水的光泽度，缓解烫金效应，没有光油的手感，光泽度相对光油的贼光也要差很多。亮光色墨水在铜版纸上耐刮擦，不清楚这两款机型用水性颜料墨水搭配亮光色墨水能不能固色。

  • 水性光油

  工业用的水性光油，线棒涂布不能流平有纹路，丝网会黏住，塑封机会留下胶辊上的纹理，还会黏到灰尘。喷漆枪要喷的厚一点，但还是免不了会有麻子，往往喷涂量过大，导致纸张吸水，干燥后纸张会翘曲变形。橡胶滚筒不能均匀涂抹。

铜版纸只有表面的蜡有一定防水性能，纸芯不防水。

  • 油墨光油

  油墨光油，黄黄的，不是很透。橡胶滚筒效果凑合，太厚，干的太慢，要几十个小时，免不了会沾到灰尘。油墨要用天拿水清洗，用水冲没用。油墨本身味道较淡，但是天拿水味道相当大，相当上头。

皮肤沾到可以用风油精，含甘油的护手霜等擦一遍再打肥皂清洗。
