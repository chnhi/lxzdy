# 切卡

三国杀：62x87mm-R3mm
万智牌：63x88mm-R3mm（2.48x3.46寸）
标准扑克尺寸： 63.5x88.9mm，（2.5寸x3.5寸）
有一些网站标注万智是63.5x88.9mm，但据老外分享，DIY做63x88mm正好。

  • 切纸刀+省力型倒角钳，成本低，就是费事。

  • 定制重型冲卡机， 淘宝·印后 相册 广告设备专销店 (opens new window)，是我找到报价最低的，200+12邮费，可能还有更便宜的。

重型冲卡机，出血留大了目视对齐容易歪斜，想要避免歪斜，目视对齐的一面出血要小，靠白边的大小来判断是否歪斜，然后平移消弭白边。缺点是要注意力集中，其实也是比较费事的。

  • 定制电动名片切卡机+手动圆角机相对省力一点，只是机器成本要900+200。

小型机器只能切不覆膜的300g纸。能切的切卡机要5000+，不一定能定制，另一种能切还可以调切割大小的报价1万。

在实践后发现，这种方案最好搭配印刷彩印。

手动圆角机的话R3圆角太小，稍微有点误差会很明显。主要表现就是倒角不对称。

其次，只能切不覆膜的纸张，那么只能颜料墨水，染料墨水裸奔耐候性太差。搭配颜料墨水可以不覆膜的纸张表现力会差一些。

  • 印刷厂用模切机

核心原理就是刀模，然后机器施加压力，有直上直下的，有滚筒施加压力的。进纸有人工对准的，有人工上料后自动对准的，有自动上料的。

搜索关键词：扑克自动冲卡机，早教卡自动冲卡机。

扑克牌印刷厂看视频有些先切成出血大小，然后叠一起用同一个模具模切，不知道是不是为了一致性。

搜索关键词：商标液压模切机

# 印刷厂彩印

数码印刷机一般对起订量要求很低，不过要注意很多是激光彩印，效果不是最理想，输出一致性很差，碳粉还可能掉。要HP indigo，电子油墨，会有类似于胶印的印刷网纹，官方一般应该是胶印。

同城或淘宝找HP indigo机器的商家，有些商家可能可以上光油或覆膜。上光油一般是更顺滑，不上光油一般就像标准版2022，2023那样，不顺滑，但是油墨还是会反光有光感。一般300g铜版纸A4双面彩印5毛~1块，覆哑膜加5毛~1块，过油一般会贵一点。找不到就找招indigo操作员的公司。

参考前文自行切卡，既印刷又切卡的往往收费不菲，但是如果只印刷的话，一般便宜很多。

# 覆膜·塑封

  • 冷裱膜是pvc，特别厚，而且很软，手感像套了个手机软壳。
  • 覆膜（热裱膜）普通bopp预涂膜不防刮，指甲可以划出白痕，有些膜不顺滑。
    • 光膜是镜面反射，会成像，和原卡差异较大。
    • 哑膜几乎没有反光，同原卡有差异。
    • 防刮BOPP哑膜手感会有一点粘腻，会有一点新的银行卡上面贴的保护膜那种感觉，很没有质感。
    • 阿里巴巴天宝印刷，一款21柔光膜，一款微米30哑膜。光泽度介于哑膜亮膜之间，柔光膜会成像偏漫反射，观感和过光油原卡相对接近些。哑膜则是有一定光泽度的漫反射，相对比较接近覆膜原卡。
    • **DIY建议PET哑膜，可以改善走纸印和棘轮印。**PET不是很怕刮，防刮BOPP只是刮了不会破，还是会有痕迹。
  • 印刷厂覆膜后还会过压纹机，切割后还是很服帖，diy覆膜切割后可能翘边。
  • 印刷厂过压纹机可以做凹凸手感和纹路；diy定制十字布纹预涂膜，只有纹理没有凹凸手感。

# 塑封膜

  • 无论是覆膜还是塑封，都是热熔胶合膜层和打印面，开胶撕掉会把打印的图案撕走。

    我定制了【72x99】10丝在售，用于【无损塑封】，既套牌套塑封，牌套的封口对准塑封膜的封口。卡牌可以无损拆出，但是可能有气泡，拿在手里，或轻捏卡牌都能明显感知到有空洞感，牌和牌套之间是空的，而牌套已经黏住塑封膜。

  • 塑封膜不足8丝留边像照片纸的硬度，10丝更硬一点，8丝折中。

    • 商家塑封套餐一般用6~8丝。

  • 推荐大厂艺都千帆定制（淘宝或阿里巴巴），10包起订，算上邮费，10丝7块多一包，8丝5块多一包。

大厂倒角圆润，热熔胶涂布均匀，收缩率低，透明度高。

  1. 把卡牌推到封边底，塑封后约4mm留边。
  2. 封短边则上下留边居中比较稳定，左右目视对齐，无法避免偏差。封长边则左右留边比较稳定，我更倾向于封长边。
  3. 塑封膜外径主要影响留边边距，我定制一般四边都留边4mm。
  4. 三国杀官正出血四边都是2~3mm，所以我倾向于留边四边一致。
  5. 通常留边至少3mm，但有一边是目视对齐，稳妥起见预留7mm，避免误差，留8mm。

三国杀卡牌【62x87】，St尺寸（万智）【63x88】。我会定制三国杀【70x95】，St【71x96】。

# 塑封机

几十块塑料壳的塑封机只有两个辊不如铁壳的4辊平整。

加热方式分为石英管和云母片，石英管是红外灯管。石英管分为内加热和外加热，云母片暂时只有外加热，内加热是把加热管装在胶辊里。外加热是加热管在胶辊旁边加热胶辊。

  • 内加热更均匀，但是加热速度慢，胶辊贵。

  • 云母片加热速度快，预热时间短，传热效率高，连续作业效果好，但是成本高。

    • 200￥雷盛LM320-S(T)内加热，塑料齿轮，19mm胶辊。

    • 350￥平达330C，云母片，金属齿轮（跟客服要云母片的）。

    • 500￥平达330CN是内加热，金属齿轮，25mm胶辊。

    • 720￥平达330TD，云母片+可调高速，适合量很大。

量小不在乎速度慢尽量买内加热，量大尽量买云母片+可调高速，石英管外加热温度不均匀，容易出瑕疵。

塑封机并不是绿灯亮了就可以塑封了，绿灯亮只代表温度传感器检测的点达到设定温度，并不代表整根胶辊的温度都够了。所以使用中可以考虑先调到80~90度，绿灯亮起后预热5分钟左右，再调到需要的温度，再次亮起再酌情开始塑封。温度尽量不要猛猛调高。

# 常见塑封异常

雾状白点，不粘，温度不够（设定过低或胶辊温度不均匀或绿灯没亮）。

泡点（一点一点，点之间有一定间距）温度过高或胶辊需要清洁。

起皱不平整，温度过高或放置不平整。

留边不粘或起泡（类似于手机贴膜有气泡），温度不够或胶辊压力不够，有两个有弹簧的螺丝拧一下。

# 打印机

  家用打印机都是RGB打印机，驱动接收RGB编码，不需要改成CMYK。由喷墨程序将RGB编码转换为CMYK喷墨量。要效果好应进行色彩管理，请参考 (opens new window)。附扫描仪校色打印机分享。

喷墨打印机校色后色域容积通常大于CMYK。

  印刷厂如胶印是CMYK四色油墨分别刷一遍，调整CMYK通道可以优化效果，比如把四色黑改成单色黑可以避免重影。

# 堵头

  • 原装颜料太久不打都可能堵死喷头，至少每周打一次可以是喷头检查图。
  • 染料就算是兼容墨水都不会堵，断线基本清洗就正常。

# 效果

  • 染料观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。
  • 具有通透感，亮光感是颜料无法实现的。

    • 缺点一：易褪色，色不稳定，色准和色阶都受到限制，有层次上的损失。

    • 缺点二：染料黑光谱有明显的红色偏移（看起来泛红），既660nm以上波长的可见光反射率上升。

  • 颜料墨水不易褪色，色准高。
    • 有凹凸感（高光纸表面明显墨水颗粒的痕迹）
    • 输出大面积黑色，由于光线漫射反光会产生烫金效应，看着泛白。

# 耐候性

  • 覆膜避光，原厂染料不易感知褪色，兼容染料撑不过一年。

  • 兼容染料包括所谓抗UV墨水，不覆膜，避光3~6个月褪色，见光1个月，阳光直射3天就褪色泛红。

  • 原厂染料，能坚持两三倍时间，但最终一样会褪色。

  • 颜料是整体褪色，显旧，不易感知，而染料黑褪色发红，特别显眼。

  • L8168的原装014墨水是干扩机同类染料，覆膜避光耐候性堪比颜料，就是成本高。

# 打印机

  **打厚纸必须后进纸。**爱普生L8058机型可以打300g纸。

如果打印头蹭纸，可以尝试在设置中勾选“厚纸与信封纸”。

  • 爱普生微压电，兼容墨水成熟，价廉物美，可以换兼容颜料墨水。

一般4万张左右打印头状态不佳，4万张之后基本要用最高质量加大喷墨量掩盖瑕疵。

  • 佳能热发泡，墨水稀，用量大，成本高。换颜料墨水很容易堵死喷头。

但是佳能g580的原装墨水是chromalife100染料墨水，耐候性比爱普生普通原厂染料好蛮多，不如爱普生L8168的干式彩扩机墨水。打算用原装墨水可以考虑佳能机型。

爱普生一般4万张左右打印头状态不佳，4万张之后基本要用最高质量加大喷墨量掩盖瑕疵。

惠普对热发泡商用大幅面的建议是每色1.5L墨水换打印头，既4500~6000张左右A4幅面照片更换打印头。一般diy很难用掉那么多墨水。而且佳能打印头容易买还便宜，爱普生打印头坏了，不如买新机。

# 开箱即用：

爱普生L8058，L8168，L805官方驱动带校色文件，原装墨水直接打印不用折腾效果就不错。

  • L8058，L805建议参考后文换墨水并校色。
  • L8168只建议用原装墨水，耐候性显著强于普通原厂染料，色域也明显更广。
  • 佳能六色g580/g680电脑驱动不带校色文件，手机app却有。原装墨水可以用红河纸业提供的 (opens new window)ICC，马甲有G650，G680。ICC对应红河纸张可以选Pecos Gloss既高光相纸。

校色请参考 (opens new window)，对原装墨水相对屏幕偏色，也提供了土办法。

极不推荐爱普生四色机的原装墨水

  四色机的原装墨价格不便宜，但是耐候性甚至连L8058的原装墨都不如，而且官方驱动也不提供校色文件。

  如果已经有爱普生四色机原装墨水，可以考虑用红河纸业提供的校色文件。打印机校色文件使用方法。 (opens new window)

  • 红河-ET-2800 (opens new window)，4色染料，海外叫522墨水，可能是004墨水。
  • 红河-502墨水机型 (opens new window)，黑颜料+3色染料，可能是002墨水。
  • 红河ET-8500 (opens new window)，L8168的马甲。

红河打印设置推荐 (opens new window)；选对光面，亚光一般大差不差。

红河的照片纸基本都是RC相纸，只有Pecos River是铸涂相纸。

glossy是高光；satin和luster国内都叫RC绒面，好像satin会叫细绒，luster叫粗绒；在爱普生驱动里都用高质量亚光纸（luster）。粗绒的光泽度会比细绒高一些。

# 兼容墨水：

只要不是原机墨水都要校色。请参考 (opens new window)。

  • A4单面打印，爱普生机型通常共消耗墨水约1ml左右，佳能热发泡可能会多消耗一些。

  • 不打算用原装墨水可以买拆墨空机。

爱普生大幅面染料墨水，p10080d染料卖230￥/1L(1袋)，p11080d卖320￥/1.6L。耐候性是爱普生普通染料水平，同干扩机同类染料有差距，但是成本低。

干扩机同类染料：爱普生L8168的原装014墨水，L8058可以买干扩机墨水，如爱普生D580墨瓶。D880，D1080等墨盒抽墨，另有富士DX100,DE100等。覆膜避光耐候性堪比颜料，就是成本高。

单单黑色墨水用干扩机同类染料，彩色墨水用大幅面染料，可以显著改善耐候性，因为普通染料黑色褪色会迅速发红，而干扩机染料黑在褪色初中期不会发红。

兼容墨水强烈推荐墨库·百度西，价格实惠，商用数码墨水中国出货量第一，至少杂质过滤合格。不重视零售，可以向墨库淘宝客服 (opens new window)索要产品型号表。

天威也是大牌，相对比较重视零售，需要小瓶，易加墨瓶可以考虑天威，性价比差些。

  • 兼容染料色域不差，但是耐候性太差。
  • 兼容墨水只推荐颜料，校色icc色域容积达到爱普生原装颜料自行校色的75%左右，原厂ICC的65%~70%。
  • 但是自印卡用颜料墨水，没有特别合适的纸，具体参考后文，各有各的短板。
    • 覆膜或塑封，注意热熔胶寿命不算理想，说不定还没褪色先开胶了。
    • 颜料至少每周打1次可以是打印头检查图，太久不打可能堵死。
    • 兼容颜料低温粘度变高，易斜喷。
  • 颜料墨水有烫金效应（深色墨水反光会泛白泛青铜色），观感不如染料自然。黑墨酌情用大幅面颜料墨盒抽墨。P6080的HD系列墨水，比K3黑，K3如果便宜也可以考虑，大面积堆黑反光会泛白。P7580的Pro10，12系列黑墨反光不泛白，但是反光可能呈现出青铜色，泛白可能还自然一点。
  • 爱普生大幅面P5080，P7580，P9580的大容量墨盒抽墨。原厂颜料，1.5~2￥/ml。
  • 原装粗面黑在哑光纸，粗面纸上黑得多，但是不耐刮擦，如果不覆膜不建议用原装粗面黑墨水。

详细了解请参考【色彩管理-打印机】 (opens new window)

# 纸

  • 印刷厂俗称铜版纸的纸张，准确来说是 印刷涂布纸coated art paper。

没有喷墨涂层，喷墨打印机必须用铜版纸颜料墨水（快干颜料墨水），最好改装加热装置。墨水掺了溶剂有腐蚀性，不仅影响打印头寿命，而且低温下粘度会变高，打印容易斜喷或断线。

  • 喷墨用的” 高光铜版纸 “，准确来说是“喷墨铜版纸”inkjet coated paper。

有喷墨涂层，可以吸收更多墨水增加色密度，色彩表现力相当不错。但是：

  1. 涂层也计入克重，相同克重下更薄更软。
  2. 反光能感知走纸印和棘轮印，两张卡摩擦有白痕，必须要保护。
  3. 覆PET哑膜可以改善走纸印和棘轮印。
  4. 只建议用染料墨水，颜料墨水的烫金效应会导致黑色反光光感与其他颜色不同，显得怪异。
  • ”白卡纸“。有350g，400g。

白卡纸克重就是纯纸，最硬挺。但是靠纸基吸墨，色密度严重受限，影响表现力，由于墨水被纸张吸收，会泛纸张本色，显得色密度不够。颜料要原装粗面黑才比较黑，但是不耐刮擦，手一摸就变黑。

  • 哑光喷墨铜版纸，matte inkjet coated paper。

所谓“哑光铜版纸”有两类，一类实质就是白卡纸。另一类有涂层。有涂层的我买到过两种：

1、骏马纸业代工，很多小牌子的所谓哑光铜版纸就是贴牌。这个纸我认为可以称为“丝光彩喷纸”。说是哑光，实际上还是有轻微的反光。相对高光铜版纸，走纸印和棘轮印有很大的改善，基本要反光细品才能感知到有走纸印。但是表现力相对高光铜版纸要稍差些。

这个纸经常本身涂层上就有斑驳的痕迹，覆膜后会很明显。

必须覆膜，不覆膜切割后，切口处涂层会有皲裂。覆膜后染料颜料都可以用。

2、我称为”哑光彩喷纸“，有涂层但是不会反光，表现力相对“丝光彩喷纸”要差。我买到过力武的“哑光铜版纸”是这类。

走纸印和棘轮印相对“丝光彩喷纸”改善一点，基本要浅色部分反光细品才能感知。搭配颜料墨水，黑色墨水堆积的地方会有反光，其他地方又没有反光，反光就会有一点斑驳。这款纸如果天气潮湿，颜料墨水干的慢了，会有明显走纸印。

  • 增强粗面艺术纸

走纸印和棘轮印相对哑光铜版纸还要进一步改善，不怎么能感知到，但是由于是涂层和纸基一起吸墨，所以精度锐度相对会差一些，成本还高。

搭配颜料墨水没有烫金效应，观感比较“自然”温暖，但是要写实的作品才能体现，自印卡基本没有。

  • RC双面细绒相纸搭配颜料墨水有非常强的表现力，但又软又薄，只适合塑封。

RC相纸色域最广，完全是涂层吸墨，纸张和涂层之间有PE膜隔绝，二氧化硅或氧化铝涂层，不易感知走纸印，细品可能感知棘轮印。

  • 莱妮纹 珠光纸，布纹效果。
  • diy局部镭射闪卡请参考白羽ことり (opens new window)的分享 (opens new window)。其用到的镭射透明不干胶，透明pet等可以打颜料墨水。

工业墨水：弱溶剂墨水，乳胶墨水，涂料墨水（DTF烫画墨水），UV墨水等，腐蚀打印介质或通过不同原理固化成膜，极易堵头，还具有强腐蚀性，非工业打印头寿命按天计算。

# DIY-PVC卡

DIY可行的PVC卡有两种，一种是PVC免层压，另一种是喷墨PVC白卡。

PVC免层压是PVC卡上涂有热熔胶，打印在PET透明膜上（有喷墨涂层）然后过塑封机，把PET透明膜和PVC卡胶合，然后用重型冲卡器冲卡。热熔胶寿命不长会脱胶，甚至刚做完就起泡。

L8058机型有PVC证卡托架，可以打印在PVC白卡上，必须是喷墨PVC白卡，涂布了喷墨涂层。缺点是只有86x54尺寸。最好用颜料墨水。

88x63尺寸我估计应该是可以打的，只不过喷墨白卡和托架都要定制，不知道在哪定制托架。

L8168机型有一个后部进纸槽，把屁股掀开可以平行进纸。网上搜 L8168 打印证卡看看视频，自己3d打印一个类似的托盘，自己在InDesign软件微调出适配的模板文件。

# 测试不适合diy的

# 切卡

  手动模切机配刀模，挨得太近压力不够会撕扯打印面，而且免不了会有一点位移刮伤打印面，感觉电动模切机也够呛。

  刻字机，要打标记点给机器巡边，A4只能切6张卡，而且机器不能校正角度偏斜，要自己对齐摆放，会有刀口毛刺，而且切割要有垫板保护刀头，垫板有粘性，所以必须要给纸覆膜，或本身就是不干胶有背纸，不然打印面会粘在垫板上。

# 保护

  • 双面RC细绒相纸+水性光油。

  水光油用丝印尖口刮刀在RC相纸的打印面刷一遍，不需要丝印网板，就能实现不错的光泽度和顺滑的手感。裸卡重叠在一起相互摩擦，分离非常顺滑，但是相对原卡明显薄软很多。RC绒面观感更有质感，但与原卡明显不同。

但手动涂布免不了出现瑕疵。

RC相纸表面有一层淋膜，防水，水光油不会马上成膜，而且是孔隙结构，用水光油中的树脂颗粒填充孔隙后，光泽度好，手感顺滑。铜版纸是一层蜡，手动涂布不够均匀，成膜效果不佳。

  • uv光油

  uv打印机打光油，手感不润。手动涂布光油墨水会有油性物质渗进纸张，不清楚是溶剂还是稀释剂之类的成分。

  UVC光油，丝印尖口刮刀不需要丝网，可以薄薄的挂一层，要用高压汞灯，对皮肤，人眼都有伤害，有刺鼻的刺激性气味。LED-UV灯，365或395nm波长的不行。

  原卡一般是过水光油，硬度高，手感顺滑，还会压光（磨光）磨薄涂层。UV光油硬度差很多，顺滑程度明显不如。

橡胶滚筒瑕疵多一些，较厚，明显感知。

丝印网板缺点是没有喷枪清洗不干净，效果比滚筒好，但是从网板拿下来的时候会有痕迹。

  • 亮色墨水

  爱普生P408和佳能pro-300有亮光色墨水通道，但都只是薄薄的覆一层，不能在哑面纸上实现光泽，但是可以实现卡变得很顺滑。

  • 水性光油

  工业用的水性光油，线棒涂布不能流平有纹路，丝网会黏住，塑封机会留下胶辊上的纹理，还会黏到灰尘。喷漆枪要喷的厚一点，但还是免不了会有麻子，往往喷涂量过大，导致纸张吸水，干燥后纸张会翘曲变形。橡胶滚筒不能均匀涂抹。

铜版纸只有表面的蜡有一定防水性能，纸芯不防水。

  • 油墨光油

  油墨光油，黄黄的，不是很透。橡胶滚筒效果凑合，太厚，干的太慢，要几十个小时，免不了会沾到灰尘。油墨要用天拿水清洗，用水冲没用。油墨本身味道较淡，但是天拿水味道相当大，相当上头。

皮肤沾到可以用风油精，含甘油的护手霜等擦一遍再打肥皂清洗。
