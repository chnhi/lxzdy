# 自印卡

显示器是自发光，而印刷物是反射光，颜色模型是对物理世界的数学模拟，无论是CCD，CMOS还是显示系统，墨水系统，对光线的响应与人眼不同，不可避免存在偏差。

需要蓝芯纸（更硬挺），游戏王尺寸（59X68），镭射覆膜，淘宝·游家桌游吧 (opens new window)。仅供参考，无利害关系。

# 简述

满29包邮并享浮动折扣，消费越多折扣越大。

全流程色彩管理，有黑白校色可选。

  • 柯达300g高光铜版纸+爱普生原厂染料+切卡，0.6￥/卡。

    • 塑封或覆膜后避光褪色不易感知。
    • 300g铜版纸比原卡软，裸卡重叠摩擦有明显粘滞感。

  • 覆膜+0.1￥/卡。支持牌套，塑封。

    • 覆膜或塑封都是热熔胶粘合打印面和膜层，开胶撕掉会破坏打印面。
    • 光膜镜面反射，哑膜漫反射，都和原卡不符。定制光泽度30GU，半哑半光，既有漫反射又有一点镜面，极富质感，防刮耐磨。

  • 支持切换兼容颜料黑避免染料黑的红色偏移，增强耐候性。

  • 牌背只能保证没有露白，不能保证绝对居中。

  • 高光纸覆膜后有明显走纸印，可换单光单哑纸。

    • 没有走纸印，棘轮印。
    • 纸是米黄色本色，没有漂白。
    • 哑面表现力弱些，反光影响更大。
    • 光面牌背效果与高光铜版纸相似。

# 爱普生原厂墨水，比兼容墨贵8倍

  • 染料效果通透鲜艳，裸奔会褪色。

    • 爱普生原厂墨水，覆膜避光褪色不易感知。兼容染料覆膜避光1年内明显褪色。

  • 颜料不褪色，色准高，但观感黯淡，暗部反光会衍射泛白（烫金效应）。

    相同墨水通道数下，颜料的色阶过渡，暗部细节与染料存在差距。

  • 导入灰色墨水，灰阶好很多。

    相对6色，少浅洋红和浅青，主要短板在肤色过渡和浅绿色阶，但是浅绿在自然中非常罕见。

  • 可以切换颜料黑避免染料黑的红色偏移。

    染料黑在光谱有红色偏移，660nm以上波长的可见光反射率不为0。

    • 单单使用颜料黑墨水，可以延后褪色能感知的时间。

      颜料墨水曝晒也会褪色，但整体褪色程度相近，而人眼对颜色间关系更敏感，所以只会显旧。染料褪色则黑色先泛红，观感非常明显。

  • 5色染料：爱普生大幅面P10080D原装染料。

# 塑封，牌套

铜版纸裸卡重叠在一起相互摩擦有明显粘滞感。

  • 双面覆膜，0.8￥/A4。

    • 定制 仿原卡观感 质感膜，防刮耐磨，手感顺滑。
    • 覆膜或塑封都是热熔胶粘合打印面和膜层，开胶撕掉会破坏打印面。
    • 印刷厂会过压纹机，压出凹凸手感，胶合更紧密，几乎不开胶。

    普通光膜镜面反射，哑膜漫反射，都和原卡不符。定制光泽度30GU，半哑半光，既有漫反射又有一点镜面，极富质感。

  • 【塑封】有留边，硬挺。手工费1毛/张，材料费另算。

    • 7丝留边像纸一样软，8丝的留边会硬一点但还是偏软，10丝比较硬挺。
    • 自扫没有收录的桌游，扫描可以免覆膜/牌套的手工费。

    塑封膜有±1mm的误差。

    68x95-7丝，3分/张；10丝，5分/张。

    69x95-8丝，3分/张。

    71x96-10丝，7分/张。更适合63x88。

    72x99-10丝，5分/张。

  • 【牌套】可代套，手工费1毛/张。

【无损塑封】手工费2毛/张。定制10丝塑封膜【72x99】，套牌套塑封，开胶不伤牌，拆出无损，但是可能有气泡，拿在手里，或轻捏卡牌都能明显感知到有空洞感，牌和牌套之间是空的，而牌套已经黏住塑封膜。
