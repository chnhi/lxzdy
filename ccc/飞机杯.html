# 飞机杯

2022.11.14

网上评测牛逼吹的震天响，刺激与否很大程度上就是看阴茎受到的压力，用丝袜包住或者给炮架垫毛巾，给飞机杯施加很大的压力，甚至能实现秒射。

飞机杯之家 (opens new window)有大量评测，但是毕竟是商家开的，免不了雕琢粉饰，诸如出油，是否翻洗等还是可以参考。

# 具体推荐

  1. 女优代言款性价比很低。
  2. 日本品牌300克以上重量产品体验非常完备，更贵的提升有限，性价比不高。
  3. 200克换成350克以上产品体验有明显提升，其他的升级感知不明显。
  4. 炮架体验提升很大，还可以垫毛巾增大压力。
  • Maccos 糖果萌妹 cockfit蓝盒 雪板娘 阿里巴巴￥99 飞机杯之家￥119

maccos主打性价比高，99块500克。

糖果萌妹我推测耐用度相对较高，有肉粒有宫口，较为平衡中庸。

红绳姐姐2代主打多重突破，飞机杯才有的特色玩法。

雪板娘主打类似真实性行为。

这几款产品翻洗都要嗯翻，从底部使劲往里塞，入口会小裂，对红绳姐姐2这种双层素材的寿命影响很大，内层可能脱落。

  • Ride Japan 三位一体 阿里巴巴￥100出头

  Ride Japan的素材耐用度高，可翻洗。

  • tomax Succubus 2.0（魅魔）Bumpy Road regular 日常￥269 海淘￥220 双十一￥200

tomax通贩大魔王很可惜只能买到零售价，性价比一般，单论玩到发射没什么特别，但还是有特色体验，详见后文-tomax通贩大魔王。

Succubus 2.0（魅魔）Bumpy Road soft，从他的设计来看，我推测比较耐用，tomax特色，温柔的刺激，但是温柔差点味道，不如Lilith Spiral-dots soft把温柔的刺激做的更平衡，但是耐用度相对差一些。

# 购买方式推荐

  电商平台有价格保护，给零售商留的利润率很高，淘宝闲鱼pdd有很多低价乱市。也可以自己上阿里巴巴买一件代发，注意要自己咨询客服。

以前大代理都在阿里巴巴开店，不知道是不是因为知道的人多了，现在大都关掉了，现在阿里巴巴很多都是二道贩子，价格没以前实惠。

  飞机杯通常营销支出较大，靠软文绘声绘色的描写让你感觉要起飞，最早是营销高利润率的大牌子，如npg的003。但由于有低价乱市的现象，很容易为别人做嫁衣。零售商就开始各自挖掘没名气的小牌子，拿下独占再营销，维持高利润率。

  tomax通贩大魔王也是这个路子，不分销，维持高利润率。

# tomax通贩大魔王

  最大的特点是，弹性非常足，又较为狭窄，依靠材料被撑开释加压力，慢慢剐蹭能感知到纹理的轻拢慢捻抹复挑；又比如宫口设计，友商是咯噔一下，tomax是啵的一下。快速运动相对友商则没有明显区别。

  推荐**tomax日本官网 (opens new window)** 看看，不瞎吹，细节图全，还有类似内窥镜的视频。

  只可惜代理加价后，此消彼长显得性价比不理想，比如女妖Lilith，日本卖150，转运费100出头，女神日本卖250，转运费120左右。大促期间天猫熠享叠加消费券和跨店满减，跟海淘价格差不多。

  tomax官网列出的的中国代理：淘宝·七色谷和11区玩具屋 (opens new window)，天猫熠享保健品专营店 (opens new window) 京东允淼成人用品专营店 (opens new window)。除了11区玩具屋，另3家其实是同一家。

# 配件

  • 零售商会送一堆劣质配件，建议联系客服都不要，折现。

  • 在翻洗的情况下，有毛巾用来擦干就够了。

  • 不翻洗，需要一个细长的棒子可以用毛巾裹住伸进去擦干，或者用卫生棉条吸水。

  • 长期闲置最好在杯身上抹爽身粉，不然会发粘。超市里几块一盒的玉米爽身粉就可以。

# 润滑油

  我自己只用玻尿酸类，易清理。

  主要有两类，一类是玻尿酸混甘油，另一类是甘油为主要成分。玻尿酸类体感上更像水（H2O）。鸡儿对纹路的感知能力甚至比粘稠度是稀的甘油类更强，也更容易清理。缺点是容易干，如果比较持久还要再加。翻洗的时候更不易翻开，售价高。

  甘油类稀的容易感知纹理，稠的不容易感知纹理，随着内酚酞的分泌，就算是稀的也渐渐感知不到纹理，稠的能明显感觉到油层更厚。

  我用过最拉跨的是银离子抗菌和女优冠名淫臭液，睡一觉起来特别臭。

# 炮架

炮架对于体验的提升较大，而且可以垫毛巾增大压力。飞机杯套丝袜，或者带洞抱枕，下面垫个枕头什么的，实惠，体验也不错，就是不仿真而已。

海绵炮架有气味，手感也一般，但是造型相对仿真。充气炮架，摸一摸气球，你就知道有多假了。我觉得都不如带洞抱枕。

这几种自重都很轻，要按住或者用别的东西架住才行。

萌涩幻想之类的抱枕，还能加义乳，也凑合。

贵价一点的，搜·名器伴侣，有一款由密炮架三代，就是有点贵，将近400块，相当于实体娃娃的屁股部分。

# 阴茎没那么灵敏，感知不大

  • 评测飞机杯的以卖飞机杯的居多，形容的绘声绘色。但阴茎的知觉没有手指触觉那么灵敏，很多设计用手指能感觉，但阴茎无法感知。

  • 随着摩擦，大脑会分泌内酚酞，导致感知进一步下降，一开始能感知的一些设计，渐渐会无法感知。同时主要感知能力在龟头，阴茎其他部分几乎无感。

  1. 纹路设计一开始能感知到区别，但渐渐无感，快速运动完全无感。
  2. 大肉粒能感知，也是很多产品的主要刺激点。
  3. 榨汁机一般材料偏硬，内部镂空偏小，营造更大压力。慢玩一般是材料相对更软，用料也多。
  4. 假舌头，假牙齿能明显感知，但是快感一般。
  5. 宫口设计突出一大圈材料，能明显感知，所谓突破感就是龟头能感觉到咯噔一下，宫口设计可以创造一个变化。临门一脚，直来直去的可能脚拇指扣紧还要折腾一两分钟，但有变化会加速。
  6. 无论是套丝袜还是炮架垫毛巾，施加压力后，感知会变得明显一些。

# 包茎，敏感度和快玩慢玩

  生理性包茎可以用飞机杯辅助翻出龟头。龟头一直没有外露，硬翻会有点疼，但用飞机杯辅助会分泌内酚酞，当时就没那么疼。事后的酸爽是因为里面的粘膜长期被包住，没有经历过内裤的摩擦，适应几天就好了。

  快玩慢玩产品的设计不同，往往只在一开始能感知到差异，随着时间的推移，感知趋同。我认为影响快慢的主要还是压力大小，如果觉得慢玩产品的压力太小，套丝袜或者炮架垫毛巾，增大压力。除非长期包茎，否则所谓的敏感度完全是伪命题，

  • 内酚酞的分泌，感知会越来越弱。
  • 射精阈值与兴奋程度负相关，越兴奋越容易射。

# 不同飞机杯明显差异点

# 重量

  重量很容易感知，但是对快感影响不大。通常相近的材料越重越难撑开材料，有更大的压力。

  • 200g上下太薄，手能感知到阴茎，阴茎也能感知到手，太拉跨。

  • 350g上下不会太薄，而且翻洗方便。

  • 500g以上基本和翻洗无缘。重量继续增加已经几乎感知不到区别，反而手很酸，整个顶起来感觉太重了。不过有特别的玩法，飞机杯可以用自重吃掉阴茎。

  • 一旦重量增加到1kg上下，最好放在桌子上或者有个炮架，太重了。这种有一点撞击的感觉，但阴茎的感知和快感，我认为没提升。

  翻洗清洗起来方便太多，不能翻洗的话要冲洗十几分钟，还要伸进去擦干，往往清洗的不彻底。用安全套的话，体验差很多。似乎因为材料的摩擦力还是比较大，很容易变成阴茎摩擦套套，甚至捅破套套。   不太好翻洗的，在玩具底部使劲往里塞能嗯翻。

# 我用过的产品自我感觉

  同一品牌不同产品材料可能不同，但大部分厂家的产品我只体验过一两款，以偏概全仅供参考。材料越软，纹理设计越难感知。

  • ride Japan材料耐用，性价比高，中等偏软。
  • MACCOS 性价比拉满，100块就有500克的玩，柔软度中等偏硬，有一定弹性，不是过硬过窄生造的压力，是弹弹的压迫感。
  • tomax，性价比较低，但是不可否认软弹的材料配上通道设计，确实提供了与友商不一样的体验。
  • 对子哈特R20，性价比低，但也是很有特色的产品，主打榨汁机人设，纹路设计大开大合，突出一大圈材料，能明显感知。但我不觉得对于快感有明显提升，靠过硬过窄生造压力，我不喜欢。
  • Magic Eyes 性价比较低，不如买maccos。
  • npg（水菜丽，ys003）性价比特低，特别软，不耐用，会掉渣，用了一段时间居然发霉了，不排除是个例。
  • tamatoys （上原亚衣）会掉渣。中规中矩。
  • exe（某女优代言款）材料偏硬，压力偏大但却感觉空洞，不得劲。
  • 国产买过一款19块标称重量400克，压力小，空洞，不得劲。更高价位没试过。
