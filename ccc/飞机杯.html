# 飞机杯

更新于2021.2.8

  我所说飞机杯是纯硅胶的产品，不包括tenga有外壳的一次性飞机杯，入口带有海绵无法重复利用，胶体体积也比较小，性价比不高。

  可以跳过经验分享，直达文末具体推荐。

# 阿里巴巴最便宜

  淘宝京东有价格保护，上阿里巴巴找代理买便宜，闲鱼，拼多多，微商大多是阿里巴巴代销。

# 阴茎没那么灵敏，感知不大

  评测飞机杯的以卖飞机杯的居多，形容的绘声绘色，评测比实际用爽多了。阴茎的知觉没有手指触觉那么灵敏，很多设计用手指能感觉，但阴茎无法感知。随着摩擦，大脑会分泌内酚酞，感知进一步下降，导致一开始能感知的一些设计，渐渐无法感知。同时主要感知能力在龟头，阴茎其他部分几乎无感。

  1. 纹路设计一开始能感知到些微区别，渐渐无感。
  2. 大肉粒如果做的比较大，材料又偏硬一点是能感知的。
  3. 榨汁机一般材料偏硬，内部镂空偏小；慢玩一般是材料相对更软，用料也多。
  4. 假舌头，假牙齿能明显感知，但是快感一般。
  5. 子宫设计是突出了一大圈材料，能明显感知，所谓突破感就是龟头能感觉到咯噔一下，子宫设计最大的意义是创造了一个变化。摩擦了一段时间，阴茎会渐渐适应，到临门一脚的时候，可能还要积蓄一两分钟左右的快感，才能完成。这个时候如果引入一些变化，比如速度快慢变化，或者子宫设计，能大大加速这个积蓄过程。

# 包茎，敏感度和快玩慢玩

  生理性包茎可以用飞机杯辅助翻出龟头。龟头一直没有外露，硬翻会有点疼，但用飞机杯辅助会分泌内酚酞，当时就没那么疼。事后的酸爽是因为里面的粘膜长期被包住，没有经历过内裤的摩擦，很嫩，适应几天就好了。

  快玩慢玩一开始能感知到一点差异，但随着时间的推移是趋同的。我经过多次测试，并且用稀的油，时间上没有显著差异。不排除我适应了快玩。

  除非是长期包茎，否则所谓的敏感度完全是伪命题，

  • 内酚酞的分泌，感知越来越弱。
  • 射精阈值与兴奋程度负相关，越兴奋越容易射。天天撸，阈值自然会越来越高。

  所以所谓快玩慢玩，我觉得是无所谓的。我倾向于1、要有变化 2、便宜

# 不同飞机杯明显差异点

# 重量

  从200g到1kg的飞机杯用下来，重量很容易感知，但是对快感影响不大。通常相近的材料越重越难撑开材料，自然有更大的压力。

  • 200g上下太薄，手能感知到阴茎，阴茎也能感知到手，太拉跨。

  • 350g上下不会太薄，而且翻洗很方便。黄金重量。

  • 500g以上基本和翻洗无缘。重量继续增加已经几乎感知不到区别，反而手很酸，整个顶起来感觉太重了。不过有特别的玩法，飞机杯可以用自重吃掉阴茎。

  • 一旦重量增加到1kg上下，最好放在桌子上或者有个炮架，太重了。这种有一点撞击的感觉，但阴茎的感知和快感，我认为没提升。

  翻洗清洗起来方便太多，不能翻洗的话要冲洗十几分钟，还要伸进去擦干，往往清洗的不彻底。用安全套的话，体验差很多。似乎因为材料的摩擦力还是比较大，很容易变成阴茎摩擦套套，甚至捅破套套。   不太好翻洗的，在玩具底部使劲往里塞能嗯翻。

# 材料

  同一品牌不同产品材料可能不同，但大部分厂家的产品我只体验过一两款，以偏概全仅供参考。材料越软，纹理设计越难感知。

  • ride Japan的材料耐用，便宜，性价比高，中等偏软，强烈推荐。
  • exe的材料几乎是最拉跨的，偏硬，压力偏大但却感觉有点空洞，不得劲。
  • npg的材料特别软，但是不耐久，会掉渣，用了一段时间居然发霉了，当然不排除是个例。
  • 对子哈特主打榨汁机人设，纹路设计大开大合，突出一大圈材料，能感知到。但对于快感没有明显提升，性价比不高。
  • Magic Eyes材料中等偏硬一点点，比较中庸。
  • tamatoys 会掉渣。软硬中等。
  • MACCOS 性价比拉满，100不到就有500克的玩，柔软度中等偏硬，弹性比较强，不是exe那种太硬的压力，而是弹弹的压迫感。有网友说maccos胶体味道和膳魔师硅胶件很像，哈哈。

# 润滑油

   分类细分有拉丝，粘稠度，保湿性还有气味。

  粘稠度能感知到明显区别。稀的容易感知纹理，稠的不容易感知纹理，但随着内酚酞的分泌，就算是稀的也渐渐感知不到纹理，稠的能明显感觉到更厚的油层。

  时间上，我经过测试没有显著差异。

  除非猎奇买加香精的，不然买便宜不发臭的普通粘度足够。我用过最拉跨的是银离子抗菌和女优冠名的淫臭液，如果不洗，睡一觉起来特别臭。

# 具体推荐

  越贵的一般主要是品牌溢价，营销溢价以及用料更多和新材料新工艺。除了从200g换成350g以上有提升，基本没有显著影响快感的差异，没必要买贵的。

  女优代言款性价比很低。

  国产的我买过19块的一款，标称重量大概400g，材料压迫感很小，感觉很空洞。更高价位的我没试过。

  日本品牌300g以上重量挑便宜的买，体验非常完备，更贵的提升有限，性价比不高。

  • Ride Japan CQ子宫 ￥100以内

  ride japan营销主打产品是三位一体，但我更推荐这款，一样的材料，还便宜，还有所谓子宫设计。

  • Ride Japan 天下一穴 100出头

  前段有一大块凸起，能感知到，材料也会稍硬一点。明显比cq子宫更有快感。

  • Maccos 蜜莉酱 ￥90

  90块600g，业界良心。翻洗要嗯翻，从底部使劲往里面塞，会导致入口有小裂口。这款性价比高的一逼，100不到这么重一大块，还有子宫设计。

  • Maccos cockfit蓝盒 雪板娘 ￥90

飞机杯之家 (opens new window) 的评论中有一位百杯斩的Howard，他的描述完全符合我的体验，不过他感受到的细节远比我多。他在评论中指出这款和实战很像。不过我没有实战经验，不清楚。
