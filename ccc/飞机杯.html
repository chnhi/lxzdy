# 飞机杯

网上评测牛逼吹的震天响，刺激与否很大程度上就是看阴茎受到的压力，用丝袜包住或者给炮架垫毛巾，给飞机杯施加很大的压力，甚至可能实现秒射。

飞机杯之家 (opens new window)，B酱评测 (opens new window)，毕竟是商家开的，免不了有所夸大。

TOYDEMON (opens new window)

倭国评测 (opens new window)

飞机杯从十几年前国产的非常空洞，体验非常差，到今天，相对性价比高一些的工厂都在中国，甚至NPG二次元系列，子品牌DOSO工厂也在中国，体验都非常完备。

巨变态的小日子做飞机杯甚至都做不过中国制造，以小见大，经济战中国必胜。

# 假倭鬼子

单纯正本清源，很多倭国品牌都是中国制造。

假倭鬼子我买过Demontoys和RQS的产品，我主观认为，快速运动基本接近MACCOS，可能细节上有差距，这种应该不断迭代累积经验，会慢慢变好，但这类假倭鬼子愿意总结经验吗？我持怀疑态度。

最糟糕的是气味非常刺鼻。

假倭鬼子最大的问题是，评测严重失真，严重夸大。在我看来完全没有理由卖得比MACCOS还贵。

GXP，Demontoys。

AkumaToys、Yokubou可能是台巴子搞的假倭鬼子，搞个日文官网欲盖弥彰，网站上的日文全看得懂。

RQS、rakukoto一开始在倭国运营，可能是中国老板投资的，不知道是不是打不开倭国市场，现在好像不在倭国活动了。Demontoys可能是同一个团伙。

# 购买方式推荐

电商平台有价格保护，给零售商留的利润率很高，淘宝闲鱼pdd有很多低价乱市。有些甚至就是授权品牌，整个都在中国开发制造，根本不需要担心低价有假。

还可以上阿里巴巴咨询客服买一件代发。不过现在阿里巴巴上的代理大都不愿意优惠了。

  飞机杯通常营销支出较大，靠软文绘声绘色的描写让你感觉要起飞，最早是营销高利润率的大牌子，如npg的003。但由于有低价乱市的现象，很容易为别人做嫁衣。商家就开始各自挖掘没名气的小牌子，拿下独占再营销，维持高利润率。

  tomax通贩大魔王也是这个路子，不分销，维持高利润率。

# 具体推荐

  1. 女优代言款性价比极低。
  2. 350克以上重量产品体验完备，更贵的提升不大。快速运动我认为基本差不多。
  3. 200克换成350克以上产品体验有明显提升，其他我认为是锦上添花。
  4. 600g左右的开始比较软的可能有一点撞击感，不过意义不大。
  5. 炮架体验提升较大，还可以垫毛巾增大压力。

体验非常主观，见仁见智。

# Tips

  • 零售商会送一堆劣质配件，联系客服都不要，折现。

  • 在翻洗的情况下，有毛巾用来擦干就够了。

  • 不翻洗，需要一个细长的棒子可以用毛巾裹住伸进去擦干，或者用卫生棉条吸水。

  • 长期闲置最好在杯身上抹爽身粉，不然会发粘。超市里几块一盒的玉米爽身粉就可以。

  • 翻洗清洗起来方便太多，不能翻洗的话要冲洗十几分钟，还要伸进去擦干，往往清洗的不彻底。用安全套的话，体验差很多。似乎因为材料的摩擦力还是比较大，很容易变成阴茎摩擦套套，甚至捅破套套。

  • 不太好翻洗的，在玩具底部使劲往里塞能嗯翻。

# 润滑油

  我自己只用玻尿酸类，易清理。

  主要有两类，一类是玻尿酸混甘油，另一类是甘油为主要成分。玻尿酸类体感上更像水（H2O）。阴茎对纹路的感知能力甚至比粘稠度是稀的甘油类更强，也更容易清理。缺点是容易干，如果比较持久还要再加。翻洗的时候更不易翻开，售价高。

  甘油类稀的容易感知纹理，稠的不容易感知纹理，随着内酚酞的分泌，就算是稀的也渐渐感知不到纹理，稠的能明显感觉到油层更厚。

  我用过最拉跨的是银离子抗菌和女优冠名淫臭液，睡一觉起来特别臭。

# 炮架

炮架对于体验的提升较大，而且可以垫毛巾增大压力。飞机杯套丝袜增大压力，或者带洞抱枕，下面垫个枕头什么的，实惠，体验也不错，就是不仿真而已。

海绵炮架有气味，手感也一般，但是造型相对仿真。充气炮架，摸一摸气球，你就知道有多假了。我觉得都不如带洞抱枕。

这几种自重都很轻，要按住或者用别的东西架住才行。

萌涩幻想之类的抱枕，还能加义乳，也凑合。

贵价一点的，搜·名器伴侣，有一款由密炮架三代，就是有点贵，将近400块，相当于实体娃娃的屁股部分。

# 阴茎没那么灵敏，感知不大

  • 评测飞机杯的以卖飞机杯的居多，形容的绘声绘色。但阴茎的知觉没有手指触觉那么灵敏，很多设计用手指能感觉，但阴茎无法感知。

  • 随着摩擦，大脑会分泌内酚酞，导致感知进一步下降，一开始能感知的一些设计，渐渐会无法感知。

  • 通道设计主要感知能力在龟头，感知会下降。阴茎其他部分主要能感知包裹感，但一般主观上感觉影响没那么大，因为通常会集中在龟头的感知。

  1. 纹路设计一开始能感知到区别，但渐渐无感，快速运动完全无感。
  2. 大肉粒能感知，也是很多产品的主要刺激点。
  3. 榨汁机一般材料偏硬，内部镂空偏小，营造更大压力。慢玩一般是材料相对更软，用料也多。
  4. 假舌头，假牙齿能明显感知，但是快感一般。
  5. 宫口设计突出一大圈材料，能明显感知，所谓突破感就是龟头能感觉到咯噔一下，宫口设计可以创造一个变化。临门一脚，直来直去的可能脚拇指扣紧还要折腾一两分钟，但有变化会加速。
  6. 无论是套丝袜还是炮架垫毛巾，施加压力后，感知会变得明显一些。

# 包茎，敏感度和快玩慢玩

  生理性包茎可以用飞机杯辅助翻出龟头。龟头一直没有外露，硬翻会有点疼，但用飞机杯辅助会分泌内酚酞，当时就没那么疼。事后的酸爽是因为里面的粘膜长期被包住，没有经历过内裤的摩擦，适应几天就好了。

  快玩慢玩产品的设计不同，往往只在一开始能感知到差异，随着时间的推移，感知趋同。我认为影响快慢的主要还是压力，如果觉得慢玩产品的压力太小，套丝袜或者炮架垫毛巾，增大压力。除非长期包茎，否则所谓的敏感度完全是伪命题，

  • 内酚酞的分泌，感知会越来越弱。
  • 射精阈值与兴奋程度负相关，越兴奋越容易射。

# 重量

  重量很容易感知，通常相近的材料越重越难撑开材料，可能有更大的压力。

  • 200g上下太薄，手能感知到阴茎，阴茎也能感知到手，太拉跨。

  • 350g上下不会太薄，而且翻洗方便。

  • 500~700g可能嗯翻还能翻，是我觉得比较好的重量区间。

  • 700g以上基本和翻洗无缘。重量继续增加已经几乎感知不到区别，反而手很酸，整个顶起来感觉太重了。飞机杯可以用自重慢慢下滑吃掉阴茎，不过意义不大。

  • 一旦重量增加到1kg上下，最好放在桌子上或者有个炮架，太重了。这种撞击的感觉会比较明显一点，但阴茎的感知我认为没提升，甚至我觉得是负作用。
