# 喷墨

只考虑墨仓式或喷头墨盒分体机型；喷头墨盒一体的虽然机器便宜，但是耗材贵，加墨麻烦，故障率非常高。

  • 打印速度比激光慢很多，商用机型800喷孔一色速度接近激光。

  • 打印效果

    • 黑度相对激光打印机不够。
    • 爱普生四色颜料都能达到118%cmyk色域容积，喷墨以外打印方式一般都是100%cmyk。
    • 颜料优点是不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹），大面积深色墨水在相纸上由于光线漫射，会产生烫金效应（深色墨水反光会泛白泛青铜色），观感不如染料自然。
    • 染料饱和度高，通透感，亮光感是颜料无法实现的。观感更吸睛，但色不稳定导致色准和色阶都受到限制，有层次上的损失。

  • 家用打印机一般都是RGB打印机，打印机驱动接受RGB编码，不要画蛇添足改成CMYK模式调色。

# 耐候性

  • 颜料非常好。
  • 染料很差，易褪色，便宜染料保护（覆膜、塑封、淋膜）后能坚持数年，但还是怕高温，怕潮湿。
    • 贵价耐候性好很多，但是同微喷颜料墨盒抽墨成本相近。
  • L8168的原装墨水，同爱普生的干式影像输出设备用的活的色彩D6r-S墨水的成分相似，以及富士的干式打印机的墨水，可能具体的显色性不同，但他们是成分相似的同类墨水。
  • 佳能G580的染料墨水是ChromaLife100，比爱普生6色原厂染料强，要pro100的chromalife100+才能和干扩机染料PK。

激光银盐怕UV怕空气；热升华怕高温；染料怕高温，怕潮湿，怕空气；

颜料墨水曝晒也会褪色，但是由于色料外包裹的树脂颗粒先被破坏，使得整体褪色程度相近，只是看起来显旧，颜色间相对联系不变，这也是人眼最敏感的部分。

wilhelm-research的标准可能是某个墨水的色密度衰减35%，我的结论基于打印84个色块中挑个别色块计算色密度衰减35%，平均dE00基本在4以上。 (opens new window)

染料褪色第一难题，灰平衡。

染料墨水机型灰阶主要由CMY墨水混合，而CMY墨水褪色速率不一，这往往导致灰平衡破坏的非常快，DS6墨水短短3天，中性灰块的dE00就超过2.5。

但是灰平衡如果没有参照对比，不一定明显感知。

染料C墨水褪色速率最快

以色密度计算，C墨水褪色速率达到M墨水的2.5~4倍。如果无视灰平衡，把C墨水替换成耐候性更好的墨水，可以明显改善整体的耐候性。

染料黑色墨水褪色发红

不仅堆黑显眼，而且会参与到彩色增加色密度，跟C墨水一起劣化整体发红的趋势。干式彩扩黑色墨水褪色更偏向饱和度下降显旧，不至于那么显眼。

如果打算用染料墨水，最好做保护，覆膜、塑封、淋膜。

  • 堵头

    • 染料就算是兼容墨水都不会堵，即使断线了也很容易泡开冲开。
    • 颜料原厂墨都有可能堵，特别是气候干燥地区。长期不打存在堵死冲不开的可能。至少每周打一次，可以是喷头检查图。

  • 连供

    • 填充墨盒稳定性，寿命好。颜料墨水只推荐用填充墨盒，连供要自己用上出墨的uv机用的墨瓶改装，下出墨的连供很容易堵管线影响流动性。
    • 墨仓机型墨囊结构不一样，运行清洗就行，稳定性好很多。
    • 佳能热发泡墨水消耗量大，打印头寿命短，酌情。

爱普生机型用兼容染料，约4万张A4后喷头状态下降，标准打印质量下降，需要调高打印质量增大喷墨量掩盖瑕疵，其寿命可见一斑。惠普建议热发泡每色1.5L左右墨水，约4500~6000张A4幅面照片更换打印头。显然，对于非商用来说，不太可能打那么多，如果打算用原装墨水，又觉得L8168机型太贵，佳能机型也不是完全不能考虑。

# 照片纸

  • 染料用铸涂相纸（又叫光面或高光相纸）成本较低。
  • 有些铸涂会掉粉，不如买高光喷墨铜版纸，效果相近。要200g以上，不然要调低质量，防止洇墨。
  • 颜料用RC相纸或艺术纸。

泛太克，柯达。汇美有珍珠面，明瑞号称代工爱普生相纸。

颜料分照片黑和粗面黑，粗面黑通常更不容易渗入介质，适合艺术纸。照片黑颜料在艺术纸上容易渗入介质，导致看起来泛纸白不够黑。但是在照片纸上，因为有涂层，粗面黑墨水往往显得死黑一片，损失暗部色阶，原装粗面黑会好一点，但原装粗面黑在照片纸上往往不耐刮擦。

需要有喷墨涂层的介质才能吸收更多墨水而不洇墨，进而增加色密度。通常相纸纸基上有一层PE膜，基本完全靠涂层吸墨，所以相纸往往有最高的色密度，通常有最广的色域，色域广了，自然也更容易表现更多的色阶。

而白卡纸，普通纸则几乎完全靠纸基吸墨，表现力严重受限，而艺术纸则在白卡纸的基础上，涂布有涂层，由涂层与纸基吸墨，色域远胜于白卡纸普通纸，相对相纸还是要差一些。从校色文件的色域容积来看，通常白卡纸普通纸不到相纸的四分之一，艺术纸大概是三分之二左右。

那相纸岂不是无敌了吗？也不尽然。人眼对颜色间相对联系更敏感，既除非AB对比，不然对饱和色的差距其实感知不明显。艺术纸哑光的特性看起来更自然，往往具有一种温暖的感觉。

尽管高光喷墨铜版纸，铸涂相纸在调低质量后都可以打水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，静置15分钟左右干燥后才附着。干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。

RC细绒相纸，艺术纸不易感知走纸印和棘轮印，高光相纸包括RC高光，高光喷墨铜版纸，反光能感知有走纸印和棘轮印。

据网友@逸 实测分析分享，校色后对比平滑度和色域空间大小，佳能旗舰相纸是消费级领域比较容易买到的顶级相纸，但是性价比实在太低，淘宝·华林兴性价比高，但sku多，有好有坏。泛太克代工的柯达可圈可点，国内纸品牌NO.1，但是贴泛太克品牌的不如贴柯达品牌的。

佳能相纸可以考虑买卷纸，自己先反卷，放置两三天后（根据实际情况判断反卷时间）自行分切，分切可以考虑买滚切刀。之后平放一段时间，等其自然整平。

如果打算自行分切，可以考虑在想要的尺寸上，头和尾都留出4厘米，优先纸尾，纸头可以不预留，在打印机驱动中自定义介质大小。

爱普生打印机的纸头尾4cm打印质量可能会下降，可能是因为头尾4cm不是完全压住。

# 荧光增白剂

荧光增白剂曝晒约2~3周降解，一旦降解会导致纸变得比不加荧光增白剂的还黄得多，不过只要不受到光照就不会降解。

要检测是否不含荧光增白剂，用365nm的紫外线手电筒照射即可，注意必须要365nm，有荧光增白剂的纸会发出幽幽的蓝光。395nm的不易分辨。

除此之外，考虑【天威】，便宜，尽管没有吹嘘耐光抗UV，但耐候性同友商吹嘘的“耐光抗UV”染料差不多，黑墨最好另外买。

黑墨褪色发红

绝大部分兼容染料，包括爱普生大幅面染料抽墨的黑色墨水褪色发红我主观感知特别显眼，鸿盛EverNew是斑斑驳驳发黄显旧那种感觉。但是测量的dE00，色密度，这种更拉跨。2周后，黑块的dE00，DS6是3.97，鸿盛是10.05。

另色鬼的影像专用黑墨在褪色初期，变色速度主观观察比友商慢。

得力51K墨水、代工 米家喷墨打印机【墨水TEJ4000CN】、代工 华为 PixLab V1【华为臻彩墨水】。我测试米家原装墨水的黑墨褪色初期不会变红，黑墨色密度衰减速率也明显碾压其他兼容染料。尽管是热发泡机型的墨水，但实际上微压电机型用也没什么问题，无非热发泡的墨水粘度会更低一点，影响不大。

大幅面染料抽墨

据我测试耐候性同兼容抗UV染料差不多，但是色域明显更广。

**大幅面染料：**P10080D（T42V），P11080D（T53Y），T5680D（T54A）。 其中T42V直接倒出来，另两款需要转接针头，我用2.5mm缺口针。

由于056墨水为了耐候性牺牲了色域，所以大幅面染料抽墨的浅色墨水用在056墨水机型上不匹配喷墨程序，会损失色域容积，由于耐候性同兼容抗UV染料相近，我会倾向于浅色用兼容耐光染料，用大幅面染料的深色墨水就足以扩展色域。或自行稀释 (opens new window)。

B9080，B9580等系列的活的色彩HDD墨水中高浓度黑墨水的色料根据海外披露是分散染料，既热升华墨水的色料，彩色墨水不清楚。我认为该系列墨水的上机或更换最好清洗墨路，根据海外信息，粘度也更高，低温下可能要注意流动性。我没有实际测试过，该系列墨水是否在桌面机型上正常使用亦不清楚。因为其黑墨成分是分散蓝360，不清楚是否需要喷墨程序配合才能喷出相对比较中性的黑。

# 兼容墨水更推荐颜料

兼容墨水大厂：墨库，鸿盛，天威，嘉兴百川，传美讯。

**墨库·百度西，商用数码墨水中国出货量第一，价廉物美，杂质过滤合格。颜料40多块500毫升。

兼容颜料普遍色域只有原装颜料的70%。往往低温流动性较差，低温可能斜喷，断线。

颜料墨水在高光相纸上有烫金效应（深色墨水反光会泛白泛青铜色），观感不如染料自然。用微喷墨盒抽墨相对兼容颜料可以显著改善烫金效应泛青铜色。

通常优先替换黑墨。P6080的HD系列墨水，比K3黑，但K3如果便宜也可以考虑，两者大面积堆黑反光都会泛白。P7580的Pro10，12系列黑墨反光不泛白，但是反光可能呈现出青铜色，泛白可能还自然一点。

# 换颜料墨水务必审慎

普通染料换来换去一般影响不大。颜料各家的分散剂成分可能不同，更换墨水存在堵头风险，务必审慎，颜料换染料，要特别审慎。

墨仓式打印机，加墨在墨仓中一般约50ml，打印头上有个墨囊（内墨盒），约3ml，打印头内还会残余一些墨水。

染料间或染料换颜料按该视频 (opens new window)操作即可，务必要深度清洗。

  • 利用深度清洗

颜料间，或颜料换其他墨水，墨仓内旧墨水抽走。用清洗液稀释液，10ml清洗液，60ml纯净水即可，墨库的清洗液就可以。加进墨仓，再抽走，直到抽出来是清澈的。墨库清洗液就是清墨水加“洗洁精”，不必妖魔化清洗液。

不介意成本，按染料换墨水的视频 (opens new window)，在墨仓内加注20~30ml清洗液，运行一次深度清洗，注意观察墨仓液面是否全部下降。再次加满稀释液清洗液，24小时后再次清洗墨仓，直到抽出来是清澈的，用清洗液再运行一次深度清洗。

保险起见，深度清洗的时候最好直接加清洗液。清洗液稀释液粘度太低，有可能从打印头上流出去，存在风险。

24小时后再次清洗墨仓，换上新墨水，清洗3~5次，3次后开始打喷头检查图。最好清洗液稀释液+深度清洗运行三次再上新墨水，尤其是颜料换染料，即使有颜料颗粒残留在打印头前的滤网上，打检查图一样很正常，但可能不稳定。

  • 拆墨囊手动清洗

希望最保险，参考该视频 (opens new window)把墨囊拆下来清洗。拆墨囊清洗的话，墨仓还是要灌满清洗稀释液，泡24小时再抽走，直到泡24小时后抽出来还是清澈的，最好用清洗液中转运行一次深度清洗后，再上墨水。

  • 检查图断线

清洗后如果没有好转迹象，先不要不停清洗，先观察分析。如果固定位置断线，可能是打印头堵了，可能最好找售后，或自己死马当活马医，用那种强效清洗液泡一下，或试试下文的终极一战看看能不能冲开。

如果不是固定位置断线，说明供墨不畅，先检查墨路供墨是否正常。运行清洗观察管线内是否有墨水流动，或墨仓液面是否下降，只要有下降，一般墨路应该是没有大问题。墨仓机型改过打印头结构，微压电晶体之前有一个很细的过滤网，这个过滤网堵的可能性相对大一些。那么就是终极一战，用10ml针筒打6~8ml清洗液回去，针筒轻轻拉满往回抽，会有黑墨被抽回来，是墨栈里的废墨，这样重复换新的清洗液推拉2~3次，可能可以冲开过滤网，最后再打4ml左右清洗液回去。

# 家用喷墨·影像输出

  染料的通透感，亮光感是颜料无法实现的。观感更吸睛，饱和度高，色彩还原能力更好，不存在光线漫射。

缺点一：易褪色，色不稳定，色准和色阶都受到限制，有层次上的损失。

缺点二：染料黑光谱有明显的红色偏移（看起来泛红），既660nm以上波长的可见光反射率上升。

  颜料优点是不易褪色，色准高，缺点有成本高，同色异谱，凹凸感（高光纸表面明显墨水颗粒的痕迹），输出大面积黑色时，由于光线漫射，会产生烫金效应，反光泛白泛青铜色。

相对P708，6色机型最大的短板是灰阶靠彩墨LUT；P708不需要无关墨水增加色密度，靠对应颜色配合黑灰墨水就可以。而6色机型，往往需要无关墨水中和增加色密度，更容易因为同色异谱而产生色偏。

# 小幅面推荐激光银盐冲印

6~8寸小幅面不如上淘宝冲印，更细腻，耐候性也强于普通原厂染料。

富士晶雅；阳光印像；确认是富士魔术坤，激光银盐第一坤。我猜测可以找支持富士晶彩旗舰版相纸的店铺，这个相纸好像不是阿猫阿狗都会备货，我猜测备货的都是自认为自己是激光银盐第一坤的。

L8168缺浅色墨水，浅色阶过渡上细看有明显颗粒感。

# 原装墨水一定选L8168

爱普生L8168的原装墨水是干式彩扩机同类染料，耐候性显著提升，在色域上也有明显扩展，既不需要专色也可以打出还算可以接受的大红色和蔚蓝色。

L8168的机壳明显更精巧，L8058整体透露出一种廉价感，比如8168的机盖有阻尼，出纸器自动伸缩，8058要自己拉出来。8168驱动会估算墨水余量，有触控屏维护很方便。

8058的速度明显更快，300g左右厚纸进纸稳定性更高。8168还能进一步调慢速度延长墨水干燥时间，8058没有提供类似功能。8168速度慢，可能更倾向于喷小墨滴。

8168的驱动开发，固件的开发明显更用心，而8058可能为了速度而降低了质量，也因此，8058在打印头状态下降后往往很容易出横纹，特别是标准质量。

# 原装墨水也可以考虑佳能

佳能G580墨水是chromalife100染料，耐候性相当不错，比爱普生普通染料强蛮多。但是要pro100用的chromalife100+重点加强黑墨耐候性才能和干式彩扩机染料PK。

但是爱普生机型用兼容染料，约4万张A4后喷头状态下降，标准打印质量下降，需要调高打印质量增大喷墨量掩盖瑕疵，其寿命可见一斑。惠普建议热发泡每色1.5L左右墨水，约4500~6000张A4幅面照片更换打印头。显然，对于非商用来说，不太可能打那么多，如果打算用原装墨水，又觉得L8168机型太贵，佳能机型也不是完全不能考虑。

# 灰阶不够中性

相对P708，4~6机型最大的短板是灰阶靠彩墨混合；P708不需要无关墨水增加色密度，靠对应颜色配合黑灰墨水就可以。而6色机型，往往需要无关墨水中和增加色密度，更容易因为同色异谱而产生色偏。

如果换用兼容颜料墨水，彩墨混合的灰阶会表现得更糟，因为染料墨水存在一定程度的晕染。但是染料墨水由于灰阶是CMY墨水混合而成，褪色速度不一，灰平衡破坏的速度太快几乎没有意义。

L8168驱动选大部分介质都只用014墨水通道既5色染料墨水，灰阶是彩墨加灰墨混合。但是介质VVE【爱普生天鹅绒美术纸】和【普通纸】，灰阶是用013颜料黑和014灰墨混合，灰阶明显要中性非常多，但问题在于喷墨程序是针对粗面纸，显著减少喷墨量，在RC相纸上会有明显颗粒感。

只用014墨水通道喷墨的介质选项，灰阶由彩墨混合，颜料墨水对灰阶的劣化在我看来浪费了导入的灰色墨水，同L8058半斤八两，所以L8168不推荐兼容颜料墨水。

如果用微喷颜料墨盒抽墨，L8168可以切换粗面黑/照片黑，对需要微喷才能处理的比较理想的深橄榄绿和紫色系，不清楚是否是喷墨程序的原因，L8168的处理相对L8058可能会更好一些。

爱普生L8168搭配原装墨水，提供了黑白模式，算是一个入门级可用的黑白输出，需要指出的是在该模式下输出时，013墨仓的黑墨，014墨仓的黑墨都参与输出。由于该模式喷墨程序略显复杂，所以不用原装墨水可用性可能较低，试图人为调整色环可能摁下葫芦起了瓢。

原装013墨水是粗面黑，在任何纸上都不耐擦。

# 晶亮色墨水

如果打算全部用兼容墨水，需要打有光泽度的纸张，考虑选佳能pro系列机型。晶亮色墨水可以改善兼容颜料的反光泛青铜色观感怪异。

如果有一定要求，Y，RED，BLUE，CO最好都用原装墨。可以说成也CO，败也CO，因为CO墨消耗量很大。据我测试观察，原装CO墨基本不改变原来的颜色，但是兼容CO墨像蒙了一层轻微的灰色滤镜，其中墨库的CO墨反光的光泽度相对比较接近原装CO墨，但是味道很大很冲，而且比过期墨盒便宜的不多。

爱普生系列则考虑，专色、深色墨水CMYK用原装墨水可以扩展色域，浅色墨水LC、LM、LK、LLK用兼容墨水。浅色墨水我认为可以尝试自行用深色墨水稀释 (opens new window)。

其中Y墨水是除了专色兼容相对原装差距最大的，兼容墨水的专色很大概率是用CMYK勾兑。

原装粗面黑墨水更不易渗进介质中，兼容粗面黑要么是浓度更高的照片黑，要么就是高速图文机用的，会死黑。

照片黑墨水，爱普生PRO10,PRO12一代，原装墨水会大幅度减轻反光泛白，但是喷墨量大了反光会泛青铜色。

据我测试观察，墨库兼容颜料用m3偏振测量黑位并不差，既兼容颜料的黑位观感差可能是因为树脂。

# 佳能PRO-200，8色染料机是大坑

1、佳能染料耐候性不如爱普生的干扩机同类墨水，墨水还更贵。

2、染料黑具有红色偏移（660nm以上波长的光线反射率呈上扬趋势），佳能PRO-200的灰阶一样靠彩墨LUT，黑灰墨水只是辅助，直接导致灰阶往往不够中性。这是喷墨程序的局限性，换颜料墨水没用。

# 佳能PRO500 vs 爱普生P908

pro500海外叫pro1000。

佳能大幅面已更新PRO2600，主要提升好像是墨水扩展色域，粗面黑更黑，提升耐刮擦性。

PRO500多出来的大红色发挥的场景显著扩展色域，如夕阳，花卉，中国红。

P908的紫罗兰墨水扩展Lab空间紫色方向的色域，而佳能的BLUE只扩展蓝色方向。

如果打6~8寸小幅面，佳能热发泡的墨滴更大，细腻度可能差些。热发泡老化的快，在一定打印量之后可能锐度差些，甚至色准下降，但这种细微的差异要细看才可能感知，实际上很难说有什么意义。

爱普生提供color base3软件，提供了入门级的cmyk线性化校色，会调整喷墨量。佳能pro1000支持RIP软件EFI和ONYX，不清楚pro500是否支持。RIP软件往往要价不菲。

PRO500有真空吸附，机器很重动静也大，要30多kg，P908才20kg不到。A2幅面性价比比较高的选择是自己分切卷纸，往往由于没有整平设备，平整度同预切纸有差距。真空吸附显然适应性更好，爱普生要P5380才有真空吸附。

PRO500的打印头有一部分喷嘴处于待机状态，准备进行断线补偿，打喷嘴检查图永远是完美的，所以一旦异常往往是惊吓。而P908在大幅面输出前，要先打喷头检查图确定喷头状态。

# 成本

爱普生P708/908原装墨盒3~4￥/ml，而大幅面机器P7580的墨盒抽墨，1.5~2.5￥/ml，过期墨盒1￥/ml。

这两款机型有墨盒芯片重置器（解码器），但墨盒只有25/50ml，用到25%左右就要加墨重置，防止用到15%锁死芯片。加墨后会清洗喷头，很浪费。

如果全部用原装墨水，考虑P5380等机型，不会浪费墨水在加墨清洗上。

佳能PRO500没有可加墨的填充墨盒卖，只能用针筒打回墨盒，或自己在墨盒屁股上钻洞，用堵头堵上就是填充墨盒了。但是墨量没有显示，只能自己毛估估或称重量。抽大幅面PFI-53(57)墨盒的墨水，每ml 1块多一点，拆机墨更便宜。

PRO1000能买大幅面的打印头PF-10替换，pro500不知道能不能用。不像爱普生打印头坏了基本报废。

爱普生不会积极自动清洗喷头，而佳能机型会积极自动清洗喷头，再加上热发泡墨水用量更大，实际耗材成本不一定。

# 佳能PRO300 vs 爱普生P708

佳能PRO-300要啥没啥，爱普生P708完胜。

P5380改为P908的墨水系统。打印头也从P5080的PrecisionCore换成了MicroPiezo既P908和P708的打印头。从360每色降级为180每色。P908一个人就能搬动，P5380要两个人才能搬运。

P5380相对P908最大的升级就是卷纸单元，自带卷纸刀，而且自动调节卷纸的松紧。有“真空系统”，既动静超大的风扇，能保证纸张不翘曲变形蹭到打印头，进灰也少。

用原装墨水打卷纸，不用多想，直接P5380。

如果用于有橙色绿色墨水的大幅面打样，可能还是考虑P5080。

用原装墨水，908买卷纸单元，用掉大约2L左右墨水之后，P5380更低的墨水成本就抵消掉机器成本了。就1组5380的墨水。

如果不买P908的卷纸单元，大概要打掉2.8L墨水。A4无边距（5档质量里第4档，1440dpi最高档）大概共消耗1ml不到的墨水。

如果一个月打不了几张，可能机器报废了，还没打掉1L墨水。那就没必要考虑P5380。

# 激光银盐冲印

  • 彩色激光打印巨便宜，通常用兼容耗材，不做色彩管理，色准一塌糊涂。

  • 艺术微喷：6寸1.5~2块/张。通常会做色彩管理。

  • 激光银盐冲印：6寸6毛~1.5一张

银盐能堆积出立体感，不是半色调，浅色没有喷墨的颗粒感。但是锐度低，宽容度小（丢失高光或暗部细节），灰阶不够中性。富士魔术手等软件会自动优化，机器自带密度计，不需要商家自行色彩管理。

6~8寸小幅面的细腻度还是可以接受的。

# 喷墨与特殊纸

# 热转印

热转印分为普通转印纸（几毛一张），水转印纸(3块起/A4)，纯棉T恤转印纸（烫画纸）（1.5￥起/A4）。

普通转印纸只能用热升华墨水，加热后墨水升华，染到印刷物上，手机壳，键盘键帽等常用该方法。热升华墨水一样会褪色。

水转印纸和纯棉T恤转印纸兼容水性墨水，转印后是把“纸”粘在被印物上。

  • 染料遇水会晕染，也容易褪色，尽量用颜料墨水。
  • 烫画纸有热熔胶和喷墨涂层，加热后胶水融化黏住，有胶质感，不透气。
  • 水转印纸有一层水溶胶。

纯棉T恤转印纸需要和烫画膜（PET离型膜）区分，烫画膜需要搭配数码烫画涂料墨水(DTF)，墨水中含有聚氨酯与异氰酸酯，溶剂蒸发后会固化成膜。再用热熔胶把图案粘到被印物上，烫画膜是拿走的。印有图案的地方不透气。

直喷涂料墨水(DTG)原理类似，只是会涂预处理液，预处理液既交联面料又吸附墨水。

涂料墨水具有腐蚀性，非工业打印头寿命可能是用周来计量，白墨更短。

# 涂层不匹配或无涂层纸

水性墨水需要被打印物有涂层，不然不吸墨，或者普通纸那样墨水渗入纸张但是分辨率较低。这些情况下，会有3种表现。洇墨，墨水有流动痕迹，一摸就花。

  • 洇墨，既墨水有轻微化开的感觉。通常考虑调整打印机驱动中介质选择改变喷墨程序，调低喷墨浓度来调整。

常见于介质选择不适配，或菲林纸，PET喷墨防水透明胶片 (水性防水菲林片)。

  • 一摸就花，不耐刮擦。

最好静置15~30分钟干燥，如果静置后依旧不耐刮擦，说明墨水不适合该介质。

尽管高光喷墨铜版纸，铸涂相纸都可以正常打印水性颜料墨水，但颜料颗粒是纸张表面的一层凸起物，干燥前，打印过程中不耐刮擦，这可能导致走纸印，棘轮印正视时也能感知。而染料墨水在高光纸上的走纸印需要反光才能感知。

  • 墨水有流动的痕迹，常见于无涂层纸，如胶版纸（道林纸）没有涂层。

无涂层，即使调整到可以打印了，效果也是很差的，只能考虑快干颜料墨水或工业墨水。

印刷厂俗称的铜版纸是涂布印刷纸coated art paper。没有喷墨涂层，喷墨打印机必须用铜版纸颜料墨水（快干颜料墨水）搭配加热装置。快干颜料墨水掺了弱溶剂墨水的溶剂，低温下粘度会变高，易出现斜喷等异常现象。

据墨库介绍，没有加热装置最好室温30度以上。

# 热发泡微压电墨水互用性

热发泡打印头用微压电墨水，正常工作，但是染料墨水加热后黑色会变红褐色。颜料墨水恐怕堵头概率会很大。

微压电打印头用热发泡墨水也能正常工作。

# 工业墨水

都是通过不同原理能固化成膜，极易堵头，往往具有强腐蚀性，非工业打印头寿命按天计算。

  • 弱溶剂墨水

非工业打印头用溶剂墨水可能3个月甚至更短就报废。

爱普生SC-S40680等机型的墨盒是溶剂墨水，可以抽墨。墨盒260块/700ml，国产溶剂墨水，便宜的80块/L。

  • 乳胶墨水（latex ink）

乳胶墨水也是水性墨水，在50~60度左右树脂分散乳液会固化成膜。HP用这个墨水的打印机用的是热发泡喷头，我推测腐蚀性可能较弱，堵头概率可能相对较小，但毕竟是会成膜的墨水，也不好说。

1688上东莞奥彩的兼容墨水卖325/L，可以零售。

鸿盛有适配微压电喷头的乳胶墨水，官网介绍打印介质加热到40~50度即可。咨询后，不零售，必须对接他们区域业务员，只接待商用客户，可能要买他们的机器或改装方案。他们好像用到空气干燥加热，可能必须要蒸发水分。

其介绍的工艺非常繁琐，不过可能是用在特殊介质上，先用不同喷头打一遍无色墨水，再数码打印，之后还要涂后处理液固色剂140度加热5分钟固色。

乳胶墨水好像要100多度才能完全充分反应，达到最强的黏着力，但是如果不是打印纺织品，可能自然风干也可以，只是黏着力弱些。

后处理液固色剂好像有二氧化硅使其摸起来滑，有会固化成膜的树脂材料等其他成分。

无色墨水据介绍有点像光油，但是只能打底不能盖面，主要起桥接作用，加强介质适应性，还可以平整提高精度，好像还可以提升交联性。好像之所以能实现吸墨涂层是因为会和白彩墨水产生成膜的反应，喷头要保持一定距离。

涂料墨水（DTF烫画墨水），UV墨水，都是通过不同原理固化成膜的墨水，都具有强腐蚀性，还极易堵头。

# 工业打印机

工业用墨水，包括UV墨水，涂料墨水，溶剂墨水都具有强腐蚀性，且极易堵头，喷头完全是消耗品，爱普生民用打印头往往一两个月就报废，几千上万的工业打印头维护得当也基本就一两年寿命，要靠商用分摊成本。

维护不得当甚至可能新打印头都马上报废。

就细节上稍微改改，打水性墨水就是写真机，打溶剂墨水就是喷绘机，打uv墨水就是uv打印机。

厂家都是联想式攒电脑的组装厂，基本都不掌握核心竞争力，同价位机器都差不多，主要还是看当地售后师傅是否热心。前提是正经厂家。

机型往往会用幅面代称，比如6090就是支持60mmX90mm，1610既支持1.6米x1米。

# 打印头

工业头基本都有加热功能，uv墨水低温下粘度会变高，爱普生I3200没有加热，T3200有加热功能。

工业头比较耐压，可以负压供墨，而家用头都是正压供墨，工业用墨粘度高，很容易供不上墨。

工业打印头基本都能应付溶剂墨水两年左右，应付uv墨水则层次不齐：

工业小头比民用头贵了很多，寿命也不算理想，整套机器也不便宜，有点尴尬。

  • 爱普生T3200（2.65寸），T3200有加热，价格小贵。

  • 理光G5/G6，1头2色。

G5标称7PL，G6标称5PL，G5大概1万2一个头。

# 专色好像不参与cmyk

我浅浅了解后发现，工业打印机曲线似乎只兼容CMYK和LC，LM，LK，LLK。爱普生微喷导入紫色，橙色，RGB等墨水会调整喷墨程序用于扩展色域。但是工业打印机好像只能把新导入的颜色作为专色，既指定具体几组RGB色值用这个墨打，或在图像文件中新建一个通道，这个通道同cmyk不产生联系，既用于打金属银，白墨，光油之类的特殊墨水。
