# Linus：作为消费者，成为所谓N粉，A粉或者是其他厂商的什么粉都是蠢蛋!

买数码产品常见误区:（1）BV1Th41117Lz (opens new window)（2）BV1SK4y1p7nH (opens new window)（3）BV1Pp4y1W7d6 (opens new window)

  1. 只看品牌不看型号，或者只看纸面参数。
  2. 没有取舍，忘记自己真实的需求，或者需求是KOL创造的。
  3. 过度在意负面信息。没有完美不会坏的产品。
  4. 信用要用来参考，不要盲从。实践出真知。
  5. 天下熙熙，皆为利来，形象越光辉越要警惕，务必对本人保持怀疑。
  6. 数码博主提前拿到样机，说辞雷同，对产品的称呼特别正式冗长，你懂的。
  7. 数码博主不会收钱故意诋毁。
  8. 用发展的眼光看问题，不要有刻板印象。

保持怀疑，尊重不同。
