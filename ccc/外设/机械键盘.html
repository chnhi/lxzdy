# 光轴

  传统轴寿命长短全看镀金层的厚薄，可再厚还是会有磨损，所以过个两三年总有几个轴会犯病。而光轴就不存在这个问题，可以一直用到主控报废。比如网鱼网吧就用光轴。

双飞燕已经迭代了三个版本的光轴。光轴的经验应该更丰富。但是双飞燕旗下键盘都是杀马特造型，而雷蛇又贵的一逼。

# 百元机械键盘最好不要买

  机械键盘犯病基本都是轴的镀金层舍不得镀的厚一点。入门价位的国产轴表现通常层次不齐，差的可能一年左右就开始犯病。

# 樱桃轴选什么牌子？

富勒=高斯=斐尔可=海盗船=铁修罗=诺普=Ducky=钛度=ikbc=一切用cherry轴的键盘

ikbc由信必成代工，而高斯也是由其代工，ikbc和高斯哪个便宜买哪个。

  斐尔可和海盗船以及雷蛇等品牌溢价那么多，那都是无耻！用高价让你以为玄学手感会好！但事实上提供的体验和别的樱桃轴键盘没任何区别。他们用一样的pcb板，一样采购的轴，一样的主控是一个客观事实，不受主观意志转移。

  厂家除了推出APP，就只会在外壳和键帽上做文章。所以选择品牌应该看：

  • 哪个颜值打动你，如阿米洛的樱。
  • 哪个更人性化，比如海盗船k68的防泼水。
  • 哪个原厂键帽手感更好。
  • 哪个性价比最高。
