# 机械键盘

  外设体验没有绝对的好也没有绝对的坏，通常只能对客观参数进行介绍，主观上的体验还是要自己去体验。

# 便宜的樱桃轴省心省力

  机械键盘犯病基本都是轴的镀金层舍不得镀的厚一点。入门价位的国产轴表现通常层次不齐，差的可能一年左右就开始犯病。通常和cherry轴一个价位或者便宜一点点的那些国产轴当中有比较好的，对于普通消费者而言，买一把便宜的樱桃轴省心省力。

  这几年国产的高端轴，定制轴，青出于蓝而胜于蓝，广受好评，机械键盘核心玩家大都更愿意拥抱国产客制化轴。

# 樱桃轴选什么牌子？

富勒=高斯=斐尔可=海盗船=铁修罗=诺普=Ducky=钛度=ikbc=一切用cherry轴的键盘

  斐尔可和海盗船以及雷蛇等品牌溢价那么多，那都是无耻！用高价让你以为玄学手感会好！但事实上提供的体验和别的樱桃轴键盘没任何区别。他们用一样的pcb板，一样采购的轴，一样的主控是一个客观事实，不受主观意志转移。

  厂家除了推出APP，就只会在外壳和键帽上做文章。所以选择品牌应该看：

  • 哪个颜值打动你，如阿米洛的樱。
  • 哪个更人性化，比如海盗船k68的防泼水。
  • 哪个原厂键帽手感更好。
  • 哪个性价比最高。

# 选什么轴体

  黑轴摁久了累，理论上伤关节，青轴太吵。茶红的主要区别是茶轴有段落感。极限击打的速度，青轴会慢一点，茶红表现差不多，茶或红对游戏的适用性差不多。

# 主流性价比

  对大部分消费者来说，买樱桃轴肯定不会犯错，但是黑峡谷GK715采用的凯华BOX轴我的体验比樱桃轴好。客制化键盘的散轴采购价，box轴也是高于樱桃轴。黑峡谷GK715唯一的败笔是华而不实的拔插底座。最好买pbt版，abs键帽容易打油，不在乎键帽不透光的，可以20块买一套pbt键帽自己换掉。

# 打字机

  每天大量打字的，可以考虑买一把静电容。一些网络写手出现键盘手的症状后改用静电容加每天按摩热敷，症状有一定改善。首推NIZ宁芝，人老板都用女儿名字当品牌了，能不用心么？

# 核心玩家

NIZ宁芝PLUM84电竞机械键盘

  佳达隆定制轴，镀金层比樱桃厚。全键无冲，最低125Hz、最高3000 Hz扫描频率。牛逼在有一堆固件可以刷，都是音游，fps的核心玩家联合职业选手开发的，真正的from gamer for gamer。

# 极致性价比

   淘宝咸鱼二手cherry轴键盘几十块。20块买套新的pbt键帽，准备好电烙铁和替换轴。
